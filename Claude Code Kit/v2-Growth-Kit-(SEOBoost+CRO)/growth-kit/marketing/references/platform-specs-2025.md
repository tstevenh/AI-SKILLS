## LinkedIn

### Algorithm Signals

| Signal               | Weight | What It Means                     |
| -------------------- | ------ | --------------------------------- |
| Dwell time           | High   | Time spent reading/watching       |
| Topic authority      | High   | Consistent posting on same topics |
| Early engagement     | High   | First 60 min activity             |
| Authentic engagement | Medium | Real comments vs pod activity     |
| Native content       | Medium | No external links performs better |
| Relevance            | Medium | Shows older posts if relevant     |

### Format Specifications

| Format       | Specs                               | Notes                           |
| ------------ | ----------------------------------- | ------------------------------- |
| Carousel     | 1080x1350px, 5-10 slides            | Highest engagement format       |
| Text Post    | 1,200-1,500 characters              | 3-line hook visible before fold |
| Document/PDF | 10-15 pages max                     | Good for frameworks             |
| Video        | 30-90 seconds, square or vertical   | Captions required               |
| Image        | 1200x1200px (square) or 1080x1350px | Single image less effective     |

### Best Practices

**Do:**

- Post 2-3x/week max (quality over quantity)
- Use carousel format for highest engagement
- Hook in first 3 lines (visible before "see more")
- Include CTA (follow, comment, share)
- Use 3-5 hashtags at the end

**Don't:**

- Post external links in post body
- Use engagement pods (detected by AI)
- Post daily (algorithm penalizes)
- Hashtag stuff

### Optimal Posting Times

- Tuesday-Thursday
- 7-8am or 12pm (local time of audience)
- Avoid weekends and Monday morning

## Twitter/X

### Algorithm Signals

| Signal           | Weight | What It Means                    |
| ---------------- | ------ | -------------------------------- |
| Replies          | High   | Especially from mutual followers |
| Quote tweets     | High   | 2x value of plain retweet        |
| Time spent       | Medium | On tweet and linked content      |
| Profile clicks   | Medium | Curiosity signal                 |
| Media engagement | Medium | Images/video boost visibility    |
| Early engagement | High   | First hours critical             |

### Format Specifications

| Format       | Specs                          | Notes                        |
| ------------ | ------------------------------ | ---------------------------- |
| Single tweet | <280 chars (<100 optimal)      | Short performs better        |
| Thread       | 8-15 tweets                    | Best for depth and followers |
| Image        | 1200x675px (16:9)              | 35% more engagement          |
| Video        | Up to 2:20, vertical preferred | Lower reach than static      |
| Quote tweet  | Add value to original          | 2x engagement vs RT          |

### Best Practices

**Do:**

- Keep tweets short and punchy
- Use threads for longer content
- Quote tweet with added value
- Engage with replies
- Post consistently (3-5x/day)

**Don't:**

- Thread everything (save for substantial content)
- Self-promote every post
- Use excessive hashtags
- Auto-DM new followers

### Optimal Posting Times

- 8am-10am, 12pm, 5pm-6pm (audience local time)
- Consistent daily posting works best
- Schedule threads for morning/midday

## Instagram

### Algorithm Signals

| Signal               | Weight  | What It Means                         |
| -------------------- | ------- | ------------------------------------- |
| DM shares            | Highest | "Sends per reach" is discovery signal |
| Saves                | High    | Content worth returning to            |
| Watch time (Reels)   | High    | Completion rate matters               |
| Likes per reach      | Medium  | Quality signal                        |
| Early velocity       | High    | First 30-90 min determines reach      |
| Relationship signals | Medium  | DMs, profile taps, comment history    |

### Format Specifications

| Format       | Specs                               | Notes                  |
| ------------ | ----------------------------------- | ---------------------- |
| Carousel     | 1080x1350px, 6-10 slides (up to 20) | Highest engagement     |
| Reel         | 1080x1920px, 7-60 seconds           | Best for discovery     |
| Single Image | 1080x1350px                         | Getting more support   |
| Story        | 1080x1920px, <15 seconds            | Best for DM engagement |

### Best Practices

**Do:**

- Focus on carousels and Reels
- Create "saveable" content
- End with CTA (save, share, comment)
- Use Reels for reach, Feed for engagement
- Post daily or nearly daily

**Don't:**

- Rely heavily on hashtags (reduced impact)
- Cross-post TikToks with watermark
- Ignore Stories (relationship building)
- Post low-quality images/video

### Optimal Posting Times

- 11am-1pm, 7pm-9pm
- Consistency matters more than exact time
- Stories throughout the day

## TikTok

### Algorithm Signals

| Signal                       | Weight  | What It Means                  |
| ---------------------------- | ------- | ------------------------------ |
| Watch time (early)           | Highest | First 3 seconds critical       |
| Completion rate              | Highest | Full watch + rewatch           |
| Shares (DMs)                 | High    | Direct sharing signal          |
| Niche alignment              | High    | Favors specialized over broad  |
| Video quality                | Medium  | Lighting, sound, editing       |
| Engagement (likes, comments) | Medium  | Less weight than watch metrics |

### Format Specifications

| Format     | Specs              | Notes                       |
| ---------- | ------------------ | --------------------------- |
| Short-form | 15-30 seconds      | Highest completion          |
| Medium     | 30-60 seconds      | Favored if retention strong |
| Long-form  | 1-3 minutes        | Good for tutorials          |
| All video  | 1080x1920px (9:16) | Vertical required           |

### Best Practices

**Do:**

- Hook in first 3 seconds (visual + verbal)
- Target niche communities
- Use trending sounds when relevant
- Post 1-3x daily
- End abruptly to drive rewatch

**Don't:**

- Add slow intros
- Cross-post without optimization
- Ignore trending formats
- Over-polish (authentic often wins)

### Optimal Posting Times

- 7pm-11pm (audience local time)
- Experiment with posting times
- Consistency matters

## YouTube

### Algorithm Signals

| Signal              | Weight | What It Means                           |
| ------------------- | ------ | --------------------------------------- |
| CTR                 | High   | Click-through rate from impressions     |
| Watch time          | High   | Total time watched                      |
| % watched           | High   | Completion percentage                   |
| Session time        | High   | Does your video keep people on YouTube? |
| Viewer satisfaction | Medium | Surveys + behavior signals              |
| Topical authority   | Medium | Focused channels recommended more       |

### Format Specifications (Shorts)

| Element    | Spec                             | Notes                       |
| ---------- | -------------------------------- | --------------------------- |
| Length     | 10-60 seconds (30-60 sweet spot) | Up to 3 min in some regions |
| Aspect     | 9:16 (vertical)                  | Required for Shorts         |
| Resolution | 1080x1920px                      | Higher quality preferred    |

### Format Specifications (Long-form)

| Element    | Spec                      | Notes                         |
| ---------- | ------------------------- | ----------------------------- |
| Length     | 8-12 minutes (sweet spot) | Longer OK if retention strong |
| Aspect     | 16:9 (horizontal)         | Standard                      |
| Thumbnail  | 1280x720px                | CTR target: 4-10%             |
| Resolution | 1080p or 4K               | Quality matters               |

### Thumbnail Best Practices

- 3 elements max (face, text, object)
- High contrast colors
- Readable at small size
- Emotion on face (if showing face)
- Curiosity gap (show outcome, not process)

### Best Practices

**Do:**

- Invest heavily in thumbnails and titles
- Hook in first 30 seconds
- Use verbal signposting ("First... Second...")
- Include engagement prompts throughout
- End with clear CTA + end screen

**Don't:**

- Use clickbait that doesn't deliver
- Have slow intros
- Ask for subs at the start
- Ignore audience retention analytics

### Optimal Posting Times

- Publish 2-3 hours before peak viewing
- Typical peaks: evenings and weekends
- Shorts: morning or evening

## Reddit

### Algorithm Signals

| Signal             | Weight  | What It Means               |
| ------------------ | ------- | --------------------------- |
| Upvote velocity    | Highest | Early upvotes in first hour |
| Comment engagement | High    | Discussion = quality signal |
| Community fit      | High    | Match subreddit culture     |
| Account karma      | Medium  | Trust signal for automod    |
| Post timing        | Medium  | Weekday mornings best       |

### Format Specifications

| Format        | Specs               | Notes                        |
| ------------- | ------------------- | ---------------------------- |
| Text post     | 500+ words          | Headers, bullets, formatting |
| Link post     | Compelling title    | Valuable destination         |
| Image/Gallery | Context in comments | Best for data, comparisons   |

### Best Practices

**Do:**

- Lead with value, not promotion
- Format with headers and bullets
- End with question to drive comments
- Build karma before promoting
- Be authentic and transparent

**Don't:**

- Over-promote (10:1 rule)
- Use marketing speak
- Cross-post identical content
- Delete underperforming posts
- Evade bans with new accounts

### SEO Value

- Domain Authority: 91/100
- Posts index within 24 hours
- Target "Best [X]" keywords in titles
- 1,348% visibility growth in Google

### Optimal Posting Times

- Weekday mornings (US time)
- Sunday evenings
- Varies by subreddit

## Email

### Deliverability Factors

| Factor            | Impact | Best Practice                     |
| ----------------- | ------ | --------------------------------- |
| Sender reputation | High   | Warm up new domains gradually     |
| List hygiene      | High   | Remove bounces and unengaged      |
| Authentication    | High   | SPF, DKIM, DMARC configured       |
| Content quality   | Medium | Avoid spam trigger words          |
| Engagement rate   | Medium | Opens and clicks improve delivery |

### Format Specifications

| Element      | Spec             | Notes                      |
| ------------ | ---------------- | -------------------------- |
| Subject line | 30-60 characters | Longer truncated on mobile |
| Preview text | 40-90 characters | Shows in inbox             |
| Body width   | 600px max        | Standard email width       |
| CTA button   | 44px min height  | Mobile tap target          |

### Best Practices

**Do:**

- Write like a human, not a brand
- One CTA per email
- Use preview text strategically
- Test subject lines (A/B)
- Segment your list

**Don't:**

- Use spam trigger words (FREE!!!, Act Now)
- Send without authentication
- Buy email lists
- Ignore unsubscribe requests
- Send to cold lists without warming

### Optimal Send Times

- B2B: Tuesday-Thursday, 9-11am
- B2C: Tuesday-Thursday, 7-9am or 7-9pm
- Avoid: Monday morning, Friday afternoon

## Updates and Changes

This document reflects platform states as of early 2025. Platforms change frequently. Key recent changes:

**LinkedIn:**

- Organic reach down ~50%
- Quality over quantity now enforced
- Pod detection improved

**Instagram:**

- DM shares now major discovery signal
- Photos getting more feed support again
- Hashtags significantly deweighted

**TikTok:**

- Longer content (30-60s) favored if retention high
- Niche communities boosted over broad virality
- Deeper AI personalization

**YouTube:**

- Shorts expanded to 3 minutes in some regions
- Stronger topical authority weighting
- AI-driven hyper-personalization

Monitor platform announcements and adjust strategies accordingly.
