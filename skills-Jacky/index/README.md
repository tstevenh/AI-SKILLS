# Indexsy Skills

**A comprehensive library of AI skills for digital marketing, growth, and automation.**

Teach your AI agent to perform expert-level work across SEO, content creation, paid advertising, automation, security testing, and more. Each skill includes detailed methodologies, templates, and actionable frameworks.

---

## 🚀 Quick Install

### For Clawdbot
```bash
npx degit indexsy/skills/ab-test-design ~/.clawdbot/skills/ab-test-design
```

### For Claude/Cursor/Other Agents
```bash
npx degit indexsy/skills/ab-test-design ./skills/ab-test-design
```

### Or clone all skills
```bash
git clone https://github.com/indexsy/skills.git
```

## 📝 Usage

After installing a skill, tell your agent what you want to accomplish:

> "Help me write a cold email sequence for SaaS founders"

> "Check if these backlinks are indexed by Google"

> "Create an A/B test plan for our pricing page"

> "Analyze our competitor's SEO strategy"

The agent will reference the skill's SKILL.md and provide expert-level guidance.
