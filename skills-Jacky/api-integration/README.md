# API Integration

Master API integration for connecting systems, automating workflows, and building robust data pipelines.

## What It Does

This skill provides comprehensive knowledge for integrating with REST APIs, GraphQL, webhooks, and automation platforms. Covers authentication, error handling, rate limiting, and building reliable integrations.

## Features

- REST API fundamentals and best practices
- HTTP methods, status codes, and headers deep dive
- OAuth 2.0 and authentication methods
- GraphQL basics and query construction
- Webhook design and handling
- Rate limiting strategies
- Error handling and retry patterns
- iPaaS platforms (Zapier, Make, n8n)

## Installation

1. Copy the `api-integration/` folder into your OpenClaw skills directory
2. Restart OpenClaw

## Usage

Use this skill when:
- Integrating with third-party APIs
- Building webhook handlers
- Designing API architectures
- Handling authentication flows
- Creating data sync pipelines
- Troubleshooting API issues

## Requirements

- API credentials for target services
- HTTP client (curl, Postman, or code libraries)
- Programming environment (Python/Node.js)
- Understanding of JSON and data formats
