# Pricing Checklist

## Pricing Strategy

### Value Understanding
- [ ] Customer ROI quantified
- [ ] Value metric identified
- [ ] Segments defined by value received
- [ ] Competitive pricing analyzed

### Research
- [ ] Willingness-to-pay research conducted
- [ ] Customer interviews completed
- [ ] Competitor pricing documented
- [ ] Market benchmarks gathered

---

## Pricing Structure

### Tiers
- [ ] Number of tiers decided (typically 3-4)
- [ ] Target tier identified
- [ ] Features allocated to each tier
- [ ] Price points determined
- [ ] Tier names chosen

### Model
- [ ] Pricing model selected (seat/usage/flat/hybrid)
- [ ] Billing cycles defined (monthly/annual)
- [ ] Annual discount determined (typically 15-25%)
- [ ] Enterprise pricing approach defined

---

## Pricing Page

### Design
- [ ] Clean, scannable layout
- [ ] Recommended tier highlighted
- [ ] Monthly/annual toggle works
- [ ] Mobile-responsive

### Content
- [ ] Clear tier names and prices
- [ ] Feature comparison table
- [ ] FAQ section
- [ ] Social proof (logos, testimonials)
- [ ] Clear CTAs

### Technical
- [ ] Pricing dynamically generated (if localized)
- [ ] Currency handling
- [ ] Tracking and analytics

---

## Implementation

### Systems
- [ ] Billing system configured
- [ ] Subscription plans created
- [ ] Proration rules defined
- [ ] Tax handling confirmed

### Communication
- [ ] Sales team trained
- [ ] Support team briefed
- [ ] Documentation updated
- [ ] Customer FAQ ready

---

## Pricing Change (If Applicable)

### Planning
- [ ] Change rationale documented
- [ ] Grandfather policy defined
- [ ] Timeline established
- [ ] Affected customers identified

### Communication
- [ ] Email templates prepared
- [ ] High-value accounts identified
- [ ] FAQ for changes ready
- [ ] Support team prepared

### Execution
- [ ] 60+ days notice given
- [ ] Personal outreach to key accounts
- [ ] Changes implemented
- [ ] Monitoring in place
