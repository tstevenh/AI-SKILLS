# Onboarding Checklist

## Strategy

### Definition
- [ ] Activation metric defined
- [ ] Time-to-value target set
- [ ] Key milestones identified
- [ ] Success criteria established

### Segmentation
- [ ] User segments defined
- [ ] Onboarding varies by segment
- [ ] Use cases identified
- [ ] Personalization strategy

---

## First-Run Experience

### Signup Flow
- [ ] Minimal required fields
- [ ] Social login options
- [ ] Email verification (if needed)
- [ ] Progressive profiling

### First Session
- [ ] Clear first action
- [ ] Quick win achievable
- [ ] Value delivered
- [ ] Next steps visible

### Setup
- [ ] Smart defaults set
- [ ] Templates available
- [ ] Sample data option
- [ ] Progressive complexity

---

## In-App Guidance

### Product Tour
- [ ] Tour designed (3-5 steps)
- [ ] Skippable
- [ ] Action-oriented
- [ ] Contextually triggered

### Checklist
- [ ] 5-7 items maximum
- [ ] First item pre-completed
- [ ] Progress indicator visible
- [ ] Completion reward defined

### Empty States
- [ ] Helpful guidance
- [ ] Clear next action
- [ ] Templates/examples linked

### Tooltips
- [ ] Key features highlighted
- [ ] Non-intrusive
- [ ] Dismissible

---

## Email Sequence

### Content
- [ ] Day 0: Welcome email
- [ ] Day 1: First action check
- [ ] Day 3: Feature highlight
- [ ] Day 5: Use case inspiration
- [ ] Day 7: Progress check
- [ ] Day 10: Social proof
- [ ] Day 12-14: Trial reminder

### Technical
- [ ] Triggered by signup
- [ ] Personalized with name
- [ ] Behavior-based variations
- [ ] Unsubscribe option

---

## Measurement

### Tracking
- [ ] Activation rate tracked
- [ ] Time-to-value measured
- [ ] Funnel steps monitored
- [ ] Drop-off points identified

### Analysis
- [ ] Cohort analysis by channel
- [ ] A/B tests running
- [ ] Regular review cadence
- [ ] Continuous optimization
