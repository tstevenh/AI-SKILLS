# Launch Checklist

## Pre-Launch (4 Weeks Out)

### Product
- [ ] Product is stable and tested
- [ ] Core features complete
- [ ] Known bugs documented
- [ ] Load testing completed
- [ ] Backup systems verified

### Content
- [ ] Landing page live
- [ ] Pricing page complete
- [ ] Documentation ready
- [ ] Blog post drafted
- [ ] Help center articles written

### Assets
- [ ] Logo in all sizes
- [ ] Product screenshots (5-10)
- [ ] Demo video created
- [ ] Social media images
- [ ] Press kit prepared

---

## Pre-Launch (2 Weeks Out)

### Marketing
- [ ] Email to waitlist drafted
- [ ] Social media posts scheduled
- [ ] Product Hunt submission prepared
- [ ] Community posts drafted
- [ ] Influencer outreach done

### Support
- [ ] Support team briefed
- [ ] FAQ published
- [ ] Response templates ready
- [ ] Escalation path defined

### Team
- [ ] Launch roles assigned
- [ ] Communication channels set
- [ ] On-call schedule confirmed
- [ ] Celebration planned

---

## Launch Day

### Morning
- [ ] All systems operational
- [ ] Team in position
- [ ] Monitoring dashboards open
- [ ] Product Hunt goes live (if using)

### Execution
- [ ] Email sent to waitlist
- [ ] Social media posts published
- [ ] Community announcements made
- [ ] Press release sent (if applicable)

### Monitoring
- [ ] Traffic monitored
- [ ] Signups tracked
- [ ] Errors monitored
- [ ] Support queue managed

### Engagement
- [ ] Respond to all comments
- [ ] Thank supporters
- [ ] Share updates

---

## Post-Launch (Week 1)

- [ ] Analyze launch metrics
- [ ] Follow up with new signups
- [ ] Collect feedback
- [ ] Fix critical issues
- [ ] Publish recap/learnings
- [ ] Thank team and supporters
- [ ] Plan next steps
