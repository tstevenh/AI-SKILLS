# Loom Growth Case Study

## Company Overview

**Founded:** 2015 (as Opentest, rebranded 2016)
**Category:** Async video communication
**Pricing Model:** Freemium + per-seat
**Exit:** Acquired by Atlassian for $975M (2023)

---

## Growth Trajectory

| Year | Metric |
|------|--------|
| 2016 | Launch |
| 2019 | 1.5M users |
| 2020 | 10M+ users (pandemic) |
| 2021 | 14M+ users |
| 2023 | 25M+ users, $975M acquisition |

---

## What Made Loom Successful

### 1. Built-In Viral Loop

**The Mechanism:**
```
User records video
→ Shares link with recipient
→ Recipient watches video
→ Sees Loom interface/branding
→ Thinks "I could use this"
→ Signs up
→ Records and shares video
```

**Key:** Every video shared was a product demo.

### 2. Async Video Timing

**Why It Worked:**
- Remote work explosion
- Meeting fatigue
- Async communication trend
- Quick to record vs. schedule meeting

### 3. Generous Free Tier

**Free Includes:**
- 25 videos
- Up to 5 minutes each
- Basic features

**Upgrade For:**
- Unlimited videos
- Longer recordings
- Team features
- Analytics

### 4. Chrome Extension Distribution

**Strategy:**
- Browser extension = easy installation
- One-click recording
- Low friction
- Chrome Web Store distribution

### 5. Category Creation

**Positioning:**
- Created "async video" category
- First-mover advantage
- Became verb ("just loom me")

---

## Key Lessons from Loom

1. **Every share is marketing**: Viral loop built into core action
2. **Timing matters**: Right product for remote work moment
3. **Category creation**: Defining new category = owning it
4. **Low friction wins**: Browser extension = easy adoption
5. **Generous free drives viral**: Free tier enables sharing
