# Email Conversion

## Introduction

Email remains one of the most effective conversion channels for SaaS. Unlike social media where algorithms control reach, email gives you direct access to your audience. Lifecycle emails, upgrade campaigns, re-engagement sequences, and win-back campaigns all play crucial roles in maximizing customer value.

---

## 1. Lifecycle Email Strategy

### Email Lifecycle Stages

**Stage 1: Prospect**
- Lead nurture emails
- Educational content
- Problem awareness

**Stage 2: Trial/Free**
- Onboarding sequence
- Activation prompts
- Feature education

**Stage 3: Customer**
- Success check-ins
- Feature adoption
- Expansion opportunities

**Stage 4: At Risk**
- Re-engagement
- Health check
- Save campaigns

**Stage 5: Churned**
- Win-back campaigns
- New feature announcements
- Return incentives

### Lifecycle Email Map

```
Prospect → [Lead Nurture] → Sign Up
                              ↓
                         [Onboarding]
                              ↓
Trial User → [Activation] → [Conversion]
                              ↓
Customer → [Success] → [Expansion]
                              ↓
At Risk → [Re-engagement] → [Save]
                              ↓
Churned → [Win-back] → Return?
```

### Email Frequency Guidelines

| Stage | Frequency | Content Focus |
|-------|-----------|---------------|
| Onboarding | Daily for 7-14 days | Activation |
| Trial | Every 2-3 days | Conversion |
| Active Customer | Weekly-Monthly | Value/Education |
| At Risk | Increased frequency | Re-engagement |
| Churned | Monthly | Win-back |

## 2. Upgrade Campaigns

### Upgrade Campaign Types

**Limit-Based:**
```
Trigger: User at 80% of limit
Message: Approaching limit notification
Goal: Prevent disruption, prompt upgrade
```

**Value-Based:**
```
Trigger: User achieves milestone
Message: Celebrate and show next level
Goal: Expand on momentum
```

**Time-Based:**
```
Trigger: X days on free/trial
Message: What they're missing
Goal: Convert before habit without paid
```

**Feature-Based:**
```
Trigger: User tries gated feature
Message: Full access with upgrade
Goal: Convert on demonstrated need
```

### Upgrade Email Examples

**Approaching Limit:**
```
Subject: Heads up: You're almost at your limit

Hi [Name],

You've used 8,500 of your 10,000 monthly events. 
You're clearly getting value from [Product]!

Upgrade now to avoid any interruption:
[Button: Upgrade to Pro]

Or, your usage resets on [Date] if you want to wait.

[Your Name]
```

**Milestone Celebration:**
```
Subject: 🎉 You've hit a milestone!

Hi [Name],

Congrats! You just [achieved milestone]:

📊 [Metric]: [Value]
📈 [Metric]: [Value]

You're in the top 10% of [Product] users!

Ready to take it to the next level? Pro users 
typically achieve [additional result].

[Button: Explore Pro Features]

[Your Name]
```

**Feature Teaser:**
```
Subject: You tried [Feature]—here's how to unlock it

Hi [Name],

I noticed you explored [Premium Feature]. Great choice!
Here's what you could do with full access:

• [Benefit 1]
• [Benefit 2]
• [Benefit 3]

[Customer Name] used [Feature] to [specific result].

[Button: Unlock [Feature] with Pro]

[Your Name]
```

### Upgrade Campaign Metrics

| Metric | Target |
|--------|--------|
| Open rate | >30% |
| Click rate | >3% |
| Upgrade conversion | >1-3% |

## 3. Re-Engagement Campaigns

### Identifying Disengaged Users

**Warning Signs:**
- No login in X days
- Usage declining
- Features unused
- Support tickets increasing
- NPS declining

**Segmentation:**
```
Engaged: Active in last 7 days
At Risk: No activity in 14-30 days
Dormant: No activity in 30-60 days
Churning: No activity in 60+ days
```

### Re-Engagement Email Sequence

**Day 14 (First Touch):**
```
Subject: Everything okay, [Name]?

Hi [Name],

I noticed you haven't logged into [Product] recently. 
Just checking in—is everything okay?

If you hit a roadblock, I'm here to help. Just reply.

In the meantime, here's what's new:
• [New feature 1]
• [Improvement]
• [Resource]

[Button: Log Back In]

[Your Name]
```

**Day 21 (Value Reminder):**
```
Subject: Don't forget what you built

Hi [Name],

Your [projects/data/work] in [Product] misses you!

You have:
• [X] [items] created
• [Y] [items] pending
• [Z] [achievements]

Pick up where you left off:
[Button: Continue Your Work]

Need help? Reply and I'll assist.

[Your Name]
```

**Day 30 (Resource Offer):**
```
Subject: Quick wins to get back on track

Hi [Name],

Getting back into a tool after a break can feel 
overwhelming. Here are some quick wins:

1️⃣ [Simple action] — takes 2 minutes
2️⃣ [Simple action] — takes 5 minutes
3️⃣ [Simple action] — takes 10 minutes

Or book a 15-minute call and I'll help personally:
[Calendar Link]

[Your Name]
```

**Day 45 (Honest Check-In):**
```
Subject: Is [Product] still right for you?

Hi [Name],

I've reached out a few times and wanted to be direct:
Is [Product] still useful for you?

If yes but you're stuck: Let me help. [Calendar Link]

If not right now: No problem. I'll stop emailing. 
Your account stays active whenever you're ready.

If you've found a better solution: I'd genuinely love 
to know what's working better. Just reply.

No hard feelings either way.

[Your Name]
```

## 4. Win-Back Campaigns

### Win-Back Strategy

**Timeline:**
- Day 0: Churned (cancellation)
- Day 1: Exit survey
- Day 7: First win-back
- Day 30: What's new
- Day 60: Incentive offer
- Day 90: Final attempt
- Ongoing: Occasional updates

### Win-Back Email Examples

**Day 7 (Post-Churn):**
```
Subject: We miss you, [Name]

Hi [Name],

It's been a week since you left [Product]. I hope 
you're doing well and found what you were looking for.

If you ever want to come back, your [data/work] is 
still here. Just log in and pick up where you left off.

[Button: Reactivate Account]

Is there feedback you'd like to share? Reply anytime.

[Your Name]
```

**Day 30 (What's New):**
```
Subject: Here's what's new at [Product]

Hi [Name],

Since you've been gone, we've been busy:

✨ [New feature]
🚀 [Improvement]
🔧 [Fix for common pain point]

Thought you might want to know in case any of these 
address why you left.

[Button: See What's New]

[Your Name]
```

**Day 60 (Incentive):**
```
Subject: Come back with 30% off

Hi [Name],

I'd love to welcome you back to [Product].

Use code COMEBACK30 for 30% off your first 3 months.

This code expires in 7 days.

[Button: Reactivate with 30% Off]

Questions? Just reply.

[Your Name]
```

**Day 90 (Final):**
```
Subject: Last note from me

Hi [Name],

I've reached out a few times and want to respect your 
inbox. This is my last win-back email.

If you ever want to return:
• Your data is safe for [X months]
• Pricing starts at $[X]/month
• [Button: Reactivate Anytime]

Thanks for being a customer. If there's ever feedback 
you want to share, I'm always listening.

Best,
[Your Name]

P.S. I'll still send occasional product updates 
(quarterly), but no more win-back emails.
```

### Win-Back Benchmarks

| Metric | Target |
|--------|--------|
| Win-back open rate | >20% |
| Win-back return rate | 3-10% |
| Time to return | 30-90 days |

## 5. Email Best Practices

### Subject Lines

**Good Subjects:**
- Personal: "[Name], quick question"
- Curiosity: "Did you see this?"
- Value: "Your weekly results"
- Urgency: "Trial ends in 24 hours"
- Specific: "You've used 8,000 of 10,000 events"

**Avoid:**
- ALL CAPS
- Excessive punctuation!!!
- Misleading clickbait
- Spammy words
- Too long (>50 chars)

### Email Design

**Structure:**
- Clear from name
- Preview text set
- Single column
- Mobile-optimized
- Clear CTA
- Unsubscribe link

**Length:**
- Transactional: Short (50-100 words)
- Educational: Medium (150-300 words)
- Nurture: Can be longer with value

### Deliverability

**Best Practices:**
- Authenticate (SPF, DKIM, DMARC)
- Clean list regularly
- Monitor bounces
- Easy unsubscribe
- Consistent sending
- Engage engaged segments

**Metrics to Monitor:**
- Deliverability rate (>98%)
- Open rate (benchmark: 20-30%)
- Click rate (benchmark: 2-5%)
- Unsubscribe rate (<0.5%)
- Spam complaints (<0.1%)
