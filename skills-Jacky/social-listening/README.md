# Social Listening

Comprehensive social listening for marketing, research, and business intelligence.

## What It Does

This skill enables systematic capture and analysis of consumer conversations across social platforms. Monitor brand mentions, track sentiment, identify trends, and extract actionable business intelligence.

## Features

- Social listening vs monitoring distinctions
- Platform-specific strategies (Twitter, Reddit, LinkedIn, TikTok)
- Sentiment analysis and emotion detection
- Competitor tracking and share of voice
- Crisis detection and early warning systems
- Influencer identification and analysis
- Dark social and private conversation insights
- Integration with VoC (Voice of Customer) programs

## Installation

1. Copy the `social-listening/` folder into your OpenClaw skills directory
2. Restart OpenClaw

## Usage

Use this skill when:
- Monitoring brand reputation
- Tracking campaign performance
- Identifying emerging trends
- Analyzing competitor mentions
- Detecting potential crises
- Understanding customer sentiment
- Finding influencers and advocates

## Requirements

- Social listening tool (Brandwatch, Sprout Social, or similar)
- API access to social platforms
- Data visualization tools
- Text analysis capabilities
- Storage for historical data
