## Tech and Review Thumbnails

### Example 1: Product Comparison

**Visual description**:
- Split screen: Product A vs Product B
- Large product images
- "VS" in the middle (often in red)
- Clean background (white or gradient)
- Minimal text (product names only)

**Why it works**:
- Clear comparison premise
- Viewers know what to expect
- Products are recognizable
- Clean and professional
- Creates decision-making appeal

**Replicable pattern**:
```
[Product A] [VS] [Product B]
On white/clean background
Product names or nothing else
```

### Example 2: Winner/Best Pick

**Visual description**:
- Featured product prominently displayed
- Large checkmark or crown
- "BEST" or "#1" text
- Satisfied/excited face optional
- Clean, minimal design

**Why it works**:
- Clear recommendation
- Saves viewer research
- Authoritative positioning
- Simple message

**Replicable pattern**:
```
[Product] + [Crown/Checkmark/Award symbol]
"BEST [Category] 2024"
Professional background
```

### Example 3: The Problem Reveal

**Visual description**:
- Product with warning signs
- Concerned or surprised face
- Red circles, X marks
- Text: "Don't buy" or "WARNING"
- Before/after showing problem

**Why it works**:
- Fear of wrong decision
- Protective instinct
- Insider information appeal
- Negative emotion (stronger than positive)

**Replicable pattern**:
```
[Product] + [Warning symbol/red circle]
Concerned face expression
Text implying caution
```

### Example 4: Unboxing/First Look

**Visual description**:
- Hands holding product (coming out of box)
- New/sealed product visible
- Excited expression
- "FIRST LOOK" or "UNBOXING" text
- Clean background

**Why it works**:
- Exclusivity feeling
- New product excitement
- Participation in discovery
- Curiosity about contents

### Example 5: Long-Term Review

**Visual description**:
- Product showing wear/use
- "1 YEAR LATER" text
- Honest expression
- Before/after comparison
- Authenticity signals

**Why it works**:
- Unique perspective (time)
- Honest review expectation
- Useful information
- Differentiates from initial reviews

## Key Patterns Across Niches

### Universal Elements That Work

**1. Faces win**
- Expression tells story
- Human connection
- Emotional resonance
- 75%+ of top thumbnails include faces

**2. Contrast creates interest**
- Before/after
- Good/bad
- Old/new
- Expectation/reality

**3. Specificity builds trust**
- Numbers over vague claims
- Exact amounts/time periods
- Concrete results
- Specific methods

**4. Simplicity scales**
- 3 or fewer main elements
- Clear focal point
- Minimal text
- Readable at any size

**5. Emotion drives clicks**
- Strong expressions
- High energy colors
- Drama or intrigue
- Positive or protective

### Elements by Niche

| Niche | Primary Element | Color Tendency | Face Style |
|-------|----------------|----------------|------------|
| Tech | Product | Clean, professional | Moderate |
| Education | Data/proof | Varied | Engaged |
| Entertainment | Reaction | Bright, saturated | Extreme |
| Business | Money/results | Green, professional | Serious |
| Gaming | In-game + face | High saturation | Extreme |
| Fitness | Body/transformation | Bold, energetic | Determined |
| Cooking | Food | Warm, appetizing | Joyful |

## Thumbnail Templates by Type

### Template 1: The Comparison

```
┌─────────────────────────────────┐
│  [Option A]  │VS│  [Option B]   │
│    image     │  │    image      │
│              │  │               │
│  [Label]     │  │   [Label]     │
└─────────────────────────────────┘
```

**Use for**: Reviews, matchups, decisions

### Template 2: The Transformation

```
┌─────────────────────────────────┐
│  BEFORE  │    ➔    │   AFTER    │
│  [image] │         │  [image]   │
│          │         │            │
│  [text]  │         │  [text]    │
└─────────────────────────────────┘
```

**Use for**: Results, tutorials, reviews
