# YouTube Skill - AI Assistant Instructions

## Purpose

This skill enables AI assistants to provide expert-level YouTube guidance for creators. Use this when helping with:

- YouTube strategy and content planning
- Title and thumbnail optimization
- Script writing, hooks, and retention
- Shorts strategy and production
- Analytics interpretation and optimization
- Monetization and growth strategies
- Content repurposing workflows

## Quick Reference

### When Asked About Titles

1. Check `titles/formulas.md` for 100+ proven templates
2. Apply psychology from `titles/title-psychology.md`
3. Ensure title + thumbnail synergy (see `titles/title-thumbnail-synergy.md`)
4. Keep titles under 60 characters (50 is better)
5. Front-load keywords in first 40 characters

**Title Formula Quick Picks:**
- Tutorial: "How to [Result] in [Timeframe] (Even If [Objection])"
- Listicle: "[Number] [Adjective] Ways to [Desired Outcome]"
- Story: "I [Did X] for [Time Period]. Here's What Happened."
- Controversy: "Why [Popular Thing] Is Actually [Contrarian Take]"

### When Asked About Thumbnails

1. Reference `thumbnails/frameworks.md` for proven patterns
2. Apply psychology from `thumbnails/thumbnail-psychology.md`
3. The 3-element rule: Face + Text + Visual = Click
4. High contrast, readable at small sizes
5. Emotion must match topic

**Thumbnail Checklist:**
- [ ] Clear focal point?
- [ ] Readable text (2-4 words max)?
- [ ] High contrast colors?
- [ ] Expressive face/emotion?
- [ ] Works at 120x68px?
- [ ] Complements (not duplicates) title?

### When Asked About Hooks/Retention

1. Use `retention/hooks.md` for 50+ hook examples
2. First 30 seconds are critical (see `retention/first-30-seconds.md`)
3. Pattern interrupt every 20-30 seconds
4. Structure templates in `retention/structure-templates.md`

**Hook Formula Quick Picks:**
- Promise: "By the end of this video, you'll know exactly how to..."
- Tease: "There's one thing that changed everything for me..."
- Challenge: "I bet you've been doing this wrong..."
- Story: "Last week, something happened that completely changed..."
- Proof: "I tested this for 30 days. The results shocked me."

### When Asked About Shorts

1. Different algorithm - see `shorts/shorts-algorithm.md`
2. Hook in first 1 second (not 3 seconds)
3. 30-60 seconds optimal for most content
4. Loop optimization matters
5. Use `shorts/shorts-that-perform.md` for ideas

### When Asked About Algorithm

1. Primary reference: `algorithm/how-youtube-works.md`
2. Signals explained: `algorithm/signals-deep-dive.md`
3. Key insight: YouTube optimizes for viewer satisfaction, not creator success
4. CTR + AVD = Primary ranking signals
5. Session time matters more than single video metrics

### When Asked About Analytics

1. Focus on actionable metrics only
2. Reference `analytics/metrics-that-matter.md`
3. Compare to benchmarks in README.md
4. Weekly optimization cycle in `analytics/optimization-cycle.md`

## Response Patterns

### For Title Requests

```
When generating titles:
1. Generate 10+ options minimum
2. Vary between curiosity, benefit, and controversy angles
3. Include specific numbers where relevant
4. A/B test suggestions (give pairs)
5. Explain WHY each works
```

### For Thumbnail Concepts

```
When suggesting thumbnails:
1. Describe the visual composition
2. Specify colors and contrast
3. Suggest text overlay (2-4 words)
4. Describe facial expression/emotion
5. Explain how it works with the title
```

### For Script/Hook Writing

```
When writing hooks:
1. Start with the strongest hook type for the topic
2. Include 3 variations
3. Add pattern interrupts throughout
4. Suggest B-roll moments
5. Mark retention optimization points
```

### For Strategy Questions

```
When advising on strategy:
1. Ask about current metrics first
2. Identify the bottleneck (CTR, AVD, impressions)
3. Reference relevant benchmarks
4. Give specific, actionable advice
5. Prioritize high-impact changes
```

## Key Metrics to Remember

| Metric | Poor | Average | Good | Excellent |
|--------|------|---------|------|-----------|
| CTR | <2% | 2-4% | 4-6% | 6-10%+ |
| AVD | <30% | 30-40% | 40-50% | 50-70%+ |
| Sub Rate | <0.5% | 0.5-1% | 1-2% | 2-5% |

## Common Scenarios

### "My CTR is low"

1. Review thumbnail (95% of CTR issues)
2. Check title clarity and curiosity
3. Compare to top-performing videos
4. A/B test with YouTube's built-in tool
5. Reference `thumbnails/` section

### "Viewers aren't staying"

1. Analyze retention graph drop-offs
2. Strengthen hook (see `retention/hooks.md`)
3. Add pattern interrupts
4. Cut fluff, increase pacing
5. Reference `retention/` section

### "Not getting impressions"

1. Check topic/keyword demand
2. Evaluate competition level
3. Consider Shorts for reach
4. Review posting consistency
5. Reference `seo/` section

### "Want to post more often"

1. Batch production workflow
2. Template systems
3. Team building (editor first)
4. Repurposing system
5. Reference `production/` section

## Do's and Don'ts

### DO:
- Give specific, actionable advice
- Reference benchmarks for context
- Explain the "why" behind recommendations
- Offer multiple options/variations
- Consider the creator's niche and audience

### DON'T:
- Give vague "just make good content" advice
- Ignore the specific metrics/data provided
- Recommend strategies without context
- Assume one-size-fits-all solutions
- Forget about title + thumbnail synergy
