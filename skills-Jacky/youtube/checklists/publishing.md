# Publishing Checklist

> A comprehensive checklist to ensure every video is published with maximum optimization.

## Pre-Upload Checklist

### Video File Quality
- [ ] Exported in correct resolution (1080p or 4K)
- [ ] Frame rate consistent (30 or 60fps)
- [ ] Audio levels normalized (-14 to -12 LUFS)
- [ ] No audio clipping or distortion
- [ ] Color grading applied and consistent
- [ ] No visible editing errors
- [ ] File format correct (MP4 recommended)

### Content Quality
- [ ] Hook is strong in first 30 seconds
- [ ] All promised content delivered
- [ ] Call-to-action included
- [ ] End screen space reserved (20 seconds)
- [ ] No copyrighted music/content without license
- [ ] Captions/subtitles accurate (if using)

---

## Metadata Checklist

### Title
- [ ] Under 60 characters (50 is ideal)
- [ ] Keywords front-loaded
- [ ] Curiosity or value clear
- [ ] No clickbait that can't be delivered
- [ ] Different from thumbnail text
- [ ] Proofread for typos

### Description
- [ ] Hook in first 2 lines (visible before "show more")
- [ ] Target keywords included naturally
- [ ] Timestamps added for videos over 5 minutes
- [ ] Links included and working
- [ ] Social media links present
- [ ] Hashtags added (3-5 relevant)
- [ ] Contact info included
- [ ] Proofread for errors

### Tags
- [ ] 8-15 relevant tags added
- [ ] Mix of broad and specific tags
- [ ] Include brand/channel name
- [ ] Target keywords included
- [ ] Competitor video tags researched

### Thumbnail
- [ ] 1280 x 720 pixels
- [ ] Under 2MB file size
- [ ] High contrast and readable
- [ ] Works at small sizes (mobile)
- [ ] Complements title (doesn't duplicate)
- [ ] Face has clear expression (if using)
- [ ] Text is minimal (2-4 words max)
- [ ] No misleading imagery

## YouTube Settings Checklist

### Basic Settings
- [ ] Visibility set correctly (public/scheduled/unlisted/private)
- [ ] Publish time optimized for audience
- [ ] Language set correctly
- [ ] Category selected
- [ ] License selected (Standard or Creative Commons)

### Advanced Settings
- [ ] Made for kids? Selected correctly
- [ ] Age restriction if necessary
- [ ] Paid promotion disclosed if applicable
- [ ] Altered/synthetic content disclosed if applicable
- [ ] Allow embedding? (Usually yes)
- [ ] Allow comments? Set preference
- [ ] Allow ratings? (Usually yes)

### Cards and End Screens
- [ ] Cards added at relevant moments
- [ ] Cards link to related content
- [ ] End screen elements added
- [ ] End screen links to relevant videos/playlists
- [ ] Subscribe element in end screen
- [ ] End screen elements don't block important content

### Playlist Assignment
- [ ] Video added to relevant playlist(s)
- [ ] Playlist is public
- [ ] Video position in playlist is logical

---

## Post-Publish Checklist

### Immediate (First Hour)
- [ ] Watch published video to confirm quality
- [ ] Check thumbnail displays correctly
- [ ] Verify all links work
- [ ] Pin a comment (question or key point)
- [ ] Share to primary social platforms
- [ ] Notify email list (if significant video)

### First 24 Hours
- [ ] Respond to early comments
- [ ] Answer questions
- [ ] Thank engaged viewers
- [ ] Monitor for issues/feedback
- [ ] Share to additional platforms
- [ ] Check analytics for early signals

### First Week
- [ ] Review analytics (CTR, AVD, traffic sources)
- [ ] Note what worked/didn't work
- [ ] A/B test thumbnail if CTR is low
- [ ] Continue comment engagement
- [ ] Add to relevant community posts
- [ ] Consider video card additions to older videos

## Quick Reference: Optimal Settings

### Video Specifications
| Setting | Optimal |
|---------|---------|
| Resolution | 1920x1080 (1080p) or higher |
| Frame rate | 30 or 60fps |
| Codec | H.264 |
| Container | MP4 |
| Audio | AAC, 48kHz, Stereo |
| Bitrate | 8-12 Mbps (1080p) |

### Thumbnail Specifications
| Setting | Requirement |
|---------|-------------|
| Size | 1280 x 720 pixels |
| Format | JPG, GIF, or PNG |
| Max file size | 2MB |
| Aspect ratio | 16:9 |

### Title Specifications
| Setting | Optimal |
|---------|---------|
| Length | 50-60 characters |
| Keywords | Front-loaded |
| Formatting | Title case or sentence case |

### Description Specifications
| Setting | Optimal |
|---------|---------|
| Length | 200-500 words |
| First 2 lines | Hook and keywords |
| Timestamps | For videos 5+ minutes |
| Hashtags | 3-5 relevant |

## Emergency Fixes

### After Publishing

**Found error in video**:
- Minor: Add correction in pinned comment
- Major: Unlist, fix, reupload (lose engagement)
- Critical: Delete and reupload

**Thumbnail not showing**:
- Wait 10-20 minutes (processing delay)
- Re-upload thumbnail
- Clear cache and refresh

**Wrong video uploaded**:
- Make private immediately
- Upload correct video
- Update all metadata

**Copyright claim received**:
- Review the claim details
- If music: Swap out the audio
- If content: Edit out or dispute if fair use
