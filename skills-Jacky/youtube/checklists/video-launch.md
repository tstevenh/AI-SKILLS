## Pre-Launch Preparation

### 24-48 Hours Before Upload

**Content preparation**:
- [ ] Video exported in highest quality (1080p minimum)
- [ ] Audio checked for quality issues
- [ ] No accidental copyrighted content
- [ ] Captions/subtitles ready (if using)
- [ ] End screens planned (what to link to)

**Packaging preparation**:
- [ ] Thumbnail created and tested at small size
- [ ] Alternative thumbnail(s) ready for A/B testing
- [ ] Title finalized and compelling
- [ ] Alternative titles noted (for testing)
- [ ] Description drafted

**Promotion preparation**:
- [ ] Social media posts drafted
- [ ] Email newsletter segment ready (if applicable)
- [ ] Community post planned (if applicable)
- [ ] Clips/Shorts identified for cross-promotion

### File Organization

**Folder structure for launch**:
```
[Video Title]/
├── export/
│   └── final_video.mp4
├── thumbnails/
│   ├── thumbnail_v1.png
│   ├── thumbnail_v2.png
│   └── thumbnail_v3.png
├── assets/
│   ├── description.txt
│   ├── title_options.txt
│   └── keywords.txt
└── promo/
    ├── twitter_post.txt
    ├── instagram_caption.txt
    └── clips/
        ├── short_1.mp4
        └── short_2.mp4
```

## Upload and Optimization

### YouTube Studio Upload Process

**Basic information**:
- [ ] Title entered (includes target keyword if applicable)
- [ ] Description added (structured properly)
- [ ] Thumbnail uploaded
- [ ] Playlist(s) selected

**Details tab**:
- [ ] Audience setting confirmed (Made for kids: Yes/No)
- [ ] Age restriction set if needed
- [ ] Paid promotion disclosed if applicable
- [ ] Tags added (if using)

**Monetization tab** (if monetized):
- [ ] Monetization enabled
- [ ] Ad placement reviewed
- [ ] Mid-roll ads positioned properly

**Video elements tab**:
- [ ] End screen added (subscribe + video)
- [ ] Cards added at relevant points
- [ ] Subtitles enabled/uploaded

**Visibility**:
- [ ] Visibility set (Public/Scheduled/Unlisted)
- [ ] Premiere option considered (if applicable)
- [ ] Schedule time optimal for audience

### Description Template

```
[Hook sentence - why should someone watch?]

[1-2 paragraphs explaining video value]

⏰ TIMESTAMPS:
0:00 - Intro
[X:XX] - [Section title]
[X:XX] - [Section title]
[X:XX] - [Section title]

🔗 LINKS MENTIONED:
[Resource 1]: [URL]
[Resource 2]: [URL]

📱 CONNECT WITH ME:
[Social 1]: [URL]
[Social 2]: [URL]

📧 BUSINESS INQUIRIES: [email]

🔥 RELATED VIDEOS:
[Video title]: [URL]
[Video title]: [URL]

#[keyword1] #[keyword2] #[keyword3]
```

### End Screen Strategy

**Optimal end screen setup**:
- Subscribe button (positioned for visibility)
- Best for viewer video (algorithm selected OR manually chosen)
- Recent upload or playlist

**End screen timing**:
- Start end screen 5-20 seconds before end
- Ensure nothing critical happens during end screen
- Video should naturally conclude before end screen elements

## Launch Day Actions

### Immediately After Publish

**First 30 minutes**:
- [ ] Verify video is live and accessible
- [ ] Watch first minute to ensure quality
- [ ] Check thumbnail is displaying correctly
- [ ] Review title/description for errors
- [ ] Confirm end screen is working

### Community Engagement

**YouTube engagement**:
- [ ] Pin a comment (question, summary, or first comment)
- [ ] Post community tab update (if available)
- [ ] Respond to early comments quickly

**Pinned comment ideas**:
- Question to encourage comments: "What's your experience with [topic]?"
- Summary of key points
- Link to related resource
- Behind-the-scenes insight
- Call to action

### Social Media Promotion

**Primary platforms** (in order of priority for you):
- [ ] Platform 1: [Post with link]
- [ ] Platform 2: [Post with clip + link]
- [ ] Platform 3: [Story/status with link]

**Social media post template**:
```
[Hook that creates curiosity]

[Brief context or key insight]

[Call to action + link]

[Relevant hashtags]
```

### Email Notification (if applicable)

- [ ] Send email to list (if video warrants)
- [ ] Include compelling subject line
- [ ] Brief description + direct link
- [ ] Clear call to action

## First 48 Hours

### Hour 1-6: Initial Momentum

**Actions**:
- [ ] Monitor initial views and engagement
- [ ] Respond to all comments (builds early engagement)
- [ ] Share to any remaining platforms
- [ ] Note any issues to fix

**What to watch**:
- CTR from impressions (is thumbnail working?)
- Early retention (is hook working?)
- Engagement rate (likes/comments per view)

### Hour 6-24: Evaluate and Adjust

**Performance check**:
- [ ] Review YouTube Studio analytics
- [ ] Check CTR compared to channel average
- [ ] Check early retention compared to similar videos
- [ ] Note initial viewer feedback

**Potential adjustments**:
- [ ] Consider title adjustment if CTR is low
- [ ] Consider thumbnail swap if CTR is very low
- [ ] Adjust description if needed
- [ ] Add timestamps if missing

### Hour 24-48: Double Down or Pivot

**If performing well**:
- [ ] Continue engagement in comments
- [ ] Share update post on social
- [ ] Consider boosting with additional Shorts
- [ ] Cross-promote from older videos

**If performing poorly**:
- [ ] Analyze what's not working
- [ ] Consider A/B test with new thumbnail
- [ ] Consider title refinement
- [ ] Learn for next video

## First Week

### Day 3-4: Mid-Week Check

**Analytics review**:
- [ ] Overall view velocity
- [ ] Subscriber conversions
- [ ] Traffic source breakdown
- [ ] Average view duration vs similar videos
- [ ] Comment sentiment analysis

**Engagement maintenance**:
- [ ] Continue responding to comments
- [ ] Heart thoughtful comments
- [ ] Address any questions or feedback

**Full performance analysis**:
- [ ] Compare to previous videos at same stage
- [ ] Identify what worked/didn't work
- [ ] Note thumbnail/title effectiveness
- [ ] Evaluate retention patterns

## Ongoing Optimization

### Week 2-4: Monitor and Adjust

**Regular checks**:
- [ ] Weekly performance trend
- [ ] Retention graph analysis
- [ ] Any negative feedback to address
- [ ] Opportunity for end screen updates

**Optimization opportunities**:
- [ ] Update end screen to link to newer content
- [ ] Add to relevant playlists
- [ ] Embed on website/blog (if applicable)
- [ ] Create Shorts from video moments

### Month 2+: Long-Term Performance

**Periodic reviews**:
- [ ] Monthly check on evergreen performance
- [ ] Update description if information changes
- [ ] Update cards/end screens as needed
- [ ] Consider thumbnail refresh for older videos

**Evergreen content optimization**:
- Update titles for current year if applicable
- Refresh thumbnails periodically
- Add new related video links
- Improve SEO based on search data

## Quick Reference Checklists

### Minimum Viable Launch (5 minutes)

**Must-do for every video**:
- [ ] Title optimized
- [ ] Thumbnail uploaded
- [ ] Description with timestamps
- [ ] Made for Kids set correctly
- [ ] Visibility set

### Standard Launch (15-20 minutes)

**Includes minimum viable plus**:
- [ ] End screen configured
- [ ] Cards added
- [ ] Pinned comment posted
- [ ] One social media share

### Full Launch (45-60 minutes)

**Includes standard plus**:
- [ ] Multiple social media posts
- [ ] Email notification (if applicable)
- [ ] Community post
- [ ] All comments responded to (first hour)
- [ ] Clips/Shorts scheduled
- [ ] Analytics check (first few hours)

### Critical Launch (Major Videos)

**For your most important content**:
- [ ] Everything in full launch
- [ ] Premiere consideration
- [ ] Email sequence prepared
- [ ] Paid promotion considered
- [ ] PR/influencer outreach
- [ ] Blog post companion content
- [ ] Multiple thumbnail variations ready
- [ ] Dedicated time for first 24-hour engagement

## Launch Timing Considerations

### Best Times to Publish

**Factors to consider**:
- When your audience is online (check analytics)
- Time zone of majority audience
- Competition (other creators in niche)
- Day of week patterns

**General patterns** (varies by niche):
- Weekday mornings: Educational content
- Weekday evenings: Entertainment
- Weekends: Longer-form content
- Sunday evening: Week planning content

### Premiere vs Regular Publish

**Use Premiere when**:
- Major announcement or milestone
- Community event experience wanted
- Real-time engagement valuable
- Building hype matters

**Regular publish when**:
- Consistent content calendar
- Search-focused content
- Not time-sensitive
- Maximum convenience

## Common Launch Mistakes

### Mistakes to Avoid

**1. Publishing without checking**
- Always watch first minute after publish
- Verify thumbnail displays correctly
- Check for upload errors

**2. Abandoning the video after publish**
- First few hours are critical
- Engagement signals matter
- Community building opportunity

**3. Not having promotion ready**
- Prepare promotion materials before publish
- Have social posts drafted
- Email ready to send

**4. Ignoring early data**
- Monitor first 48-hour performance
- Be willing to adjust title/thumbnail
- Learn from every launch

**5. Same launch for every video**
- Major videos deserve major launches
- Minor videos need minimum viable launch
- Match effort to opportunity

*A good launch maximizes your video's chance. A great launch compounds over your content library. Launch every video intentionally.*
