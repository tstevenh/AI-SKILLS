# Pre-Production Checklist

> Complete this checklist before filming any video.

## Content Planning

### Concept Validation
- [ ] Topic has search demand or browse potential
- [ ] Topic fits within content pillars
- [ ] Video idea is differentiated from existing content
- [ ] Target audience is clearly defined
- [ ] Success metrics are established

### Research Complete
- [ ] Topic thoroughly researched
- [ ] Competitor videos analyzed
- [ ] Key points identified
- [ ] Examples/case studies gathered
- [ ] Statistics/data sourced

### Script/Outline
- [ ] Hook written (first 30 seconds)
- [ ] Main points outlined
- [ ] Transitions planned
- [ ] CTA determined
- [ ] Timestamps estimated
- [ ] Script reviewed for clarity

---

## Thumbnail Planning

### Concept
- [ ] Thumbnail concept sketched
- [ ] Complements title (doesn't duplicate)
- [ ] Works at small sizes (mobile)
- [ ] Emotion/expression planned
- [ ] Text decided (2-4 words max)

### Assets
- [ ] Photo/shoot planned
- [ ] Background determined
- [ ] Props gathered if needed
- [ ] Reference thumbnails saved

---

## Title Planning

### Title Options
- [ ] 5+ title options brainstormed
- [ ] Primary keywords included
- [ ] Under 60 characters
- [ ] Curiosity or value clear
- [ ] Title + thumbnail synergy confirmed

---

## Technical Setup

### Equipment
- [ ] Camera charged and memory cleared
- [ ] Lens clean
- [ ] Audio equipment tested
- [ ] Backup batteries available
- [ ] Memory cards formatted

### Environment
- [ ] Lighting tested and positioned
- [ ] Background checked and cleaned
- [ ] Acoustic environment checked
- [ ] "Recording" sign posted if needed
- [ ] Phone silenced and notifications off

### Settings
- [ ] Camera settings confirmed (resolution, frame rate)
- [ ] Audio levels tested
- [ ] White balance set
- [ ] Focus tested

---

## Content Assets

### Visual Assets
- [ ] B-roll shot list created
- [ ] Screen recordings prepared
- [ ] Images/graphics sourced
- [ ] Props ready

### Audio Assets
- [ ] Music selected (if needed)
- [ ] Sound effects identified
- [ ] Voiceover script ready (if needed)

### Information
- [ ] Links to resources ready
- [ ] Affiliate links prepared
- [ ] Sponsor integration points marked (if applicable)

---

## Personal Preparation

### Ready to Film
- [ ] Script/outline memorized or on teleprompter
- [ ] Wardrobe selected (consider multiple videos)
- [ ] Makeup/grooming done (if applicable)
- [ ] Energy level checked (hydrated, rested)
- [ ] Vocal warm-up complete

---

## Final Confirmation

### Pre-Flight Check
- [ ] All sections above complete
- [ ] Video brief fully filled out
- [ ] Team notified of filming schedule
- [ ] Time blocked for filming session
- [ ] Backup plan if something goes wrong

---

## Quick Reference: What to Skip vs Never Skip

### OK to Skip Sometimes
- Professional makeup
- Elaborate B-roll plan
- Music selection
- Perfect lighting

### Never Skip
- Script/outline
- Thumbnail concept
- Title planning
- Equipment test
- Hook planning

---

*Complete this checklist 24+ hours before planned filming time.*
