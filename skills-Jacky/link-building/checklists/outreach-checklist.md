# Outreach Checklist

## Pre-Outreach Preparation

### Prospect Verification
- [ ] Prospect is in "Outreach Queue" status
- [ ] Contact information is complete
- [ ] Qualification score documented
- [ ] No previous outreach to this prospect

### Research Completed
- [ ] Read at least one recent article on site
- [ ] Noted specific personalization point
- [ ] Understand their audience
- [ ] Know submission process (if applicable)

### Campaign Requirements
- [ ] Target URL confirmed
- [ ] Anchor text strategy clear
- [ ] Budget approved (if paid placement)
- [ ] Content available (if guest post)

---

## Email Quality Checks

### Before Sending Every Email

#### Accuracy
- [ ] Recipient name correct
- [ ] Site name correct
- [ ] Article/page reference accurate
- [ ] All URLs correct and working
- [ ] Dates accurate (if mentioned)

#### Personalization
- [ ] Personalized opening (not generic)
- [ ] Specific reference to their content
- [ ] Relevant pitch for their audience
- [ ] Shows familiarity with site

#### Content Quality
- [ ] Grammar and spelling checked
- [ ] Professional tone
- [ ] Concise (under 200 words)
- [ ] Clear call-to-action
- [ ] No promotional language

#### Technical
- [ ] Correct email address
- [ ] Subject line appropriate
- [ ] Signature complete
- [ ] No tracking links visible (if applicable)

---

## Sending Best Practices

### Timing
- [ ] Sending Tuesday-Thursday (preferred)
- [ ] Morning hours (9-11 AM recipient time)
- [ ] Not Monday morning or Friday afternoon
- [ ] Not during major holidays

### Volume Management
- [ ] Under daily send limit (50/account)
- [ ] Spread throughout day
- [ ] Not bulk sending rapidly

### Post-Send
- [ ] Updated status to "Contacted"
- [ ] Added date contacted
- [ ] Scheduled follow-up (5-7 days)
- [ ] Logged in CRM/spreadsheet

---

## Follow-Up Checklist

### Follow-Up Timing
- [ ] First follow-up: Day 5-7
- [ ] Second follow-up: Day 12-14
- [ ] Final follow-up: Day 18-21 (if used)
- [ ] Stop after maximum 3 attempts

### Follow-Up Quality
- [ ] Shorter than original email
- [ ] References original pitch
- [ ] Adds new value if possible
- [ ] Respectful tone
- [ ] Clear but not pushy

### After Follow-Ups
- [ ] Updated status appropriately
- [ ] Documented outcome
- [ ] Scheduled re-approach (if applicable)

---

## Response Handling

### Positive Response
- [ ] Responded within 24 hours
- [ ] Provided requested information
- [ ] Clarified next steps
- [ ] Updated status to "In Progress"
- [ ] Documented agreement details

### Negotiation
- [ ] Evaluated against campaign budget
- [ ] Counter-offered if needed
- [ ] Documented final agreement
- [ ] Confirmed all details in writing

### Negative Response
- [ ] Thanked them professionally
- [ ] Asked for alternative suggestions (optional)
- [ ] Updated status to "Rejected"
- [ ] Noted reason for future reference

### No Response
- [ ] Completed follow-up sequence
- [ ] Updated status to "No Response"
- [ ] Scheduled for potential re-approach (6+ months)

---

## Campaign Completion

### Link Secured
- [ ] All details confirmed
- [ ] Handed off to content/next step
- [ ] Updated status to "Accepted"
- [ ] Documented in reporting

### Quality Control
- [ ] Link verified live (see QA checklist)
- [ ] Updated status to "Live"
- [ ] Added to monitoring list
- [ ] Reported to client/campaign
