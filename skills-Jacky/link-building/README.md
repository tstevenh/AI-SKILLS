# Link Building

Comprehensive link building encyclopedia for acquiring high-quality backlinks at scale.

## What It Does

This skill teaches systematic link acquisition strategies—from guest posting and niche edits to digital PR and broken link building. Build authority and improve search rankings through quality backlinks.

## Features

- Guest posting strategy and outreach
- Niche edits (link insertions) methodology
- HARO and digital PR tactics
- Resource page link building
- Broken link building
- Linkable asset creation
- Local link building opportunities
- Link quality assessment and metrics
- Team structure and operations

## Installation

1. Copy the `link-building/` folder into your OpenClaw skills directory
2. Restart OpenClaw

## Usage

Use this skill when:
- Planning link building campaigns
- Qualifying link opportunities
- Writing outreach emails
- Creating linkable content assets
- Managing link building teams
- Pricing link building services
- Tracking backlink acquisition

## Requirements

- SEO tools (Ahrefs, SEMrush, or Majestic)
- Outreach platform or email tool
- Content creation resources
- Link tracking spreadsheet or software
- Budget for content and placements
