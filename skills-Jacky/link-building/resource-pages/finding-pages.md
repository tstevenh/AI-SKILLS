# Finding Resource Pages

## What Are Resource Pages?

Resource pages are curated lists of links to helpful resources on specific topics. They exist specifically to link out to valuable content, making them ideal link building targets.

**Common formats:**
- "Recommended Resources"
- "Useful Tools"
- "Further Reading"
- "Best [Topic] Websites"
- Industry directories
- Educational resource lists

### Why Resource Pages Are Valuable

**High intent to link:**
These pages exist to link out. The curator is actively looking for resources to include.

**Often well-maintained:**
Curators update and maintain these pages, suggesting quality standards.

**Permanent placements:**
Once included, links tend to stay (unlike guest post sites that may remove content).

**Relevant context:**
Links appear among other relevant resources, strengthening topical signals.

## Competitor Analysis for Resource Pages

### Finding Competitors' Resource Page Links

**Step 1: Export competitor backlinks**
Using Ahrefs or SEMrush, pull competitor backlink profile.

**Step 2: Filter for resource pages**
Look for referring URLs containing:
- /resources
- /links
- /recommended
- /directory

**Step 3: Identify resource-style pages**
Review anchor text and surrounding text for resource indicators:
- Listed among other resources
- Generic anchor text ("Click here," site name)
- Part of a list of sites

**Step 4: Evaluate and add to prospect list**

### The Link Intersect Method

Find resource pages linking to multiple competitors:

1. Use Ahrefs Link Intersect
2. Enter 3-5 competitors
3. Find domains linking to multiple competitors
4. Check if these are resource pages
5. If they link to competitors, they may link to you

## Evaluating Resource Pages

### Quality Indicators

**Good resource pages:**
- Curated, not auto-generated
- Updated recently
- Limited number of links (not hundreds)
- Relevant to your niche
- On a quality site (DR 20+)
- Shows signs of human curation

**Red flags:**
- Hundreds of links with no organization
- "Submit your link" forms
- Auto-generated from feeds
- No recent updates
- Mixed with spam/irrelevant sites
- Clearly abandoned

### Evaluation Checklist

**Page quality:**
✓ Is the page curated by a human?
✓ Are the existing links quality?
✓ Is the content organized logically?
✓ Has it been updated recently?
✓ Is your resource a natural fit?

**Site quality:**
✓ DR 20+ (or highly relevant)
✓ Real organic traffic
✓ Established site with history
✓ No spam signals

**Fit:**
✓ Your resource matches the page's theme
✓ Your resource adds value to the list
✓ Your resource is comparable to others listed

## Niche-Specific Resource Page Hunting

### Educational Resources (.edu)

Universities and schools maintain resource pages for students:

**Search patterns:**
```
site:.edu + [your topic] + resources
site:.edu + "recommended websites" + [your topic]
site:.edu + "student resources" + [your topic]
site:.edu + "research resources" + [your topic]

```

**Departments to target:**
- Library resource pages
- Department-specific pages
- Student services
- Research centers

### Government Resources (.gov)

Government sites often link to industry resources:

**Search patterns:**
```
site:.gov + [your topic] + resources
```

## Building Your Resource Page Pipeline

### Prospecting Workflow

**Weekly routine:**

**Monday: Search operator prospecting (2 hours)**
- Run 10-15 search operator queries
- Add results to spreadsheet
- Initial quality assessment

**Tuesday: Competitor analysis (1 hour)**
- Review 1-2 competitor profiles for resource links
- Add new finds to spreadsheet

**Wednesday: Evaluation (1 hour)**
- Deep evaluation of top prospects
- Move qualified opportunities to outreach queue

### Tracking Spreadsheet

| Column | Data |
|--------|------|
| Resource Page URL | Target page |
| Domain | Root domain |
| DR | Domain rating |
| Page Topic | What the page is about |
| Existing Links | How many, what type |
| Last Updated | When page was refreshed |
| Contact | Who manages the page |
| Your Resource | What you'll pitch |
| Status | Pipeline stage |
| Notes | Additional context |

### Volume Expectations

**Realistic numbers:**
- Search prospecting: 20-30 candidates/hour
- After quality filter: 5-10 viable prospects/hour
- Outreach success rate: 5-15%

**For 10 resource page links/month:**
- Prospect 100-200 pages
- Qualify 30-50 for outreach
- Reach 10-15 successes

## Summary

Finding resource pages requires:

1. **Search operators** - 50+ variations for thorough coverage
2. **Competitor analysis** - Find where competitors are listed
3. **Quality evaluation** - Not all resource pages are equal
4. **Niche targeting** - .edu, .gov, and industry-specific pages
5. **Systematic prospecting** - Build ongoing pipeline

With qualified resource pages identified, the next step is crafting effective outreach to earn inclusion.
