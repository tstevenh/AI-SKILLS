# Link Builder Onboarding Program

## Week 1: Foundation

### Day 1: Company and Team

**Morning:**
- Team introductions
- Company overview
- Link building team's role
- Culture and values

**Afternoon:**
- Workspace setup (accounts, tools, access)
- Tool access provisioning
- Communication channels (Slack, email)
- Documentation access

**Deliverables:**
- All tool access confirmed
- Introductions completed
- First week schedule reviewed

### Day 2: Link Building Fundamentals

**Morning:**
- Why links matter (study fundamentals chapter)
- Link metrics overview
- What makes a good link
- Ethics and risk

**Afternoon:**
- Self-study: Fundamentals section
- Quiz on fundamentals
- Q&A with manager

**Deliverables:**
- Complete fundamentals reading
- Pass fundamentals quiz (80%+)

### Day 3: Tools Training

**Morning:**
- Ahrefs walkthrough
  - Site Explorer
  - Backlinks analysis
  - Content Explorer

**Afternoon:**
- Email tools training
  - Hunter.io
  - Outreach platform (Pitchbox/BuzzStream)
- Spreadsheet systems
- Tracking processes

**Deliverables:**
- Demonstrate tool proficiency
- Complete tool exercises

### Day 4: Process Overview

**Morning:**
- Prospecting SOP walkthrough
- Outreach SOP walkthrough
- QA SOP walkthrough

**Afternoon:**
- Shadow experienced team member
- Observe prospecting session
- Observe outreach session

**Deliverables:**
- Understand core SOPs
- Document questions for discussion

### Day 5: Week 1 Review

**Morning:**
- Review week 1 learning
- Clarify questions
- Identify areas needing more focus

**Afternoon:**
- Begin supervised prospecting
- Manager feedback session
- Set Week 2 goals

**Deliverables:**
- Week 1 checklist complete
- Week 2 plan agreed

## Week 2: Hands-On Practice

### Day 6-7: Prospecting Practice

**Focus:** Finding and qualifying opportunities

**Activities:**
- Run search operator queries
- Evaluate site quality
- Build prospect lists
- Review with mentor

**Target:** 50+ qualified prospects added

### Day 8-9: Outreach Practice

**Focus:** Email crafting and sending

**Activities:**
- Write outreach emails (reviewed before sending)
- Personalization practice
- Send supervised outreach
- Handle initial responses

**Target:** 20+ outreach emails sent

### Day 10: Week 2 Review

**Activities:**
- Review prospecting work
- Review outreach quality
- Feedback on emails sent
- Adjust approach based on results

**Deliverables:**
- Quality assessment of work
- Improvement plan documented

## Week 3-4: Increasing Independence

### Week 3: Supervised Production

**Activities:**
- Full prospecting with daily review
- Outreach with reduced oversight
- Begin handling responses
- Track metrics

**Targets:**
- 100+ prospects/week
- 40+ outreach emails/week
- Meeting quality standards

**Support:**
- Daily check-ins with mentor
- Email review before key sends
- Regular feedback

### Week 4: Semi-Independent Work

**Activities:**
- Independent prospecting (spot-check review)
- Independent outreach (response review)
- Full follow-up management
- Reporting contribution

**Targets:**
- 100+ prospects/week
- 50+ outreach emails/week
- Quality standards maintained
- Self-managing pipeline

**Support:**
- Weekly 1:1 with manager
- Spot checks on quality
- Available for questions

## 30-Day Review

### Performance Assessment

**Metrics review:**
- Prospects generated
- Outreach volume
- Response rate
- Quality pass rate
- Links secured (if applicable)

**Skills assessment:**
- Tool proficiency
- Prospecting accuracy
- Outreach quality
- Communication
- Independence level

### Career Path Discussion

- Strengths identified
- Areas for development
- 60-day goals
- Training needs
- Growth opportunities

### Decision Point

- Full team integration (typical)
- Extended training (if needed)
- Role adjustment (if better fit elsewhere)
- Separation (rare, if fundamental issues)

### Practice Exercises

**Exercise 1: Site Evaluation**
Given 20 sites, evaluate each for link opportunity quality.
Pass: 80%+ accuracy vs. answer key.

**Exercise 2: Outreach Drafting**
Write 10 outreach emails for different scenarios.
Pass: Manager approval on 8+.

**Exercise 3: Prospecting Challenge**
Find 50 link opportunities in unfamiliar niche.
Pass: 40+ meet quality standards.

## Buddy/Mentor Program

### Mentor Responsibilities

- Daily check-ins during Week 1-2
- Answer questions
- Review work
- Provide feedback
- Escalate issues

### Mentee Expectations

- Ask questions early
- Accept feedback gracefully
- Take initiative
- Document learnings
- Respect mentor's time

### Mentor Selection

Choose mentors who are:
- Experienced (6+ months)
- Good communicators
- Patient teachers
- Positive attitude
- Time to dedicate

## Common Onboarding Issues

### Issue: Slow to Adopt Tools

**Signs:** Avoiding tools, asking basic questions repeatedly
**Solution:** Additional tool training, pair work until comfortable

### Issue: Quality Concerns

**Signs:** Low pass rate on site evaluation, poor emails
**Solution:** More practice exercises, frequent feedback, extended review period

### Issue: Volume Concerns

**Signs:** Missing targets consistently
**Solution:** Time management coaching, process review, expectation alignment

### Issue: Cultural Fit

**Signs:** Communication issues, attitude problems
**Solution:** Address directly, provide clear feedback, escalate if needed

## Summary

Effective onboarding:

1. **Structured program** - Clear daily/weekly plan
2. **Foundational knowledge** - Before hands-on work
3. **Supervised practice** - Learn by doing with guidance
4. **Gradual independence** - Increase autonomy over time
5. **Regular feedback** - Continuous improvement
6. **Clear milestones** - Know what success looks like

Investment in onboarding pays dividends through better performance and lower turnover.
