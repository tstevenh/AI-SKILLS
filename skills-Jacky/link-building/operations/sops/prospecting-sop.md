# Prospecting SOP

## Purpose

This SOP defines the standard process for finding and qualifying link building opportunities. Following this process ensures consistent quality and efficient prospecting.

## Scope

Applies to: All prospectors and link builders conducting prospecting activities.

## Process Overview

```
1. Define Targets → 2. Find Opportunities → 3. Qualify → 4. Add to Database → 5. Handoff
```

## Step 1: Define Targets

### Before Prospecting

**Understand the campaign:**
- Client/project niche
- Target keywords
- Quality requirements (min DR, traffic)
- Link types needed (guest post, niche edit, etc.)
- Any restrictions or preferences

**Document requirements:**
```
Campaign: [Name]
Niche: [Industry]
Min DR: [X]
Min Traffic: [X]
Link Types: [Guest post, niche edit, etc.]
Restrictions: [Any sites to avoid, etc.]
```

## Step 2: Find Opportunities

### Method A: Search Operator Prospecting

**For guest posts:**
1. Open Google
2. Run queries:
   - `[keyword] + "write for us"`
   - `[keyword] + "guest post"`
   - `[keyword] + inurl:write-for-us`
3. Open results in new tabs (10-20 at a time)
4. Quick visual assessment
5. Move promising sites to qualification

**For niche edits:**
1. Run queries:
   - `[keyword] + "best tools"`
   - `[keyword] + "resources"`
   - `[keyword] + "complete guide"`
2. Open results
3. Identify relevant existing content
4. Move promising pages to qualification

### Method B: Competitor Analysis

1. Open Ahrefs Site Explorer
2. Enter competitor domain
3. Go to Backlinks → Referring Domains
4. Filter by DR (minimum threshold)
5. Sort by DR descending
6. Review for guest post/niche edit indicators
7. Export promising opportunities

### Method C: Content Explorer

1. Open Ahrefs Content Explorer
2. Enter topic keywords
3. Filter by Referring Domains (10+)
4. Review for guest post opportunities
5. Add promising sites to list

### Time Allocation

| Method | Time/Week | Expected Output |
|--------|-----------|-----------------|
| Search Operators | 3-4 hours | 50-75 prospects |
| Competitor Analysis | 2-3 hours | 30-50 prospects |
| Content Explorer | 1-2 hours | 20-30 prospects |

## Step 3: Qualify Opportunities

### Quick Filter (30 seconds per site)

**Check:**
1. DR (Ahrefs toolbar) → Above minimum?
2. Traffic (quick check) → Has real traffic?
3. Language → Appropriate for campaign?
4. Obvious spam → Any red flags?

**If any fail → Skip immediately**

### Full Qualification (5 minutes per site)

**Site-level checks:**
- [ ] DR meets threshold
- [ ] Organic traffic validates authority
- [ ] Content is quality
- [ ] Site is active (recent posts)
- [ ] No spam signals

**Opportunity-level checks:**
- [ ] Accepts guest posts OR has content for niche edits
- [ ] Relevant to campaign niche
- [ ] Editorial standards exist
- [ ] Contact findable

**Scoring:**
Rate 1-5 on each factor. Total score 15+ = qualified.

| Factor | Score |
|--------|-------|
| DR/Traffic | /5 |
| Relevance | /5 |
| Quality | /5 |
| Feasibility | /5 |
| **Total** | /20 |

### Qualification Criteria by Tier

**Premium (18-20):** High priority, pursue first
**Standard (15-17):** Good opportunities
**Acceptable (12-14):** Pursue if pipeline needs
**Skip (<12):** Don't add to database

## Step 4: Add to Database

### Required Fields

For each qualified opportunity:

```
Domain: [root domain]
URL: [specific page for niche edit, or submission page for guest post]
DR: [number]
Organic Traffic: [estimate]
Relevance Score: [1-5]
Opportunity Type: [Guest post / Niche edit / Resource page / etc.]
Contact Email: [if found]
Contact Name: [if found]
Contact Method: [Email / Form / Social]
Submission Guidelines: [URL if applicable]
Notes: [Any relevant details]
Status: [New]
Date Added: [Date]
```

### Finding Contact Information

**Priority order:**
1. Editorial/submissions contact on site
2. Author email from relevant content
3. Hunter.io domain search
4. LinkedIn search for content manager
5. General contact form (last resort)

### Database Organization

**Use status tracking:**
- New: Just added, not contacted
- Outreach Queue: Ready for outreach
- Contacted: Outreach sent
- In Progress: Active conversation
- Accepted: Placement confirmed
- Live: Link is live
- Rejected: They declined
- No Response: No response after follow-ups

## Step 5: Handoff to Outreach

### Before Handoff

Verify:
- [ ] All required fields populated
- [ ] Contact information validated
- [ ] Qualification score documented
- [ ] Any special notes included

### Handoff Process

1. Move qualified prospects to "Outreach Queue" status
2. Notify outreach team (Slack/email)
3. Weekly sync on pipeline status
4. Address any questions about prospects

## Quality Standards

### Minimum Requirements

All prospects must have:
- DR above campaign minimum
- Real organic traffic
- Relevance to campaign
- Findable contact

### Rejection Criteria

Automatic rejection:
- DR below threshold
- Zero organic traffic
- Obvious spam/PBN
- No editorial standards
- Competing with client's other properties

### Quality Metrics

**Prospector KPIs:**
- Prospects added per week
- Qualification accuracy (spot-checked)
- Database completeness
- Pipeline health

**Benchmarks:**
- 75+ qualified prospects per week (full-time)
- 90%+ accuracy on qualification
- 95%+ completeness of required fields

## Common Mistakes to Avoid

### Mistake: Ignoring traffic

**Problem:** High DR but no traffic
**Solution:** Always verify traffic alongside DR

### Mistake: Weak relevance

**Problem:** Sites that don't match niche
**Solution:** Relevance should be 3+ on 5-point scale

### Mistake: Incomplete data

**Problem:** Missing contact info or details
**Solution:** Complete all fields before adding

### Mistake: Duplicate entries

**Problem:** Same site added multiple times
**Solution:** Check database before adding

### Mistake: Low volume

**Problem:** Not meeting prospect targets
**Solution:** Diversify methods, allocate more time

## Tools Required

- Browser extensions (Ahrefs toolbar, Check My Links)
- Hunter.io (contact finding)
- Spreadsheet/Database (tracking)
- Communication tool (Slack/email)
