- Pricing link building services
- Building link building teams

## Quick Reference

### Link Quality Thresholds (Indexsy Standards)
| Tier | DR | Traffic | Price Range |
|------|-----|---------|-------------|
| Premium | 70+ | 50k+ | $300-800 |
| High | 50-69 | 10k-50k | $150-300 |
| Medium | 30-49 | 1k-10k | $75-150 |
| Entry | 20-29 | 500-1k | $40-75 |

### Anchor Text Distribution (Safe Ratios)
- Branded: 40-50%
- Naked URL: 20-25%
- Generic: 15-20%
- Exact Match: 5-10%
- Partial Match: 10-15%

### Response Rate Benchmarks
- Cold outreach: 3-8%
- Warm outreach: 15-25%
- Follow-up impact: +50-70% responses

## How to Use This Skill

1. **New to link building?** Start with `fundamentals/`
2. **Building a team?** Focus on `operations/`
3. **Need templates?** Check `templates/all-outreach.md`
4. **Specific tactic?** Navigate to that directory
5. **Quality control?** Use `checklists/`

## Key Principles

1. **Quality > Quantity** - One DR70 link beats ten DR20 links
2. **Relevance is King** - Topical relevance amplifies link value
3. **Diversify Sources** - No single tactic should dominate
4. **Track Everything** - What gets measured gets improved
5. **Relationships Matter** - Long-term publisher relationships scale better than cold outreach
