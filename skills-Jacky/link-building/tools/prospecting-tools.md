# Prospecting Tools

## Tool Selection Philosophy

The right tools depend on your scale and budget. Expensive tools don't guarantee results—they save time and provide data. Start with what you can afford and upgrade as volume justifies cost.

## Ahrefs: Full Walkthrough

### Overview

**What it is:** All-in-one SEO tool with industry-leading backlink data
**Pricing:** $99-999/month
**Best for:** Professional link builders, agencies, serious SEO

### Key Features for Link Building

**Site Explorer:**
Core tool for analyzing any website's backlink profile.

**How to use:**
1. Enter target domain
2. Review Overview metrics
3. Explore Backlinks section
4. Analyze Referring Domains
5. Check Top Pages by links

**Content Explorer:**
Find content on any topic with backlink data.

**How to use:**
1. Enter topic keywords
2. Filter by referring domains (find link-worthy content)
3. Sort by date (find fresh content)
4. Analyze what earns links in your niche

**Link Intersect:**
Find sites linking to competitors but not you.

**How to use:**
1. Enter your domain
2. Enter 3-5 competitor domains
3. See domains linking to them, not you
4. These are proven link opportunities

### Ahrefs Prospecting Workflows

**Workflow 1: Competitor backlink mining**

1. Site Explorer → Competitor domain
2. Backlinks → One link per domain
3. Filter by DR (your minimum)
4. Sort by Domain Rating
5. Export to spreadsheet
6. Review for guest post/niche edit opportunities

**Workflow 2: Finding guest post opportunities**

1. Content Explorer → "[your topic] guest post"
2. Filter by Referring Domains (10+)
3. Review results for sites accepting guest posts
4. Check site quality
5. Add to outreach list

**Workflow 3: Broken link finding**

1. Site Explorer → Target domain
2. Backlink profile → Broken backlinks
3. Filter by DR of linking domains
4. Identify relevant opportunities
5. Create replacement content

### Ahrefs Tips and Tricks

**Save searches:** Create saved searches for ongoing monitoring
**Bulk analysis:** Use batch analysis for multiple domains
**Alerts:** Set up new backlink alerts for competitors
**Exports:** Export to Google Sheets for team collaboration

### Limitations

- Expensive for beginners
- Can miss smaller/newer sites
- Not a complete picture (no tool has all links)
- Learning curve for full utilization

## Semrush: Full Walkthrough

### Overview

**What it is:** Comprehensive marketing tool with strong SEO features
**Pricing:** $120-450/month
**Best for:** Marketers who need more than just link building

### Key Features for Link Building

**Backlink Analytics:**
Similar to Ahrefs Site Explorer for backlink analysis.

**How to use:**
1. Enter target domain
2. Review Overview metrics
3. Explore backlinks and referring domains
4. Analyze anchor text distribution
5. Check authority score

**Backlink Gap:**
Find competitors' links you don't have.

**How to use:**
1. Enter your domain and competitors
2. Identify domains linking to competitors, not you
3. Prioritize by authority score
4. Research for opportunities

**Link Building Tool:**
Built-in outreach management.

**How to use:**
1. Add prospects from research
2. Find contact information
3. Track outreach status
4. Manage campaigns

### Semrush vs. Ahrefs

| Feature | Semrush | Ahrefs |
|---------|---------|--------|
| Backlink database | Large | Larger |
| Update frequency | Daily | Every 15-30 min |
| Content analysis | Strong | Stronger |
| Built-in outreach | Yes | No |
| Keyword research | Strong | Strong |
| PPC data | Excellent | Limited |
| Price | Higher | Lower |

### When to Use Semrush

- You need all-in-one marketing tool
- You want built-in outreach features
- You rely on PPC data
- Your team already uses it

## Moz: Full Walkthrough

### Overview

**What it is:** Original SEO tool with focus on metrics and education
**Pricing:** $99-599/month
**Best for:** Beginners, smaller operations, DA-focused reporting

### Key Features for Link Building

**Link Explorer:**
Backlink analysis tool with Moz's DA/PA metrics.

**How to use:**
1. Enter target URL or domain
2. Review Domain Authority
3. Explore linking domains
4. Analyze anchor text
5. Check spam score

**Link Intersect:**
Find links competitors have that you don't.

**How to use:**
1. Enter your site and competitors
2. See "compete" sites linking to them
3. Research opportunities

### Moz Metrics

**Domain Authority (DA):**
- Predicts ranking ability
- 0-100 scale
- Well-known metric (clients understand)
- Updated monthly

**Page Authority (PA):**
- Page-level authority
- Useful for page evaluation
- Similar to Ahrefs URL Rating

**Spam Score:**
- Identifies potentially spammy links
- Helps with link cleanup
- Useful for qualification

### Limitations

- Smaller link index than Ahrefs/Semrush
- Less frequent updates
- Fewer advanced features
- DA can be misleading (easily manipulated)

### When to Use Moz

- Clients prefer DA metric
- Budget is constrained
- Simpler needs
- Educational resources valued

## Email Finding Tools

### Hunter.io

**What it does:** Finds email addresses for domains
**Pricing:** Free tier (25 searches), $49-399/month

**How to use:**
1. Domain Search: Enter domain, see all found emails
2. Email Finder: Enter name + domain, find specific email
3. Email Verifier: Check if email is valid

**Best practices:**
- Verify before sending
- Check confidence score
- Try multiple formats if not found

### Snov.io

**What it does:** Email finding + outreach automation
**Pricing:** Free tier (50 credits), $39-199/month

**Key features:**
- Domain search
- LinkedIn extension
- Email drip campaigns
- Email verification

**Best for:** LinkedIn prospecting, outreach automation

### VoilaNorbert

**What it does:** Name + domain → email
**Pricing:** $49-499/month

**Best for:** High-accuracy individual lookups

### FindThatLead

**What it does:** Email finding with social integration
**Pricing:** $49-399/month

**Best for:** Social media prospecting

### Email Finding Best Practices

1. **Use multiple tools** - No tool finds everything
2. **Always verify** - Invalid emails hurt deliverability
3. **Try patterns** - first@, first.last@, etc.
4. **Check LinkedIn** - Often has contact info
5. **Use website** - Many contacts are on About/Contact pages

## Browser Extensions

### Essential Extensions for Link Building

**SEOquake:**
- Free
- Quick metrics overlay on search results
- Shows DA, PA, link counts

**Ahrefs SEO Toolbar:**
- Requires Ahrefs subscription
- Shows metrics on any page
- Quick backlink data

**Moz Bar:**
- Free/premium
- DA/PA on any page
- SERP overlay

**Check My Links:**
- Free
- Highlights broken links on any page
- Essential for broken link building

**Hunter.io Extension:**
- Free tier available
- Find emails while browsing
- One-click email discovery

**SimilarWeb:**
- Free
- Traffic estimates on any site
- Useful for qualification

### Using Extensions Efficiently

**Workflow for quick site evaluation:**
1. Visit target site
2. Check Ahrefs/Moz toolbar for metrics
3. Use SimilarWeb for traffic estimate
4. Use Hunter to find contact
5. Decide pursue/skip in 60 seconds

## Free Tools and Alternatives

### Free Options

**Google Search Operators:**
- Free
- Powerful for prospecting
- Requires knowledge to use effectively

**Ubersuggest:**
- Free tier with limits
- Basic backlink data
- Good for beginners

**Small SEO Tools:**
- Free
- Basic backlink checker
- Limited but functional

**Backlink Watch:**
- Free
- Basic backlink analysis
- Good for spot checks

### Budget-Friendly Stack

**For beginners/small budget:**
- Ubersuggest (free tier)
- Hunter.io (free tier)
- Moz Bar (free extension)
- Google Sheets (tracking)

**Total cost:** $0

## Tool Integration

### Building Your Tool Stack

**Solo/Freelance:**
- One major tool (Ahrefs OR Semrush)
- Hunter.io for emails
- Essential extensions
- Google Sheets

**Agency:**
- Multiple data sources (Ahrefs + Semrush)
- Dedicated outreach tool
- Full email finding suite
- CRM integration

### Data Workflow

**Standard workflow:**
1. Prospect using Ahrefs/Semrush
2. Export to spreadsheet
3. Find emails using Hunter/Snov
4. Import to outreach tool
5. Track results in CRM

## Summary

Tool selection depends on your situation:

1. **Start small** - Free tools work for learning
2. **One major tool** - Ahrefs or Semrush, not both initially
3. **Email finding is essential** - Budget for at least Hunter
4. **Extensions save time** - Install the basics
5. **Tools don't replace skill** - Data is useless without expertise

The best tool is the one you actually use effectively.
