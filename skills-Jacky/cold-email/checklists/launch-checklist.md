# Launch Checklist

Complete this checklist before launching any cold email campaign.

## Infrastructure

### Domains
- [ ] Secondary domains purchased (not primary)
- [ ] Domains aged at least 2 weeks
- [ ] Website/redirect configured
- [ ] SSL certificate active

### DNS/Authentication
- [ ] SPF record configured and passing
- [ ] DKIM configured and passing
- [ ] DMARC configured (at least p=none)
- [ ] MX records correct

### Mailboxes
- [ ] Mailboxes created with human names
- [ ] Display names configured
- [ ] Signatures set up
- [ ] Connected to sending tool
- [ ] Warmup active (2+ weeks)

### Tools
- [ ] Sending tool configured
- [ ] CRM integration working
- [ ] Tracking/analytics enabled
- [ ] Suppression list synced

## List

- [ ] ICP clearly defined
- [ ] List source is reputable
- [ ] All emails verified
- [ ] Bounce rate expected <2%
- [ ] Checked against suppression list
- [ ] No purchased lists without verification
- [ ] Segmented appropriately

## Copy

### Emails
- [ ] Subject lines tested for spam words
- [ ] Opening lines personalized
- [ ] Value proposition clear
- [ ] Social proof included
- [ ] CTAs low-friction
- [ ] Mobile-optimized (short paragraphs)
- [ ] Unsubscribe/opt-out mechanism
- [ ] Physical address included

### Sequence
- [ ] 5-7 emails planned
- [ ] Different angles across sequence
- [ ] Appropriate timing between emails
- [ ] Breakup email at end

## Pre-Send

- [ ] Test emails sent to seed accounts
- [ ] Confirmed inbox placement (not spam)
- [ ] Links tested and working
- [ ] Merge fields verified
- [ ] Team reviewed and approved
- [ ] Sending schedule configured
- [ ] Reply handling process ready

## Final Approval

- [ ] Legal compliance verified
- [ ] Manager/stakeholder approval
- [ ] Metrics tracking configured
- [ ] First-week monitoring plan in place

---

**Ready to send?** If all boxes are checked, you're good to go.

**Missing something?** Don't launch until complete.
