# Email Templates by Use Case

Comprehensive templates organized by specific use case and scenario.

---

## SaaS Sales Use Cases

### Initial Product Demo Request

**Template 1: Direct Approach**
```
Subject: Demo of [Product] for [Company]

[First name],

I help [company types] like [Company] solve [specific problem].

Quick question: how are you currently handling [specific challenge]?

Most teams I talk to are [common pain point]. We built [Product] to fix that—[Similar Company] saw [specific result] after switching.

Worth 15 minutes to see if it could work for you?

[Your name]
```

**Template 2: Problem-Focused**
```
Subject: The [problem] problem

[First name],

[Observation about their situation that suggests the problem]

Most [company types] at your stage face this challenge: [specific problem].

The typical solutions ([common approaches]) don't scale.

We built something different. [Similar Company] went from [state A] to [state B] in [timeframe].

Curious if this resonates?

[Your name]
```

**Template 3: Competitor Angle**
```
Subject: Beyond [common solution/competitor]

[First name],

Noticed [Company] is using [their current tool/approach].

Most teams on [that approach] eventually hit [specific limitation].

We solve that differently—[brief differentiation].

[Similar Company] switched from [competitor] and saw [specific improvement].

Worth comparing?

[Your name]
```

### Churn Prevention/Win-Back

**Template 1: Check-In**
```

We want to make sure you're getting value. If there's an issue, let's fix it. If priorities have shifted, totally understand.

What's the situation?

[Your name]
```

**Template 2: Win-Back**
```

[First name],

A lot has changed since then. We've added:
- [New feature 1]
- [New feature 2]
- [New feature 3]

[Similar returning customer] came back recently and said: "[Quote]"

Would it be worth another look?

[Your name]
```

## Usage Notes

### Template Selection
1. Match template to your specific situation
2. Customize all bracketed elements
3. Add genuine personalization
4. Adjust tone for your brand
5. Test variations

### Customization Depth
- **High volume**: Light customization of templates
- **Medium volume**: Significant customization
- **Low volume**: Near-custom emails using templates as frameworks

---

*These templates are starting points. The best emails combine proven structures with genuine personalization.*
