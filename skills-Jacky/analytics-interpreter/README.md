# Analytics Interpreter

Comprehensive analysis and interpretation of web analytics data from multiple sources.

## What It Does

This skill enables deep analysis of Google Search Console, Google Analytics 4, and other analytics platforms. Extract actionable insights, create automated reports, and make data-driven decisions.

## Features

- Google Search Console API integration and analysis
- Google Analytics 4 Data API workflows
- Funnel analysis and cohort analysis
- Attribution modeling and revenue tracking
- Automated anomaly detection
- Dashboard design principles
- SEO reporting and competitive benchmarking
- BigQuery integration for large-scale analysis

## Installation

1. Copy the `analytics-interpreter/` folder into your OpenClaw skills directory
2. Restart OpenClaw

## Usage

Use this skill when:
- Analyzing organic search performance
- Creating analytics dashboards
- Building automated reports
- Performing cohort or funnel analysis
- Tracking SEO metrics and rankings
- Identifying traffic anomalies

## Requirements

- Google Cloud project with APIs enabled
- Service account credentials for API access
- Google Search Console property access
- Google Analytics 4 property access
- Python with pandas, google-api-client libraries
