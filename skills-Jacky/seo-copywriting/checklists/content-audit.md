# Content Audit Checklist

> Use this checklist to audit existing content. Identify what to update, consolidate, or remove.

---

## Step 1: Gather Data

### Export from Google Search Console

For each page, note:
- [ ] Total clicks (last 3 months)
- [ ] Total impressions
- [ ] Average CTR
- [ ] Average position

### Export from Analytics

For each page, note:
- [ ] Pageviews (last 3 months)
- [ ] Avg. time on page
- [ ] Bounce rate
- [ ] Conversions (if tracked)

### Export from SEO Tool (Ahrefs/Semrush)

For each page, note:
- [ ] Organic traffic
- [ ] Number of keywords ranking
- [ ] Backlinks/referring domains
- [ ] Content age (last modified)

---

## Step 2: Categorize Each Page

For each page in your audit, assign a category:

### Category A: Performing Well ✅
- Ranks page 1 for target keyword
- Getting consistent traffic
- Converting well
- **Action**: Protect and optimize

### Category B: Needs Improvement 🔧
- Ranks page 2-3 or positions 5-10
- Has potential but underperforming
- Thin content or outdated
- **Action**: Update and improve

### Category C: Consolidate 🔀
- Similar to other pages (cannibalization)
- Thin content that can be merged
- Multiple pages on same topic
- **Action**: Merge into stronger page

### Category D: Remove 🗑️
- No traffic for 12+ months
- Outdated/irrelevant content
- Poor quality that hurts site
- **Action**: Delete or noindex

## Common Issues to Look For

### Search Intent Mismatch
- [ ] Content type matches SERP (blog post vs. product page)
- [ ] Content format matches SERP (listicle vs. how-to)
- [ ] Content answers the actual query

### Thin Content
- [ ] Word count is competitive with ranking pages
- [ ] Content provides unique value
- [ ] Not just surface-level coverage

### Outdated Content
- [ ] Statistics are current
- [ ] Tools/products still exist
- [ ] Advice is still accurate
- [ ] Year references are current

### Keyword Cannibalization
- [ ] No other pages targeting same keyword
- [ ] Clear differentiation from similar pages
- [ ] Internal links don't compete

### Technical Issues
- [ ] Page is indexed
- [ ] No duplicate content
- [ ] Fast loading
- [ ] Mobile-friendly

### E-E-A-T Gaps
- [ ] Author identified and credentialed
- [ ] Sources cited
- [ ] Content demonstrates expertise
- [ ] Trust signals present

## Content Consolidation Process

When merging pages:

### Step 1: Choose the "Winner"
- Page with most backlinks
- Page with best rankings
- Page with best URL structure

### Step 2: Combine Content
- Merge unique content from both pages
- Remove redundancy
- Improve overall quality

### Step 3: Redirect
- 301 redirect old URL to new URL
- Update internal links

### Step 4: Update Sitemap
- Remove old URL
- Ensure new URL is included

---

## Content Update Process

When improving pages:

### Quick Wins (1-2 hours)
- [ ] Update title tag for CTR
- [ ] Add "Last updated" date
- [ ] Update year in title if applicable
- [ ] Add FAQ section
- [ ] Add internal links
- [ ] Update statistics

### Medium Effort (2-4 hours)
- [ ] Add missing subtopics
- [ ] Improve introduction
- [ ] Add examples/screenshots
- [ ] Enhance E-E-A-T signals
- [ ] Add comparison tables
- [ ] Improve formatting

### Major Rewrite (4+ hours)
- [ ] Complete content refresh
- [ ] New structure based on SERP analysis
- [ ] Original research/data
- [ ] Expert interviews
- [ ] Custom graphics
