# On-Page SEO Checklist

> A comprehensive checklist for on-page optimization. Use when auditing existing pages or optimizing new content.

---

## Title Tag

- [ ] **Length**: 50-60 characters (check in SERP preview tool)
- [ ] **Primary keyword**: Included near the beginning
- [ ] **Compelling**: Would you click this in search results?
- [ ] **Unique**: Different from other pages on site
- [ ] **Brand**: Included at end if appropriate

**Format**: `[Primary Keyword]: [Hook/Benefit] | [Brand]`

## Meta Description

- [ ] **Length**: 105-155 characters
- [ ] **Primary keyword**: Included naturally
- [ ] **Compelling**: Clear benefit + CTA
- [ ] **Unique**: Different from other pages
- [ ] **Matches content**: Accurately describes the page

**Optimized meta**: ________________________________________________

## URL

- [ ] **Short**: 3-5 words ideal
- [ ] **Descriptive**: Indicates page content
- [ ] **Keyword**: Includes primary keyword
- [ ] **Clean**: No unnecessary parameters or numbers
- [ ] **Hyphens**: Words separated by hyphens (not underscores)

**Current URL**: ________________________________________________
**Optimized URL** (if changing): ________________________________________________

## H1 Headline

- [ ] **One per page**: Only one H1 tag
- [ ] **Primary keyword**: Included naturally
- [ ] **Matches title**: Similar to title tag (can be longer)
- [ ] **Compelling**: Draws reader in
- [ ] **Above fold**: Visible without scrolling

---

## Heading Hierarchy

- [ ] **Logical order**: H1 → H2 → H3 (no skipping levels)
- [ ] **H2s for main sections**: Each major topic has H2
- [ ] **H3s for subsections**: Nested under relevant H2
- [ ] **Keywords in H2s**: Secondary keywords where natural
- [ ] **Scannable**: Can understand content from headings alone

**H2 structure**:

  - H3: ________________________________________________
  - H3: ________________________________________________

## Content Quality

### Length & Depth
- [ ] **Appropriate length**: Matches/exceeds top competitors
- [ ] **Comprehensive**: Covers topic fully
- [ ] **No fluff**: Every section adds value
- [ ] **Subtopics covered**: Based on competitor analysis

**Current word count**: ____________
**Competitor average**: ____________

### Structure
- [ ] **Short paragraphs**: 2-3 sentences max
- [ ] **Visual breaks**: Every 300 words
- [ ] **Lists and bullets**: Used where appropriate
- [ ] **Tables**: For comparisons/data
- [ ] **FAQ section**: Addresses common questions

### Readability
- [ ] **Reading level**: 6th-8th grade
- [ ] **Active voice**: Predominant
- [ ] **Clear language**: No unnecessary jargon
- [ ] **Flows well**: Reads naturally aloud

## Technical Checks

### Crawlability
- [ ] Page is indexable (no noindex tag)
- [ ] Canonical URL is correct
- [ ] No duplicate content issues
- [ ] Page is in XML sitemap

### Page Speed
- [ ] LCP under 2.5 seconds
- [ ] FID/INP under 100ms
- [ ] CLS under 0.1
- [ ] No render-blocking resources

**Check at**: https://pagespeed.web.dev

### Mobile
- [ ] Mobile-friendly
- [ ] Text readable without zooming
- [ ] Buttons/links are tap-friendly
- [ ] No horizontal scrolling
