# SEO Copywriting

Write content that ranks on Google AND AI search engines while converting readers.

## What It Does

This skill teaches the art of writing SEO-optimized content that ranks in traditional search and gets cited by AI systems (ChatGPT, Perplexity, AI Overviews) while driving conversions.

## Features

- Search intent matching strategies
- E-E-A-T optimization (Experience, Expertise, Authoritativeness, Trust)
- Content structure for featured snippets
- Headline formulas that rank AND get clicks
- Keyword integration without stuffing
- Listicle and comparison content mastery
- AI Search / GEO (Generative Engine Optimization)
- Schema markup for rich results
- Conversion-focused CTAs in content

## Installation

1. Copy the `seo-copywriting/` folder into your OpenClaw skills directory
2. Restart OpenClaw

## Usage

Use this skill when:
- Writing blog posts and articles
- Creating "Best X" listicles
- Optimizing for AI citations
- Writing comparison content
- Crafting SEO landing pages
- Updating existing content for freshness

## Requirements

- Keyword research tool (Ahrefs, SEMrush, or similar)
- Content management system
- Schema markup implementation capability
- Understanding of on-page SEO factors
- Access to analytics for performance tracking
