# Other Newsletter Platforms: Comprehensive Guide

Beyond the major platforms, several alternatives serve specific needs. This guide covers them all.

---

## Mailchimp

### Overview

**Best for:** Small businesses, agencies, e-commerce integration

**Founded:** 2001
**Pricing model:** Free tier, then paid by contacts
**Focus:** All-in-one marketing platform

### Pricing (2024)

| Plan | Contacts | Price/month |
|------|----------|-------------|
| Free | 500 | $0 |
| Essentials | 500-50k | $13-$350 |
| Standard | 500-100k | $20-$700 |
| Premium | 10k+ | $350+ |

### Key Features

**Strengths:**
- Extensive integrations (300+)
- E-commerce tools
- Advanced automation
- Established reputation
- Comprehensive analytics

**Weaknesses:**
- No native paid subscriptions
- Can get expensive at scale
- Less newsletter-specific
- Interface can be overwhelming
- Limited creator features

### Who Should Use

- E-commerce businesses
- Small businesses with existing Mailchimp setup
- Agencies managing multiple clients
- Those needing CRM integration

### Who Should Avoid

- Creators wanting paid newsletters
- Those prioritizing simplicity
- Growth-focused newsletters

## Migration Considerations

### Before You Switch

**Evaluate:**
1. Current platform limitations
2. New platform requirements
3. Migration complexity
4. Timeline needed
5. Cost comparison

### Migration Checklist

- [ ] Export subscriber list
- [ ] Export tags/segments
- [ ] Document automation flows
- [ ] Save email templates
- [ ] Update DNS records
- [ ] Test deliverability
- [ ] Plan re-confirmation (if needed)
- [ ] Set redirect for archive pages
- [ ] Update signup forms
- [ ] Communicate with subscribers

### Common Migration Paths

| From | Common To | Reason |
|------|-----------|--------|
| Mailchimp | Beehiiv, ConvertKit | Creator features |
| Substack | Beehiiv | Growth tools |
| AWeber | ConvertKit | Modern features |
| Drip | Klaviyo | Better e-commerce |
| Any | Ghost | Custom branding |
