# Newsletter Empire Skill

Master newsletter growth, monetization, and operations. Built for operators scaling to 250k+ subscribers.

## Quick Reference

### Growth Levers (Priority Order)
1. **Referral program** → 20-40% of growth for mature newsletters
2. **Paid acquisition** → SparkLoop/Beehiiv network ($1-3 CPA)
3. **Cross-promotions** → Free swaps with similar-sized newsletters
4. **SEO/content** → Long-term compounding asset
5. **Social → email** → Convert followers to owned audience

## Core Workflows

### Weekly Newsletter Workflow
```
Monday: Content ideation + outline
Tuesday: Draft writing
Wednesday: Edit + schedule
Thursday: Send + monitor
Friday: Analyze + iterate
Weekend: Batch content for next week
```

### Sponsor Management
```
1. Create media kit (see templates/media-kit.md)
2. Set CPM pricing: (subscribers × open_rate × CPM) / 1000
3. Outreach to aligned brands
4. Send insertion order
5. Collect creative by deadline
6. Insert + send
7. Report results
8. Invoice
```

### Growth Sprint (30-day)
```
Week 1: Audit + quick wins
- Clean list (remove 90-day inactive)
- Optimize welcome sequence
- A/B test subject lines

Week 2: Referral program
- Set up SparkLoop/Beehiiv referrals
- Create reward tiers
- Add referral widget to emails

Week 3: Paid acquisition
- Join newsletter ad networks
- Test $500 budget
- Measure CPA + quality

Week 4: Cross-promotions
- Identify 10 newsletters to swap with
- Reach out + schedule
- Track results
```

## Platform Quick Picks

**Choose Beehiiv if:**
- Growth is priority #1
- Want built-in referral program
- Need ad network access
- Focused on free newsletter with sponsors

**Choose ConvertKit (Kit) if:**
- Creator-focused (courses, products)
- Need advanced automations
- Want commerce built-in
- Selling digital products

**Choose Substack if:**
- Paid subscriptions are primary model
- Want network discovery effects
- Minimal tech/design needs
- Writer-first approach

**Choose Ghost if:**
- Want full customization
- Technical team available
- Building media brand
- Need advanced membership features

## Sponsor Pricing Calculator

```
CPM Pricing Formula:
Price = (Subscribers × Open Rate × CPM) / 1000

Example (25k list, 40% open rate, $50 CPM):
Price = (25,000 × 0.40 × 50) / 1000 = $500

CPM Benchmarks:
- Broad/lifestyle: $20-40 CPM
- Business/finance: $40-80 CPM
- Tech/SaaS: $50-100 CPM
- Niche professional: $75-150 CPM
```
