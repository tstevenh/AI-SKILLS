# Newsletter Metrics That Matter: Complete Analytics Guide

Data drives decisions. Understanding which metrics matter—and which are vanity—separates successful newsletter operators from those flying blind.

---

## The Metrics Hierarchy

### Tier 1: North Star Metrics

These are your ultimate success indicators:

**1. Revenue**
- Total revenue
- Revenue per subscriber
- Revenue growth rate

**2. Engaged Subscriber Count**
- Not just total subscribers
- Subscribers who actually engage
- Active audience size

**3. Growth Rate**
- Net subscriber growth
- Growth velocity
- Sustainability of growth

### Tier 2: Diagnostic Metrics

These explain your North Star performance:

**Email Performance:**
- Open rate
- Click rate
- Click-to-open rate

**List Health:**
- Bounce rate
- Unsubscribe rate
- Spam complaint rate

**Growth Efficiency:**
- Subscriber acquisition cost
- Organic vs. paid mix
- Referral rate

### Tier 3: Operational Metrics

Day-to-day indicators:

- Send volume
- Deliverability rate
- List growth by source
- Content performance

## Email Performance Metrics

### Open Rate

**Formula:**
```
Open Rate = (Unique Opens / Emails Delivered) × 100
```

**Benchmarks:**
| Performance | Open Rate |
|-------------|-----------|
| Poor | <20% |
| Below Average | 20-30% |
| Average | 30-40% |
| Good | 40-50% |
| Excellent | 50-60% |
| Outstanding | >60% |

**What Affects Open Rate:**
- Subject line quality
- Sender reputation
- Send time
- List quality
- Preview text
- Audience engagement

**Important Note:**
Apple Mail Privacy Protection (MPP) inflates open rates. Many "opens" are proxy servers, not real readers. Focus on click rate for accuracy.

### Click Rate (CTR)

**Formula:**
```
Click Rate = (Unique Clicks / Emails Delivered) × 100
```

**Benchmarks:**
| Performance | Click Rate |
|-------------|------------|
| Poor | <1% |
| Below Average | 1-2% |
| Average | 2-4% |
| Good | 4-6% |
| Excellent | 6-10% |
| Outstanding | >10% |

**What Affects Click Rate:**
- Content relevance
- Link placement
- Call-to-action clarity
- Content quality
- Subscriber engagement

### Click-to-Open Rate (CTOR)

**Formula:**
```
CTOR = (Unique Clicks / Unique Opens) × 100
```

**What It Tells You:**
- Content effectiveness (separate from subject line)
- How engaging your content is once opened
- Quality of your links/CTAs

**Benchmarks:**
| Performance | CTOR |
|-------------|------|
| Poor | <5% |
| Average | 5-10% |
| Good | 10-15% |
| Excellent | >15% |

### Unsubscribe Rate

**Formula:**
```
Unsubscribe Rate = (Unsubscribes / Emails Delivered) × 100
```

**Benchmarks:**
| Performance | Unsub Rate |
|-------------|------------|
| Excellent | <0.1% |
| Good | 0.1-0.2% |
| Average | 0.2-0.3% |
| Concerning | 0.3-0.5% |
| Problem | >0.5% |

**What Causes High Unsubscribes:**
- Content mismatch with expectations
- Too frequent emails
- Irrelevant content
- List decay
- Quality decline

### Spam Complaint Rate

**Formula:**
```
Complaint Rate = (Spam Complaints / Emails Delivered) × 100
```

**Critical Threshold:** <0.1%

Above 0.1% = serious deliverability risk

**What Causes Complaints:**
- Forgot they subscribed
- Too many emails
- Difficult to unsubscribe
- Content perceived as spam
- Wrong audience

## Growth Metrics

### Net Subscriber Growth

**Formula:**
```
Net Growth = New Subscribers - Unsubscribes - Bounces
```

**Example:**
- New: 500
- Unsubs: 100
- Bounces: 50
- Net: 350

### Growth Rate

**Formula:**
```
Monthly Growth Rate = (Net Growth / Previous Total) × 100
```

**Benchmarks by Stage:**
| Stage | Subscribers | Good Growth Rate |
|-------|-------------|------------------|
| Launch | 0-1k | 50-100% |
| Early | 1k-5k | 20-40% |
| Growth | 5k-25k | 10-20% |
| Scale | 25k-100k | 5-10% |
| Mature | 100k+ | 3-5% |

### Subscriber Acquisition Cost (SAC)

**Formula:**
```
SAC = Total Acquisition Spend / New Subscribers
```

**Benchmarks:**
| Channel | Typical SAC |
|---------|------------|
| Organic | $0 |
| Referral | $0.50-2.00 |
| Cross-promo | $0-1.00 |
| Newsletter ads | $1.00-3.00 |
| Paid social | $2.00-5.00 |
| Podcast | $3.00-8.00 |

### Organic vs. Paid Ratio

**Healthy Ratios:**
| Stage | Organic | Paid |
|-------|---------|------|
| Early | 100% | 0% |
| Growing | 70-80% | 20-30% |
| Scaling | 50-60% | 40-50% |
| Mature | 40-50% | 50-60% |

### Referral Metrics

**Referral Rate:**
```
Referral Rate = Subscribers from Referrals / Total New Subscribers
```

**Viral Coefficient:**
```
K = Referrals per Subscriber × Conversion Rate

K > 1 = Viral growth
K = 0.5-0.8 = Good
K < 0.3 = Needs work
```

## Revenue Metrics

### Monthly Recurring Revenue (MRR)

**Formula:**
```
MRR = Paid Subscribers × Average Monthly Price
```

**For mixed billing:**
```
MRR = (Monthly Subs × Monthly Price) + (Annual Subs × Annual Price / 12)
```

### Average Revenue Per Subscriber (ARPS)

**Formula:**
```
ARPS = Total Revenue / Total Subscribers
```

**Benchmarks by Model:**
| Model | Monthly ARPS |
|-------|-------------|
| Sponsors only | $0.02-0.10 |
| Freemium | $0.05-0.20 |
| Paid subscription | $0.20-0.50 |
| Hybrid | $0.10-0.30 |

### Subscriber Lifetime Value (LTV)

**Simple Formula:**
```
LTV = ARPS × Average Subscriber Lifespan (months)
```

**Advanced Formula:**
```
LTV = ARPS / Churn Rate
```

**Example:**
- ARPS: $0.15/month
- Churn: 5%/month
- LTV = $0.15 / 0.05 = $3.00

### LTV:CAC Ratio

**Formula:**
```
LTV:CAC = Lifetime Value / Customer Acquisition Cost
```

**Benchmarks:**
| Ratio | Assessment |
|-------|------------|
| <1:1 | Losing money |
| 1:1 to 2:1 | Break-even |
| 2:1 to 3:1 | Healthy |
| 3:1 to 5:1 | Strong |
| >5:1 | Scale faster |

### Churn Rate

**Monthly Churn:**
```
Churn Rate = Cancelled Subscribers / Total Subscribers
```

**Benchmarks:**
| Billing | Low | Average | High |
|---------|-----|---------|------|
| Monthly | 3-5% | 5-8% | 8-12% |
| Annual | 1-2%/mo | 2-3%/mo | 3-5%/mo |

## Engagement Scoring

### Building an Engagement Score

Create a composite score based on:

| Activity | Points |
|----------|--------|
| Open email | +1 |
| Click link | +2 |
| Reply | +5 |
| Refer someone | +10 |
| Purchase | +15 |
| No activity (30 days) | -2 |
| No activity (60 days) | -5 |

### Engagement Segments

**Segment subscribers by engagement:**

| Segment | Criteria | Action |
|---------|----------|--------|
| Super engaged | Score >50 | VIP treatment, upsells |
| Engaged | Score 20-50 | Normal nurturing |
| Cooling | Score 5-20 | Re-engagement campaign |
| Cold | Score <5 | Last chance, then sunset |

### 30/60/90 Day Engagement

Track what percentage of subscribers:
- Opened in last 30 days
- Opened in last 60 days
- Opened in last 90 days

**Healthy Targets:**
| Window | Target |
|--------|--------|
| 30-day | >60% |
| 60-day | >75% |
| 90-day | >85% |

## Taking Action on Metrics

### When Opens Drop

**Investigate:**
- Subject line quality
- Send time
- Deliverability issues
- List quality

**Actions:**
- A/B test subject lines
- Test new send times
- Check authentication
- Clean inactive subscribers

### When Clicks Drop

**Investigate:**
- Content quality
- Link placement
- Relevance
- Audience fatigue

**Actions:**
- Improve content value
- Test different CTAs
- Survey audience
- Refresh format

### When Growth Slows

**Investigate:**
- Channel effectiveness
- Landing page conversion
- Market saturation
- Competition

**Actions:**
- Test new channels
- Optimize landing page
- New lead magnets
- Referral program push

### When Churn Increases

**Investigate:**
- Content quality
- Frequency mismatch
- Value delivery
- Competition

**Actions:**
- Win-back campaigns
- Survey churned users
- Improve onboarding
- Add more value

## Metrics Mistakes to Avoid

### Mistake 1: Vanity Metrics Focus

**Problem:** Celebrating total subscribers while ignoring engagement
**Solution:** Focus on engaged subscribers and revenue

### Mistake 2: Not Tracking Source

**Problem:** Can't tell which channels work
**Solution:** UTM tags, source tracking on every link

### Mistake 3: Ignoring Trends

**Problem:** Looking at single data points, not patterns
**Solution:** Track week-over-week, month-over-month

### Mistake 4: Analysis Paralysis

**Problem:** Too much data, no action
**Solution:** Pick 3-5 key metrics, act on them

### Mistake 5: Infrequent Review

**Problem:** Discovering problems too late
**Solution:** Weekly review minimum
