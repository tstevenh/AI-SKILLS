# AI Automation

Leverage AI and automation to multiply output and reduce costs across your business.

## What It Does

This skill covers the complete AI automation stack—LLMs, image/video generation, automation platforms (n8n, Make, Zapier), and real-world workflows to automate repetitive tasks and scale operations.

## Features

- AI model selection guide (Claude, GPT-4, Gemini, Llama)
- Automation platform comparisons (n8n, Make, Zapier)
- Content writing and repurposing workflows
- Lead enrichment and data extraction automations
- Customer support chatbot implementations
- AI-powered research and reporting
- Cost optimization strategies
- 50+ ready-to-deploy workflow templates

## Installation

1. Copy the `ai-automation/` folder into your OpenClaw skills directory
2. Restart OpenClaw

## Usage

Use this skill when:
- Automating repetitive business tasks
- Building AI-powered workflows
- Selecting the right AI model for a task
- Integrating multiple tools and APIs
- Creating content at scale
- Setting up lead enrichment pipelines

## Requirements

- OpenAI, Anthropic, or other AI API keys
- Automation platform account (n8n, Make, or Zapier)
- Basic understanding of APIs and webhooks
- Integration tools access (Airtable, Notion, Sheets, etc.)
