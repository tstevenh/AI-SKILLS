# Community Events Strategy

## Overview

Events are the high-octane fuel of community engagement. They create shared experiences, deepen relationships, provide concentrated value, and give members reasons to show up. This guide covers everything from weekly calls to annual summits.

---

## Part 1: Event Fundamentals

### Why Events Matter

**Engagement Impact**:
- Event attendees have 3x higher retention
- Live interaction builds stronger bonds
- Shared experiences create community memory
- Events showcase community value
- FOMO drives overall engagement

### Event Types

**Regular Events**:
- Weekly calls
- Office hours
- Co-working sessions
- Networking mixers

**Special Events**:
- Guest expert sessions
- Workshops
- Challenges
- Launch events
- Milestone celebrations

**Premium Events**:
- Masterminds
- Intensives
- Retreats
- Conferences

## Part 2: Virtual Event Formats

### Q&A / AMA Sessions

**Format**:
```
Duration: 45-60 minutes
Structure:
- Brief intro (5 min)
- Pre-submitted questions (20 min)
- Live questions (30 min)
- Wrap-up (5 min)

Best for: Expert access, problem-solving
```

**Pre-Event Prompt**:
```
🎤 AMA: [Topic] with [Expert]

[Date] at [Time]

Submit your questions below! 👇

Top questions get priority during the live session.
```

### Workshops / Trainings

**Format**:
```
Duration: 60-90 minutes
Structure:
- Context setting (10 min)
- Core teaching (30-40 min)
- Live exercise/implementation (20-30 min)
- Q&A (15 min)
- Next steps (5 min)

Best for: Skill building, implementation
```

**Engagement Tips**:
- Include interactive elements
- Use breakout rooms
- Provide worksheets
- Follow up with resources

### Networking Events

**Format**:
```
Duration: 60-90 minutes
Structure:
- Welcome and rules (5 min)
- Icebreaker round (10 min)
- Breakout rotations (40-60 min)
- Large group reconnect (10 min)
- Close (5 min)

Best for: Connection building
```

**Breakout Structure**:
- 3-4 people per room
- 10-15 minutes per rotation
- Conversation prompts provided

### Co-Working Sessions

**Format**:
```
Duration: 90-180 minutes
Structure:
- Quick check-in (5 min)
- Goal setting (5 min)
- Work sprint 1 (45 min)
- Mini-break (5 min)
- Work sprint 2 (45 min)
- Wins share (10 min)

Best for: Accountability, focus
```

### Masterminds / Hot Seats

**Format**:
```
Duration: 60-90 minutes
Structure:
- Brief check-ins (5 min each)
- Hot seat 1 (20 min)
- Hot seat 2 (20 min)
- Hot seat 3 (20 min optional)
- Close (5 min)

Hot Seat Format:
- Present challenge (5 min)
- Clarifying questions (5 min)
- Advice/brainstorm (10 min)

Best for: Problem-solving, peer coaching
```

### Event Planning Checklist

```
2 WEEKS BEFORE:
[ ] Topic and format finalized
[ ] Speaker/host confirmed
[ ] Platform tested
[ ] Promotional content created
[ ] RSVP page set up

1 WEEK BEFORE:
[ ] First announcement sent
[ ] Reminder scheduled
[ ] Presentation/materials ready
[ ] Tech setup confirmed

1 DAY BEFORE:
[ ] Final reminder sent
[ ] Tech check completed
[ ] Backup plan ready

[ ] Join early (15 min)
[ ] Record if applicable
[ ] Moderate chat
[ ] Take notes on questions

POST-EVENT:
[ ] Share recording
[ ] Post key takeaways
[ ] Gather feedback
[ ] Plan next event

## Part 4: Running Great Events

### Pre-Event Preparation

**Tech Check**:
- Test platform
- Check audio/video
- Prepare screen shares
- Have backup plan

**Content Prep**:
- Outline or slides ready
- Questions prepared
- Resources queued
- Time estimates

### During the Event

**Hosting Best Practices**:
- Start on time
- Energy and enthusiasm
- Engage the chat
- Call on people by name
- Keep on schedule
- Leave time for questions

**Engagement Techniques**:
- Ask questions throughout
- Use polls
- Encourage chat participation
- Small group discussions
- Live exercises

### Common Problems

**Low Attendance**:
- Better promotion
- Right timing
- More compelling topic
- Easier access

**Low Engagement**:
- More interaction points
- Smaller groups
- Better facilitation
- Relevant content

**Tech Issues**:
- Always have backup
- Communicate quickly
- Reschedule if needed
- Follow up after

## Part 6: Event Cadence

### Building an Event Calendar

**Minimum Viable Schedule**:
```
Weekly: One live call (Q&A, teaching, or networking)
Monthly: One special event (workshop, guest, challenge)
```

**Optimal Schedule**:
```
Weekly:
- Tuesday: Office hours
- Thursday: Main event (call, workshop, etc.)

Monthly:
- Week 1: Guest expert
- Week 2: Networking event
- Week 3: Workshop
- Week 4: Community celebration
```

### Avoiding Event Fatigue

**Signs of Fatigue**:
- Declining attendance
- Same people attending
- Lower energy
- Complaints about frequency

**Solutions**:
- Reduce frequency
- Vary formats
- Make some optional
- Quality over quantity

## Part 7: Guest Expert Events

### Finding Guest Experts

**Sources**:
- Your network
- Member recommendations
- Authors/thought leaders
- Complementary businesses
- Industry experts

### Guest Outreach

**Template**:
```
Subject: Guest expert invitation - [Community Name]

Hi [Name],

I run [Community Name], a community of [X] 
[audience description].

Your work on [topic] would be incredibly valuable 
for our members.

Would you be open to doing a [format] with our 
community? [Date/Time range]

What we'd need:
- [Time commitment]
- [Format details]
- [Promotion expectations]

What we offer:
- Exposure to [X] engaged [audience]
- Recording for your use
- Promotion to our email list
- [Any payment if applicable]

Let me know if you'd be interested!

[Your name]
```

### Running Guest Events

**Pre-Event**:
- Brief guest on audience
- Align on format and timing
- Tech check
- Gather pre-questions

**During Event**:
- Proper introduction
- Facilitate well
- Manage time
- Thank publicly

**Post-Event**:
- Send thank you
- Share recording/assets
- Provide feedback
- Maintain relationship

## Part 9: In-Person Events

### When to Consider In-Person

**Good Timing**:
- Established community (500+ engaged)
- Geographic clusters
- Budget available
- Demand from members
- Clear value proposition

### In-Person Event Types

**Meetups**:
```
Size: 10-50 people
Duration: 2-4 hours
Complexity: Low
Cost: Low
```

**Workshops**:
```
Size: 20-100 people
Duration: Half-day to full-day
Complexity: Medium
Cost: Medium
```

**Retreats**:
```
Size: 20-50 people
Duration: 2-5 days
Complexity: High
Cost: High
```

**Conferences**:
```
Size: 100-1000+ people
Duration: 1-3 days
Complexity: Very High
Cost: Very High
```

### In-Person Planning

**Key Considerations**:
- Venue selection and cost
- Travel logistics
- Catering
- Programming
- Networking time
- Safety and liability
- Contingency planning

## Part 10: Measuring Event Success

### Event Metrics

**Attendance**:
```
- Registration count
- Attendance rate (attended/registered)
- Unique attendees over time
- Repeat attendees
```

**Engagement**:
```
- Chat messages
- Questions asked
- Participation in activities
- Post-event comments
```

**Satisfaction**:
```
- NPS (would you recommend?)
- Value rating
- Likelihood to attend again
- Qualitative feedback
```

**Business Impact**:
```
- Retention of attendees vs non-attendees
- Conversion (for sales events)
- Content generated
- Testimonials gathered
```

## Summary

### Event Principles

1. **Consistency**: Regular events build habits
2. **Value**: Every event should deliver
3. **Connection**: Facilitate interaction
4. **Variety**: Mix formats and topics
5. **Feedback**: Continuously improve

### Starter Event Plan

```
Month 1:
- Week 1: Launch Q&A call
- Week 2: Educational workshop
- Week 3: Networking event
- Week 4: Guest expert

Ongoing:
- Weekly: Main event (rotating format)
- Monthly: Special event (guest, challenge, etc.)
```

### Event Success Formula

```
Clear Value + Easy Access + Great Facilitation + Consistent Schedule = Event Success
