## Design Trends Observed

### Current Trends (2024-2026)

1. **3D Elements** - App&Flow, many SaaS sites
2. **Dark Mode Support** - App&Flow with scheme switcher
3. **Sticky Navigation** - Universal across almost all sites
4. **Large Typography** - Many portfolios and experimental sites
5. **Video Integration** - Bajgart Office explainer video
6. **Scroll Animations** - App&Flow, many modern sites
7. **Bento Grids** - Feature sections in SaaS sites
8. **Minimalism** - Luciano Pereira, many portfolios

### Common Features Across Categories

**Almost Universal:**
- Sticky navigation
- Clear CTA buttons
- Mobile-responsive design
- Social proof elements

**Category-Specific:**
- **SaaS:** Feature grids, pricing tables, screenshots
- **Portfolios:** Large images, minimal text, project showcases
- **Landing Pages:** Single CTA, urgency elements
- **Experimental:** Unconventional layouts, high interactivity

## How to Use This Gallery

### For Research:
1. Find your category
2. Visit 3-5 similar sites
3. Note common patterns
4. Identify what makes each unique

### For Inspiration:
1. Look outside your category
2. Note interaction patterns
3. Study color and typography choices
4. Adapt (don't copy) elements

### For Client Work:
1. Show examples from their industry
2. Reference specific patterns
3. Explain why certain choices work
4. Build on proven patterns
