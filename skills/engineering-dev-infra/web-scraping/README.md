# Web Scraping

Master web data extraction with comprehensive scraping techniques and anti-detection strategies.

## What It Does

This skill teaches systematic web scraping—from HTTP fundamentals and HTML parsing to headless browsers and anti-bot evasion. Extract data at scale while respecting legal and ethical boundaries.

## Features

- HTTP protocol and web communication fundamentals
- HTML parsing with CSS selectors and XPath
- Python (Scrapy, BeautifulSoup) and Node.js (Puppeteer, Playwright) stacks
- Headless browser scraping and JavaScript rendering
- Anti-bot detection evasion techniques
- Proxy management and rotation
- Browser fingerprinting and TLS evasion
- API reverse engineering
- Scraping at scale with cloud services

## Installation

1. Copy the `web-scraping/` folder into your OpenClaw skills directory
2. Restart OpenClaw

## Usage

Use this skill when:
- Extracting data from websites
- Building automated data collection systems
- Reverse engineering web APIs
- Scraping at scale
- Handling JavaScript-rendered content
- Managing proxies and avoiding detection

## Requirements

- Python 3.x or Node.js environment
- Web scraping libraries (requests, Scrapy, Puppeteer, Playwright)
- Proxy service (for large-scale scraping)
- Browser binaries (for headless scraping)
- Understanding of HTML/CSS/JavaScript
