## Image Generation

### Midjourney

- **Website:** midjourney.com
- **Pricing:** $10-120/month
- **Strengths:** Best aesthetics, marketing visuals
- **Limitations:** No API, Discord-based
- **Best for:** Hero images, creative content

### DALL-E 3 (OpenAI)

- **API:** api.openai.com
- **Pricing:** $0.04-0.12/image
- **Strengths:** Text in images, API access
- **Limitations:** Less artistic than Midjourney
- **Best for:** Images with text, automation

### Flux (Black Forest Labs)

- **APIs:** Replicate, Together, fal.ai
- **Pricing:** ~$0.05/image
- **Strengths:** Photorealism, speed
- **Limitations:** Newer, less documentation
- **Best for:** Product photos, realistic images

### Stable Diffusion

- **Access:** Self-host or APIs
- **Pricing:** $0.002-0.02/image
- **Strengths:** Open source, customizable
- **Limitations:** Requires setup
- **Best for:** High volume, custom models

### Leonardo.ai

- **Website:** leonardo.ai
- **Pricing:** Free tier, $12-60/month
- **Strengths:** Game/product assets
- **Best for:** Consistent style, products

### Ideogram

- **Website:** ideogram.ai
- **Pricing:** Free tier, $7-20/month
- **Strengths:** Best text rendering
- **Best for:** Logos, text-heavy images

## Quick Selection Guide

### By Use Case

| Need | Recommended |
|------|-------------|
| General AI | Claude 3.5 Sonnet |
| High volume | GPT-4o-mini |
| Marketing images | Midjourney |
| Automated images | DALL-E 3 or Flux |
| Video clips | Runway |
| Avatar videos | HeyGen |
| Voiceovers | ElevenLabs |
| Automation (tech) | n8n |
| Automation (marketing) | Make |
| Automation (simple) | Zapier |
| Lead enrichment | Clay or Apollo |
| Support chat | Intercom |
| Cold email | Instantly |
| Transcription | Whisper |

### By Budget

| Budget | Stack |
|--------|-------|
| Free | GPT-4o-mini + Zapier free + Apollo free |
| $100/mo | Claude + n8n cloud + Apollo starter |
| $500/mo | Claude + Make + Clay + Instantly |
| $2000/mo | Full stack with enterprise features |
