# SOPs & Documentation

Create, manage, and optimize standard operating procedures and organizational documentation.

## What It Does

This skill provides comprehensive frameworks for SOP creation, knowledge management, and process documentation. Build scalable, automation-ready documentation systems.

## Features

- SOP creation frameworks (WHO-WHAT-WHEN-WHERE-WHY-HOW)
- DMAIC and Six Sigma methodologies
- Documentation templates for all process types
- Visual documentation (flowcharts, diagrams)
- Version control for documentation
- Review and approval workflows
- Knowledge base architecture
- Automation-ready SOP structures
- Measuring documentation effectiveness

## Installation

1. Copy the `sops-documentation/` folder into your OpenClaw skills directory
2. Restart OpenClaw

## Usage

Use this skill when:
- Creating new standard operating procedures
- Documenting business processes
- Building knowledge bases
- Writing work instructions
- Designing runbooks
- Onboarding new team members
- Improving process efficiency

## Requirements

- Documentation platform (Confluence, Notion, or similar)
- Diagramming tool (Lucidchart, Draw.io, Miro)
- Version control system
- Template creation capabilities
- Team collaboration tools
