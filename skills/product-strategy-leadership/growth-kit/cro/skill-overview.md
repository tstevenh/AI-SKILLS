## Core Framework: The 7-Dimension CRO Analysis

For any conversion optimization, analyze across these dimensions (in order of impact):

1. **Value Proposition Clarity** - Can visitors understand what this is and why they should care in 5 seconds?
2. **Headline Effectiveness** - Does it communicate core value with specificity?
3. **CTA Placement & Copy** - Is there one clear action that communicates value?
4. **Visual Hierarchy** - Can someone scanning get the main message?
5. **Trust & Social Proof** - Are credibility signals present near decision points?
6. **Objection Handling** - Are likely concerns addressed?
7. **Friction Reduction** - Is every unnecessary step removed?

## Universal CRO Principles

### Every Element Has a Cost
- Each form field reduces completion by 10-25%
- Each step in a flow loses 20-40% of users
- Each second of load time costs 7% conversion

### Value Must Exceed Effort
- Clear benefit above every form
- Show what they get before asking for commitment
- Reduce perceived effort (field count, labels, progress)

### Reduce Cognitive Load
- One primary action per screen
- Clear visual hierarchy
- Smart defaults where possible

## Output Format

All CRO deliverables should include:

### Quick Wins (Implement Now)
Easy changes with immediate impact potential.

### High-Impact Changes (Prioritize)
Bigger changes requiring effort but significant conversion improvement.

### Test Hypotheses
Ideas to A/B test with specific metrics and expected outcomes.
