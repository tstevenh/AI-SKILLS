### 10. Web Crawler (crawl-cli)

**Your competitive intelligence tool.** Extract competitor content for analysis.

```bash

# Get their pricing page
crawl-cli https://competitor.com/pricing -o "./research/pricing.md"

# Extract their blog for content gaps
crawl-cli https://competitor.com/blog --discover -m 50 -O "./content-research/"
```

**Use when:** You need raw competitor content for analysis.

## What Makes This Different

### Frameworks, Not Fluff

Every file contains actionable frameworks with templates. Not theory. Not inspiration. Executable workflows.

**Example from Direct Response Copy:**

```
HEADLINE FORMULA BANK

- How to [Achieve Desire] Without [Common Pain]
- The [Timeframe] [Transformation] System
- [Number] Ways to [Benefit] (Even If [Objection])
- Stop [Pain]. Start [Pleasure].
- What [Authority] Knows About [Topic] That You Don't
```

### Interconnected System

Skills reference each other. Outputs from one become inputs to another. The handoff protocol ensures nothing falls through the cracks.

**Handoff Template:**

```markdown
## Context from [Previous Skill]

**Product:** [One-line description]
**Target Customer:** [Who it's for]
**Key Differentiator:** [What makes it unique]
**Current Status:** [Where you are in the journey]
```

### Battle-Tested Depth

These aren't surface-level prompts. They're the result of real-world execution:

- **Reddit Marketing**: 1,780 lines across 2 files (core execution + advanced reference)
- **Email Sequences**: Complete welcome, nurture, and conversion sequences with examples
- **SEO Content**: Two-phase system (planning + writing) with quality checklists
- **Visual Intelligence**: Deep psychology and anti-generic AI image techniques

### Context Compression

Each skill knows what to pass forward and what to leave behind. No context overload.

| From                | Pass This                | Not This                  |
| ------------------- | ------------------------ | ------------------------- |
| idea-extraction     | Core strategy (1 page)   | Full interview transcript |
| competitor-analysis | Key gaps (3-5 bullets)   | Full battle cards         |
| keyword-research    | Top cluster + 5 keywords | Full spreadsheet          |
