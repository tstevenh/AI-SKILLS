## 1. R.I.P Formula

**Performance:** 43K+ likes (highest performer)
**Use When:** Disruption angle, democratization, cost savings

### Core Structure

```
R.I.P [Expensive Solution].

You don't need a $[BIG NUMBER] [professional] anymore.

You can now [do X] using [AI tool/solution].

Here are the exact [#] [prompts/ways] I use to replicate [professional output] for free:
```

### Generic Template

```
R.I.P _______________.

You don't need a $___________ _______________ anymore.

You can now _______________ using [solution].

Here are the exact [#] prompts I use to _______________ for free:
```

### Variations

**Operations Version:**

```
R.I.P [Traditional Operations Tool].

You don't need a $[Amount]/month [SaaS subscription] anymore.

You can now [automate operation] using [solution].

Here are the exact 5 implementations I use to replace [expensive tool] for free:
```

**Consultant Version:**

```
R.I.P [Expensive Service Provider].

You don't need a $[Amount]/hour [consultant type] anymore.

You can now get [deliverable type] using [AI solution].

Here's the exact prompt that generates [output type]:
```

### Quick Starters

- R.I.P Project Management Software
- R.I.P Customer Support Platforms
- R.I.P Document Management Systems
- R.I.P Workflow Automation Tools
- R.I.P Data Analysis Services
- R.I.P Content Writing Agencies

## 2. I Tested Formula

**Performance:** 25K+ likes avg
**Use When:** Comparisons, tool battles, validation

### Core Structure

```
I tested [Tool A] and [Tool B] with same critical prompts.

The results will blow your mind.

[Tool A]          Vs.          [Tool B]

(Video demos included)
```

### Generic Template

```
I tested _______________ and _______________ with [same test].

The results will blow your mind.

[Tool A]          Vs.          [Tool B]

(Evidence included)
```

### Variations

**Operations Version:**

```
I tested [Tool 1] and [Tool 2] for [specific operational task].

The results will blow your mind.

[Tool 1]          Vs.          [Tool 2]

(Screenshots + metrics included)
```

**Speed Test Version:**

```
I timed [Tool A] vs [Tool B] on [specific task].

[Tool A]: [time]
[Tool B]: [time]

Winner: [Tool] by [X]x

Here's exactly what happened:
```

**Quality Test Version:**

```
Same prompt. Same task. Different tools.

[Tool A] output:
[Brief description]

[Tool B] output:
[Brief description]

The difference? [Specific insight]
```

## Thread Structure

### Opening Tweet (Hook)

```
[SHOCKING STATEMENT]

[Context that deepens hook]

[Promise of value with number]

[Link or thread continuation indicator]
```

### Tweet 2 (Setup - Optional)

```
Let me tell you what [traditional approach] actually does:

1. [Step]
2. [Step]
3. [Step]

But guess what?

AI can now do [X%] of that [instantly/for free].

Let me show you how:
```

### Tweets 3-10 (Value Delivery)

```
1/ [Method Name]

[What it does]

Prompt to use:

"[EXACT COPY-PASTE PROMPT]"

[Expected outcome]

---

2/ [Method Name]

[What it does]

Implementation:

"[EXACT STEPS]"

[Real results from your work]
```

### Final Tweet (CTA)

```
[Transformation statement]

[Product/Service CTA with urgency if applicable]

I hope you've found this helpful.

Follow me @[handle] for more.

Like/Repost if you can
```

## Formatting Rules Quick Reference

### Line Breaks

Always between distinct thoughts:

```
GOOD:
"Hook line.

Context line.

Promise line."

BAD:
"Hook line. Context line. Promise line."
```

### Emoji Usage

- 1-2 strategic emojis per tweet max
- Function emojis only (alerts, checks, shock indicators)
- Never decorative spam

### ALL CAPS

- Only in prompts/placeholders
- Never for emphasis in regular text

### Numbers

- Always specific ("$80K" not "a lot")
- Ranges when testing ("5-10x faster")
- Exact counts for lists ("7 ways" not "several ways")

### Character Limits

- Tweet max: 280 characters
- Hook target: Under 200 characters
