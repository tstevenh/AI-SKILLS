# Content Atomizer

One piece of content should become ten. The best creators don't create more--they distribute better.

This skill transforms any source content into platform-optimized assets. Not generic repurposing. Platform-native content that performs.

**The math:** A single blog post can become 1 LinkedIn carousel + 2 LinkedIn text posts + 1 Twitter thread + 3 single tweets + 2 Instagram carousels + 1 Reel script + 2 TikTok scripts + 1 YouTube Short script + 2 Reddit posts = 15+ pieces of content from one source.

---

## The Core Job

Transform source content into **platform-native assets** that:

- Match each platform's algorithm signals
- Use format-specific best practices
- Include hooks proven to stop the scroll
- Feel native, not repurposed

---

## Input Types

### What Can Be Atomized

| Source Type           | Best Outputs                                        | Atomization Potential   |
| --------------------- | --------------------------------------------------- | ----------------------- |
| **Blog Post**         | All platforms                                       | High (lots of material) |
| **Newsletter**        | LinkedIn, Twitter, Instagram, Reddit                | High                    |
| **Podcast Episode**   | Short-form video, threads, carousels                | Very High               |
| **Long-form Video**   | Shorts, Reels, TikToks, carousels                   | Very High               |
| **Webinar/Talk**      | All platforms                                       | Very High               |
| **Case Study**        | LinkedIn, Twitter threads, Reddit                   | High                    |
| **Data/Research**     | Carousels, threads, single posts, Reddit            | Medium-High             |
| **Framework/Process** | Carousels, threads, video scripts, Reddit tutorials | High                    |

### What to Extract

From any source, identify:

1. **The Core Insight** - The one thing someone should remember
2. **Supporting Points** - 3-7 sub-points that build the argument
3. **Stories/Examples** - Concrete illustrations
4. **Data Points** - Stats, numbers, proof
5. **Contrarian Takes** - Opinions that challenge conventional wisdom
6. **Actionable Steps** - What someone can do with this
7. **Quotable Lines** - Punchy phrases that stand alone

## Platform Playbooks

### LinkedIn

**Algorithm Signals (2025):**

- **Dwell time** - How long people spend reading (still #1)
- **Topic authority** - Consistent niche posting builds algorithmic trust
- **Golden hour** - First 60 minutes determines reach to 2nd/3rd-degree connections
- **Relevance over recency** - Older posts (2-3 weeks) show if highly relevant
- **Authentic engagement** - AI detects engagement pods via comment velocity
- **Native content** - Posts without links get significantly more reach

**Reality Check:** Organic views down ~50%, engagement down ~25% as LinkedIn prioritizes quality over quantity. Post 2-3x/week max, not daily.

**Optimal Specs:**

| Format    | Specs                            | Notes                           |
| --------- | -------------------------------- | ------------------------------- |
| Carousel  | 5-10 slides, 1080x1350px         | Highest dwell time, save-worthy |
| Text Post | 1,200-1,500 chars, 3-line hook   | Depth over frequency            |
| Document  | PDF, 10-15 pages max             | Good for frameworks             |
| Video     | 30-90 seconds, captions required | Lower reach than text/carousels |

**Algorithm Signals (2025):**

- **Session impact** - Does your video keep people on YouTube longer?
- **Viewer satisfaction** - YouTube now uses surveys + behavior to estimate quality
- **Negative feedback** - "Not interested," skips, very low retention hurt you
- **Topical authority** - Channels focused on clear topics get recommended more

**2025 Changes:** AI-driven hyper-personalization (device, time of day, habits). Older evergreen videos get revived when topics trend again. Stronger emphasis on authority, depth, and "entity-rich" content.

**Optimal Specs:**

| Format    | Specs                               | Performance                            |

### Reddit

**Why Reddit Matters:**

- Domain Authority 91/100 - Posts rank on Google within 24 hours
- 90% of users trust Reddit for product discovery
- Top training source for LLMs (GEO value)
- Reaches developers not on LinkedIn or Twitter
- Authentic community discussions have long shelf life

**Algorithm Signals:**

- **Upvote velocity** - Early upvotes in first hour critical
- **Comment engagement** - Discussion signals quality to algorithm
- **Community fit** - Content must match subreddit culture
- **Account karma** - Higher karma = more trust from automod
- **Post timing** - Weekday mornings (US time) typically best

**Optimal Specs:**

| Format        | Specs                                   | Performance                     |
| ------------- | --------------------------------------- | ------------------------------- |
| Text Post     | 500+ words, formatted with headers      | Best for tutorials, discussions |
| Link Post     | Compelling title + valuable destination | Best for sharing resources      |
| Image/Gallery | Informative visuals with context        | Best for data, comparisons      |

#### Reddit Post Types That Work

**Problem-Solution Post:**

```
Title: "How I [solved specific problem]"

Body structure:
- Pain point description (relatable problem)
- Failed attempts (builds credibility)
- Solution discovery (natural mention)
- Results with specifics (proof)
- Open question (drives engagement)
```

**Comparison/Alternative Post:**

```
Title: "Compared [number] [tools/approaches] - here's what I found"

Body structure:
- Testing methodology
- Objective criteria
- Honest pros/cons for each
- Use case recommendations
- Request for community feedback
```

**Tutorial/How-To Post:**

```
Title: "Setting up [specific workflow/system]"

Body structure:
- Clear objective statement
- Step-by-step instructions
- Screenshots/code examples
- Tips for common pitfalls
- Community discussion prompt
```

#### Reddit Writing Guidelines

**DO:**

- Lead with value, not promotion
- Use casual, conversational tone
- Include specific numbers and results
- Format with bullet points and headers
- End with questions to spark discussion
- Acknowledge limitations honestly

**DON'T:**

- Use marketing jargon
- Over-promote or link in every post
- Ignore negative feedback
- Post identical content across subreddits
- Pretend to be a user if you're the creator
- Delete posts that don't perform well

#### Reddit Safe Promotion Ratios

```
Engagement Mix (10:1 Rule):
- 10 helpful comments/posts
- 1 promotional mention

Content Mix:
- 80% pure value content
- 15% soft mentions
- 5% direct promotion
```

#### SEO/GEO Value from Reddit

**For Google Ranking:**

1. Target "Best [X]" keywords in post titles
2. Include comparison terms naturally
3. Use specific, descriptive titles
4. Write comprehensive posts (500+ words)
5. Encourage discussion in comments

**For AI Training (GEO):**

1. Participate in high-traffic threads
2. Provide detailed, quotable responses
3. Include brand name in helpful context
4. Be the authoritative voice on your topic
5. Target threads LLMs use as sources

## The Atomization Workflow

### Step 1: Extract

From your source content, pull out:

```
CORE INSIGHT:
[One sentence that captures the main point]

SUPPORTING POINTS:
1. [Point + brief explanation]
2. [Point + brief explanation]
3. [Point + brief explanation]
4. [Point + brief explanation]
5. [Point + brief explanation]

STORIES/EXAMPLES:
- [Story 1]
- [Story 2]

DATA/PROOF:
- [Stat 1]
- [Stat 2]

QUOTABLE LINES:
- "[Quote 1]"
- "[Quote 2]"

CONTRARIAN TAKES:
- [Take 1]
- [Take 2]
```

### Step 2: Map to Platforms

| Content Element              | Best Platforms            | Best Formats                                 |
| ---------------------------- | ------------------------- | -------------------------------------------- |
| Core insight                 | All                       | Single posts, hooks                          |
| Supporting points (together) | LinkedIn, Twitter, Reddit | Carousel, thread, post                       |
| Individual points            | All                       | Single posts                                 |
| Stories                      | Instagram, TikTok, Reddit | Reels, Stories, text posts                   |
| Data points                  | LinkedIn, Twitter, Reddit | Image posts, carousels, discussions          |
| Quotable lines               | Twitter, Instagram        | Quote graphics                               |
| Contrarian takes             | Twitter, TikTok, Reddit   | Single tweets, video hooks, discussion posts |

### Step 3: Transform

For each platform, apply:

1. **Format** - Use the templates above
2. **Hook** - Platform-specific hook formula
3. **Length** - Match platform norms
4. **CTA** - Platform-appropriate action
5. **Voice** - Adjust formality (LinkedIn > Reddit > Instagram > TikTok)

### Step 4: Sequence

**Optimal posting sequence:**

1. **Reddit post** - Day 0-1 (SEO value, authentic discussion)
2. **LinkedIn carousel** - Day 1 (longest shelf life)
3. **Twitter thread** - Day 1-2 (good for discussion)
4. **Instagram carousel** - Day 2-3 (repurpose LinkedIn design)
5. **TikTok/Reel** - Day 3-4 (needs video production)
6. **YouTube Short** - Day 4-5 (can repurpose TikTok)
7. **Single posts** - Ongoing (extract individual points)

## Anti-Patterns: What Not to Do

### Don't:

1. **Copy-paste across platforms**

   - Each platform has different norms
   - Cross-posted content performs 40-60% worse

2. **Use the same hook everywhere**

   - LinkedIn hooks differ from TikTok hooks differ from Reddit hooks
   - Adjust energy and format per platform

3. **Ignore platform-native features**

   - No hashtags on LinkedIn carousels
   - Always use captions on video
   - Instagram needs visual-first thinking
   - Reddit needs authentic discussion tone

4. **Post everything at once**

   - Stagger across days/weeks
   - Gives each piece room to perform

5. **Forget the CTA**
   - Every platform piece needs a clear next action
   - But make it platform-appropriate

### Do:

1. **Lead with the best hook per platform**
2. **Adapt length to platform norms**
3. **Use native formatting (threads, carousels, etc.)**
4. **Front-load value (especially for video)**
5. **Create platform-specific visuals when possible**

## The Test

Good atomization means:

1. **Each piece stands alone** - Makes sense without the source
2. **Each piece feels native** - Doesn't feel "repurposed"
3. **Hooks match the platform** - Right energy, right format
4. **Value is front-loaded** - Best stuff first
5. **CTAs are appropriate** - Platform-native actions
6. **Quality over quantity** - 5 great pieces > 15 mediocre ones

## Content Pillars Framework

## Engagement Strategy

### Proactive Engagement

Engagement isn't just responding; it's actively participating.

**Daily routine (30 min):**
1. Respond to all comments on your posts (5 min)
2. Comment on 5-10 posts from target accounts (15 min)
3. Share/repost with added insight (5 min)
4. Send 2-3 DMs to new connections (5 min)

**Quality comments:**
- Add new insight, not just "Great post!"
- Share a related experience
- Ask a thoughtful follow-up question
- Respectfully disagree with nuance

### Building Relationships

- Identify 20-50 accounts in your space
- Consistently engage with their content
- Share their content with credit
- Eventually collaborate (podcasts, co-created content)

### Handling Negative Comments

- Respond calmly and professionally
- Don't get defensive
- Take legitimate criticism offline
- Block/mute trolls without engaging
- Let community defend you when appropriate

## Reverse Engineering Viral Content

Instead of guessing what works, systematically analyze top-performing content in your niche.

### The 6-Step Framework

#### 1. NICHE ID: Find Top Creators

Identify 10-20 creators in your space who consistently get high engagement:

**Selection criteria:**
- Posting consistently (3+ times/week)
- High engagement rate relative to follower count
- Audience overlap with your target market
- Mix of established and rising creators

**Where to find them:**
- LinkedIn: Search by industry keywords, check "People also viewed"
- Twitter/X: Check who your target audience follows and engages with
- Use tools like SparkToro, Followerwonk, or manual research

#### 2. SCRAPE: Collect Posts at Scale

Gather 500-1000+ posts for analysis:

**Tools:**
- Apify - LinkedIn scraper, Twitter scraper actors
- Phantom Buster - Multi-platform automation
- Manual collection - For smaller datasets

**Data to collect:**
- Post text/content
- Engagement metrics (likes, comments, shares, saves)
- Post format (text-only, carousel, video, image)
- Posting time/day
- Hook/first line
- Topic/theme

#### 3. ANALYZE: Extract What Actually Works

Sort and analyze the data to find patterns:

**Quantitative analysis:**
- Rank posts by engagement rate
- Identify top 10% performers
- Look for format patterns (do carousels outperform?)
- Check timing patterns (best days/times)

**Qualitative analysis:**
- What hooks do top posts use?
- How long are high-performing posts?
- What emotional triggers appear?
- What formats repeat?

#### 4. PLAYBOOK: Codify Patterns

Document repeatable patterns:

**Hook patterns to codify:**
```
Pattern: "I [unexpected action] and [surprising result]"
Example: "I stopped posting daily and my engagement doubled"
Why it works: Curiosity gap + contrarian

Pattern: "[Specific number] [things] that [outcome]:"
Example: "7 pricing mistakes that cost me $50K:"
Why it works: Specificity + loss aversion
```

**Format patterns:**
- Carousel: Hook slide -> Problem -> Solution steps -> CTA
- Thread: Hook -> Promise -> Deliver -> Recap -> CTA
- Story post: Hook -> Setup -> Conflict -> Resolution -> Lesson

#### 5. LAYER VOICE: Apply Direct Response Principles

Take proven patterns and make them yours:

**"Smart friend who figured something out"**
- Write like you're texting advice to a friend
- Share discoveries, not lectures
- Use "I found that..." not "You should..."

**Specific > Vague**
```
Bad: "I made good revenue"
Good: "I made $47,329"

Bad: "It took a while"
Good: "It took 47 days"
```

**Short. Breathe. Land.**
- One idea per sentence
- Use line breaks liberally
- Let important points stand alone

#### 6. CONVERT: Turn Attention into Action

Bridge from engagement to business results:

**Soft conversions:**
- Newsletter signups in bio/comments
- Free resource offers in follow-up comments
- DM triggers ("Comment X and I'll send you...")
- Profile visits -> optimized profile with clear CTA

**Direct conversions:**
- Link in comments (not post body on LinkedIn)
- Contextual product mentions within valuable content
- Case study posts that naturally showcase your work

### Reverse Engineering Checklist

- [ ] Identified 10-20 top creators in niche
- [ ] Collected 500+ posts for analysis
- [ ] Ranked by engagement rate
- [ ] Documented top 10 hook patterns
- [ ] Documented top 5 format patterns
- [ ] Documented top 5 CTA patterns
- [ ] Created voice guidelines
- [ ] Built template library from patterns
- [ ] Set up tracking for your own content performance

## Analytics & Optimization

### Metrics That Matter

**Awareness:**
- Impressions
- Reach
- Follower growth rate

**Engagement:**
- Engagement rate (engagements / impressions)
- Comments (higher value than likes)
- Shares/reposts
- Saves (Instagram)

**Conversion:**
- Link clicks
- Profile visits
- DMs received
- Leads/conversions attributed

### What to Track Weekly

- [ ] Top 3 performing posts (why did they work?)
- [ ] Bottom 3 posts (what can you learn?)
- [ ] Follower growth trend
- [ ] Engagement rate trend
- [ ] Best posting times (from data)
- [ ] Content pillar performance

### Optimization Actions

**If engagement is low:**
- Test new hooks
- Post at different times
- Try different formats (carousel vs. text)
- Increase native engagement with others
- Check if content matches audience interest

**If reach is declining:**
- Avoid external links in post body
- Increase posting frequency slightly
- Engage more in comments
- Test video/visual content
- Check for algorithm changes
