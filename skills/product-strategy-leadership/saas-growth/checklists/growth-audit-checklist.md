# Complete SaaS Growth Audit Checklist

## Introduction

This comprehensive growth audit checklist helps you evaluate every aspect of your SaaS business. Use it quarterly or when preparing for a strategic initiative. Each section includes evaluation criteria, red flags to watch for, and suggested actions.

---

## 1. Product-Market Fit Audit

### Core PMF Indicators

**Usage Patterns:**
- [ ] DAU/MAU ratio is healthy for your product type
- [ ] Users return without prompting (organic retention)
- [ ] Core feature is used by >60% of active users
- [ ] Time-to-value is decreasing over time
- [ ] Users express disappointment at the thought of losing access

**Customer Feedback:**
- [ ] NPS score >40 (or improving)
- [ ] "Very disappointed" score >40% (Sean Ellis test)
- [ ] Customers refer without incentive
- [ ] Feature requests show product evolution, not gaps
- [ ] Support tickets are about "how to do more" not "why doesn't it work"

**Growth Signals:**
- [ ] Organic signups growing month-over-month
- [ ] Word-of-mouth attribution is meaningful
- [ ] Churn is stabilizing or decreasing
- [ ] Cohort retention curves flatten (not decline forever)

### Red Flags

⚠️ **Warning signs you may not have PMF:**
- High trial signups but low activation
- Users use product once and never return
- Lots of "nice to have" but no "must have"
- Growth only when you push (ads, sales)
- High churn regardless of customer success efforts

### Actions

| Issue | Action |
|-------|--------|
| Low Sean Ellis score | More customer interviews |
| Poor retention | Activation deep dive |
| Low organic growth | PMF validation needed |
| Feature gaps | Prioritize roadmap |

## 2. Acquisition Audit

### Channel Performance

**Organic Search:**
- [ ] Keyword rankings improving month-over-month
- [ ] Organic traffic growing
- [ ] Blog driving meaningful signups
- [ ] Technical SEO issues resolved
- [ ] Content calendar executing consistently

**Paid Acquisition:**
- [ ] CAC within target by channel
- [ ] ROAS positive on mature campaigns
- [ ] Creative refresh happening regularly
- [ ] Landing pages optimized for each channel
- [ ] Attribution working properly

**Referral/Viral:**
- [ ] Referral program exists and is promoted
- [ ] Viral coefficient calculated
- [ ] Share mechanisms easy and visible
- [ ] Referred users have good LTV

**Outbound/Sales:**
- [ ] ICP clearly defined
- [ ] Outbound sequences tested and optimized
- [ ] SDR productivity at benchmark
- [ ] Pipeline coverage is 3-4x quota

### Funnel Analysis

| Stage | Healthy Rate | Your Rate | Status |
|-------|--------------|-----------|--------|
| Visitor → Signup | >2% | __% | [ ] |
| Signup → Activation | >40% | __% | [ ] |
| Activation → PQL | >20% | __% | [ ] |
| PQL → Paid | >25% | __% | [ ] |
| Trial → Paid | >15% | __% | [ ] |

### Red Flags

⚠️ **Warning signs in acquisition:**
- CAC increasing while quality decreasing
- Single channel dependency (>60% from one source)
- Declining organic while increasing paid
- Low conversion despite high traffic
- Sales cycle lengthening

### Actions

| Issue | Action |
|-------|--------|
| High CAC | Channel optimization or new channels |
| Low conversion | Landing page and funnel audit |
| Single channel risk | Channel diversification |
| Poor quality leads | ICP and targeting review |

## 3. Activation Audit

**First Run Experience:**
- [ ] Time-to-value measured and tracked
- [ ] Clear activation metric defined
- [ ] Onboarding checklist or guidance exists
- [ ] Progress indicators visible
- [ ] Success states celebrated

**Email/Messaging:**
- [ ] Welcome sequence implemented
- [ ] Emails triggered by behavior
- [ ] Open rates >30%
- [ ] Click rates >5%
- [ ] Emails drive activation actions

**In-App Guidance:**
- [ ] Tooltips for key features
- [ ] Empty states guide next action
- [ ] Help documentation accessible
- [ ] Live chat or support available
- [ ] Video tutorials for complex features

### Red Flags

⚠️ **Warning signs in activation:**

## 4. Retention Audit

### Retention Metrics

**User Retention:**
- [ ] D1, D7, D30 retention tracked
- [ ] Retention curves flatten over time
- [ ] Cohort analysis shows improvement
- [ ] Segment retention understood

**Customer Retention:**
- [ ] Logo churn <5% monthly (SMB) or <2% (Enterprise)
- [ ] Churn reasons documented
- [ ] At-risk customers identified proactively
- [ ] Win-back campaigns running

### Engagement Analysis

| Metric | Healthy | Your Rate | Status |
|--------|---------|-----------|--------|
| DAU/MAU | Product-dependent | __% | [ ] |
| WAU/MAU | >50% | __% | [ ] |
| Core feature adoption | >60% | __% | [ ] |
| Weekly sessions | >2 | __ | [ ] |

### Red Flags

⚠️ **Warning signs in retention:**
- Retention curves never flatten
- Increasing churn despite CS efforts
- Low feature adoption

| Issue | Action |
|-------|--------|

## 5. Revenue Audit

### Pricing and Packaging

**Pricing:**
- [ ] Value metric aligned with customer value
- [ ] Pricing tested within last 12 months
- [ ] Competitive pricing understood
- [ ] Willingness-to-pay data collected
- [ ] Price localization implemented

**Packaging:**
- [ ] Plans differentiated clearly
- [ ] Upgrade path logical
- [ ] Enterprise tier exists if needed
- [ ] Add-ons and modules priced
- [ ] Annual discount offered

|--------|-----------|-----------|--------|

## Summary

### Using This Audit

1. **Schedule quarterly**: Block time for thorough review
2. **Involve leadership**: Get cross-functional input
3. **Be honest**: Red flags are opportunities
4. **Prioritize actions**: Can't fix everything at once
5. **Track progress**: Compare to previous audits

### Priority Order

When resources are limited, prioritize:
1. **Retention** → Keep what you have
2. **Activation** → Convert what you get
3. **Revenue** → Monetize effectively
4. **Acquisition** → Then scale
