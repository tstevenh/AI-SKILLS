# SaaS Tech Stack

## Introduction

The right tools help your team work efficiently and make data-driven decisions. But tool sprawl and poor integration create friction. This guide covers the essential SaaS tech stack categories and top options in each.

---

## 1. CRM (Customer Relationship Management)

### Options

| Tool | Best For | Price Range |
|------|----------|-------------|
| Salesforce | Enterprise, complex needs | $25-300/user/mo |
| HubSpot | SMB, integrated marketing | Free-$120/user/mo |
| Pipedrive | Sales-focused SMB | $15-100/user/mo |
| Close | Inside sales | $25-130/user/mo |

### Selection Criteria

- Team size and complexity
- Integration requirements
- Marketing integration needs
- Budget constraints
- Customization requirements

## Stack by Stage

### Early Stage ($0-$1M)

**Essentials:**
- CRM: HubSpot Free or Attio
- Analytics: PostHog or Mixpanel Free
- Billing: Stripe
- Support: Intercom or Help Scout
- Email: Customer.io or Mailchimp

### Growth Stage ($1M-$10M)

**Add:**
- Revenue Analytics: ChartMogul
- Customer Success: Vitally
- Advanced CRM: HubSpot Pro or Salesforce
- Marketing Automation: HubSpot or ActiveCampaign

### Scale Stage ($10M+)

**Upgrade:**
- Enterprise CRM: Salesforce
- Enterprise CS: Gainsight
- Advanced Analytics: Amplitude
- Enterprise Support: Zendesk Enterprise

## Summary: Tech Stack Framework

### Selection Principles

1. **Start simple**: Add complexity as needed
2. **Integration matters**: Tools should work together
3. **Don't over-tool**: Consolidate where possible
4. **Plan for scale**: Choose tools that grow with you

### Stack Checklist

- [ ] CRM selected and implemented
- [ ] Analytics tracking user behavior
- [ ] Billing handling subscriptions
- [ ] Support tickets managed
- [ ] Marketing automation running
- [ ] Team communication tools set
