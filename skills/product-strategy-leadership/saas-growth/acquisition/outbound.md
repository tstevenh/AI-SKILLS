# Outbound Sales for SaaS

## Introduction

Outbound sales—proactively reaching out to potential customers—is essential for most B2B SaaS companies, especially those targeting mid-market and enterprise. While PLG can drive significant growth, outbound accelerates deals, captures accounts that won't self-serve, and allows you to target ideal customers directly.

---

## 1. When to Add Outbound Sales

### Signals You're Ready

**Product Signals:**
- Product-market fit validated
- Successful customer references
- Clear value proposition
- Onboarding process works

**Market Signals:**
- Target accounts identifiable
- Decision-makers reachable
- ACV justifies sales cost
- Competition requires proactive approach

### ACV Thresholds

| ACV | Sales Motion |
|-----|--------------|
| < $5K | Self-serve only |
| $5K-$15K | Light-touch sales assist |
| $15K-$50K | Inside sales |
| $50K-$150K | Field sales |
| > $150K | Enterprise sales |

**Rule of thumb:** CAC should be < 1/3 of ACV for first-year payback.

## 2. SDR/AE Structure

### Team Roles

**SDR (Sales Development Rep):**
- Outbound prospecting
- Lead qualification
- Meeting booking
- Reports to sales or marketing

**AE (Account Executive):**
- Runs demos
- Handles negotiations
- Closes deals
- Owns quota

**Typical Ratios:**
- 1 AE : 2-3 SDRs (enterprise)
- 1 AE : 1-2 SDRs (mid-market)
- 0 SDRs (SMB, AE does own prospecting)

### SDR Metrics

| Metric | Target |
|--------|--------|
| Emails/day | 50-100 |
| Calls/day | 40-60 |
| LinkedIn touches/day | 20-30 |
| Meetings booked/month | 15-25 |
| SQL conversion | 50-70% |

### AE Metrics

| Metric | SMB | Mid-Market | Enterprise |
|--------|-----|------------|------------|
| Quota/month | $30-50K | $50-100K | $100-200K |
| Deals/month | 8-15 | 3-6 | 1-2 |
| Win rate | 20-30% | 25-35% | 30-40% |
| Avg deal size | $3-5K | $15-30K | $75-200K |

## 3. Outbound Sequences

### Multi-Channel Sequence

**Day 1:** Email 1 (personalized intro)
**Day 3:** LinkedIn connect + note
**Day 5:** Email 2 (value add)
**Day 7:** Call attempt 1
**Day 10:** Email 3 (social proof)
**Day 12:** LinkedIn message
**Day 14:** Call attempt 2
**Day 17:** Email 4 (breakup)

### Email Templates

**Email 1: Personalized Intro**
```
Subject: [Trigger event] at [Company]

Hi [Name],

Noticed [specific trigger - new role, funding, job posting].
[Company] seems focused on [priority].

At [Your Company], we help [similar companies] achieve [result].
[Customer name] increased [metric] by [X%] after switching.

Worth a 15-minute chat?

[Your name]
```

**Email 2: Value Add**
```
Subject: Re: [Previous subject]

Hi [Name],

Thought you might find this useful—[relevant resource/insight].

[Brief insight relevant to their business]

Would love to share how [similar company] tackled this.

[Your name]
```

**Email 3: Social Proof**
```
Subject: How [Similar Company] achieved [result]

Hi [Name],

[Similar company in their industry] was struggling with [problem].

In 3 months with [Your Product]:
• [Result 1]
• [Result 2]
• [Result 3]

Seeing similar challenges at [Company]. Open to learning more?

[Your name]
```

**Email 4: Breakup**
```
Subject: Closing the loop

Hi [Name],

Haven't heard back—totally get it if timing isn't right.

I'll close out this thread. If [problem] becomes a priority, 
here's my calendar: [link]

Best,
[Your name]
```

### Cold Call Framework

**Opening (10 seconds):**
```
"Hi [Name], this is [Your name] from [Company].
Did I catch you at an okay time?"
```

**Reason for call (20 seconds):**
```
"I'm reaching out because we work with [similar companies] 
to help them [solve problem]. I noticed [trigger/research]
and wanted to see if this resonates."
```

**Discovery question:**
```
"How are you currently handling [problem area]?"
```

**Bridge to meeting:**
```
"Based on what you shared, I think it'd be worth 15 minutes
to show you how we've helped [similar company] with this.
How's [specific time]?"
```

## 4. Demo Optimization

### Demo Structure

**1. Rapport (2-3 min)**
- Small talk
- Agenda confirmation
- Understand their time

**2. Discovery (10-15 min)**
- Current situation
- Pain points
- Goals
- Decision process

**3. Demo (15-20 min)**
- Tailored to their needs
- Focus on value, not features
- Show, don't tell

**4. Q&A (5-10 min)**
- Address concerns
- Handle objections

**5. Next Steps (5 min)**
- Clear commitment
- Calendar invite
- Follow-up plan

### Demo Best Practices

**Before:**
- Research the company
- Review prior interactions
- Prepare tailored demo
- Test your tech

**During:**
- Ask questions first
- Customize to their use case
- Tell stories, not features
- Involve multiple stakeholders

**After:**
- Send summary email
- Include next steps
- Share relevant resources
- Book follow-up meeting

## 5. Sales Playbooks

### Qualification Framework (BANT)

- **Budget:** Do they have budget?
- **Authority:** Is this person a decision-maker?
- **Need:** Do they have the problem you solve?
- **Timeline:** When would they implement?

### Objection Handling

**"It's too expensive"**
```
"I understand budget is a concern. Let me ask—when you compare
the cost to [problem impact], how does that math look?

[Customer] was in a similar spot. They calculated that 
[problem] was costing them $X/month. Our solution at $Y/month 
gave them [Z] ROI."
```

**"We're using [Competitor]"**
```
"Got it—[Competitor] is solid for [use case].

What made you take this call? Usually when people in your 
position chat with us, it's because [common gap].

Is that something you're experiencing?"
```

**"Not the right time"**
```
"Totally fair. Out of curiosity, what would need to change 
for this to become a priority?

[Pause and listen]

Makes sense. How about we check back in [timeframe]?
I'll send some resources in the meantime that might be helpful."
```

**"I need to talk to my team"**
```
"Absolutely—who else should be involved in this decision?

Would it be helpful if I joined a call with the team?
I can answer questions and make sure everyone has the 
context they need."
```

## Summary: Outbound Sales Framework

### Building Outbound

**Phase 1: Founder-Led**
- Founders do initial outbound
- Validate messaging
- Learn objections
- Build playbook

**Phase 2: First Hire**
- Hire SDR or AE
- Document processes
- Create templates
- Build tools

**Phase 3: Scale**
- Add team members
- Specialize roles
- Optimize process
- Track metrics

- [ ] Outbound sequences created
