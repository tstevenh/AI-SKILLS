# Finding Broken Links

## What Is Broken Link Building?

Broken link building involves finding dead links on websites, creating content that could replace the dead resource, and reaching out to suggest your content as a replacement. It's a win-win: the site owner fixes their broken link, and you get a backlink.

### Why It Works

**Value proposition is clear:**
You're helping fix a problem on their site—not just asking for a favor.

**Authority transfer:**
The broken link already exists; you're inheriting its authority placement.

**Less competitive:**
Requires more effort than cold outreach, so fewer people do it.

## Finding Broken Links at Scale

### Method 1: Ahrefs Broken Backlinks

**Step-by-step:**

1. **Site Explorer → Enter competitor or target domain**
2. **Go to Backlink profile → Broken backlinks**
3. **Filter for relevant keywords**
4. **Review broken pages and linking sites**

**What you're looking for:**
- Pages that linked to now-dead content
- Topics relevant to your expertise
- Sites worth earning links from (DR 20+)

### Method 2: Ahrefs Content Explorer

**Find broken content in your niche:**

1. Go to Content Explorer
2. Search for topic keywords
3. Filter: "404 not found" or check for dead pages
4. Find pages that had links
5. Use Backlinks tab to see who linked

### Method 3: Competitor Broken Links

**Step-by-step:**

1. Identify competitors
2. Check each for broken backlinks
3. Find pages linking to competitors' dead content
4. Create replacement content
5. Pitch your content

**Why this works:**
If they linked to competitor content that's now dead, they're likely to link to your replacement.

### Method 4: Resource Page Broken Links

**Step-by-step:**

1. Find resource pages in your niche
2. Check each resource link manually (or with tools)
3. Identify broken links
4. Create replacement content
5. Pitch your replacement + resource inclusion

### Method 5: Wikipedia Broken Links

**Wikipedia [citation needed] and dead references:**

1. Go to Wikipedia articles in your niche
2. Look for "dead link" or "citation needed" tags
3. If you have authoritative content, suggest it as a source
4. Follow Wikipedia editing guidelines carefully

**Caution:** Wikipedia has strict guidelines. Don't spam; genuinely contribute.

## Tools for Finding Broken Links

### Browser Extensions

**Check My Links (Chrome)**
- Highlight all links on a page
- Shows broken links in red
- Quick manual checking

**Link Miner (Chrome)**
- Check all links on page
- Export results
- More features than Check My Links

**Broken Link Checker (Chrome)**
- Similar functionality
- Highlights broken links
- Reports status codes

### Dedicated Tools

**Ahrefs**
- Best for competitor analysis
- Shows broken backlinks
- Historical data available

**Screaming Frog**
- Crawl entire sites
- Find all broken links
- Export for outreach

**Semrush**
- Backlink audit for broken links
- Competitor analysis
- Domain-level reporting

**BrokenLinkCheck.com (Free)**
- Check any page for broken links
- Free but limited
- Good for spot checks

### Using Screaming Frog for Broken Links

**Step-by-step:**

1. Open Screaming Frog
2. Enter target URL
3. Crawl the site
4. Go to Response Codes
5. Filter for 4XX client errors
6. Export broken link report
7. Review for outreach opportunities

## Systematic Broken Link Prospecting

### Workflow 1: Competitor Focus

**Week 1:**
1. List 10 competitors
2. Run broken backlink analysis on each
3. Compile prospects in spreadsheet

**Week 2:**
1. Identify most common dead content types
2. Create (or adapt) replacement content
3. Begin outreach

### Workflow 2: Resource Page Focus

**Daily:**
1. Find 5-10 resource pages using search operators
2. Run broken link checker on each
3. Add opportunities to spreadsheet
4. Follow up with outreach

### Workflow 3: Niche-Wide Analysis

**Monthly:**
1. Use Content Explorer to find dead content in niche
2. Check backlinks to dead content
3. Prioritize by DR of linking sites
4. Create strategic replacement content
5. Execute outreach campaign

## Qualifying Broken Link Opportunities

### Opportunity Evaluation

**Is it worth pursuing?**

| Factor | Minimum | Ideal |
|--------|---------|-------|
| Linking site DR | 20+ | 40+ |
| Page still active | Yes | Yes |
| Relevance | Related | Exact match |
| Link placement | Visible | Contextual |
| Competition | Low | None |

### Questions to Ask

1. **Is the linking page still active?** (Dead page = no opportunity)
2. **Is the link still on the page?** (May have been removed)
3. **Can I create quality replacement content?**
4. **Is the site worth earning a link from?**
5. **Is this realistically achievable?**

### Red Flags

- Linking page is also dead
- Site is low quality
- Topic is completely unrelated to your business
- Content required would be massive undertaking
- Site has already replaced the link

## Building Your Broken Link Prospect List

### Spreadsheet Structure

| Column | Data |
|--------|------|
| Linking Page URL | Where the broken link appears |
| Linking Page DR | Domain rating |
| Broken Link URL | The dead link |
| Broken Content Topic | What was linked |
| Your Replacement | URL (existing or to create) |
| Contact | Who to reach out to |
| Status | Pipeline stage |
| Notes | Additional context |

### Prioritization

**Priority 1 (target immediately):**
- DR 40+
- You have existing replacement content
- Clear topic match

**Priority 2 (worth pursuing):**
- DR 25-39
- Can create replacement content easily
- Related topic

**Priority 3 (batch later):**
- DR 20-24
- Requires significant content creation
- Tangential relevance

### Volume Expectations

**Realistic numbers:**
- Broken links found: Many (depending on niche)
- Qualified opportunities: 20-30% of found
- Outreach response rate: 10-20%
- Link conversion rate: 5-15% of qualified

**For 10 broken link placements/month:**
- Find ~100 broken links
- Qualify ~25-30 opportunities
- Send ~25-30 outreach emails
- Earn ~5-10 links

## Advanced Broken Link Tactics

### Reclaim Your Own Broken Links

**Check your own backlinks:**
1. In Ahrefs, check your Broken backlinks (pages linking to YOUR dead pages)
2. If you have replacement content, reach out
3. Update the link to current content
4. If no replacement, create one

### Monitor Competitors for New Broken Links

**Set up alerts:**
1. Track competitor backlinks
2. Watch for pages going 404
3. Be first to reach out with replacement
4. Speed advantage matters

### Target Specific High-Value Pages

**Strategic approach:**
1. Identify high-authority pages that SHOULD link to you
2. Check their outbound links for dead resources
3. If dead link exists in your space, create replacement
4. Pitch specifically as replacement

## Summary

Finding broken links requires:

1. **Tools** - Ahrefs, Screaming Frog, browser extensions
2. **Systematic process** - Regular prospecting routine
3. **Qualification** - Not all broken links are worth pursuing
4. **Organization** - Track opportunities and status
5. **Content readiness** - Have replacement content ready

With qualified broken link opportunities identified, the next step is effective outreach to convert them into links.
