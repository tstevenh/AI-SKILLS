# Content Creation Checklist

## Pre-Writing

### Assignment Review
- [ ] Target site identified
- [ ] Publication guidelines read
- [ ] Topic approved by site
- [ ] Target URL for link confirmed
- [ ] Anchor text strategy clear
- [ ] Deadline understood

### Site Research
- [ ] Read 3-5 recent articles on site
- [ ] Understand their voice/style
- [ ] Note formatting preferences
- [ ] Identify internal linking opportunities

### Topic Research
- [ ] Competitive content reviewed
- [ ] Key points to cover identified
- [ ] Unique angle determined
- [ ] Supporting data gathered
- [ ] Expert quotes sourced (if applicable)

---

## Content Structure

### Outline Checklist
- [ ] Compelling title/headline
- [ ] Hook in introduction
- [ ] Clear section headings (H2/H3)
- [ ] Logical flow of ideas
- [ ] Link placement(s) identified
- [ ] Strong conclusion

### Length Requirements
- [ ] Meets minimum word count: ___
- [ ] Not excessively long
- [ ] Substantial enough to provide value

---

## Writing Quality

### Content Quality
- [ ] Original content (not plagiarized)
- [ ] Not AI-generated junk
- [ ] Provides genuine value
- [ ] Actionable insights included
- [ ] Examples and evidence included
- [ ] Addresses reader's questions/problems

### Writing Style
- [ ] Matches site's voice
- [ ] Grammar and spelling correct
- [ ] Sentences clear and concise
- [ ] Paragraphs short (2-4 sentences)
- [ ] Active voice (mostly)
- [ ] No jargon without explanation

### Formatting
- [ ] Clear headings used
- [ ] Bullet points where appropriate
- [ ] Bold/italics for emphasis
- [ ] Images included (if required)
- [ ] Tables/charts (if applicable)
- [ ] Mobile-friendly formatting

---

## Link Integration

### Link Placement
- [ ] Link is contextual (within content)
- [ ] Link flows naturally in sentence
- [ ] Surrounding content is relevant
- [ ] Not in first paragraph (too obvious)
- [ ] Not forced or awkward

### Anchor Text
- [ ] Matches agreed anchor
- [ ] Natural in context
- [ ] Not over-optimized
- [ ] Fits sentence structure

### Internal Links (to their site)
- [ ] 2-3 internal links included
- [ ] Links to relevant content
- [ ] Natural integration
- [ ] Adds value for readers

### Other Outbound Links
- [ ] To authoritative sources
- [ ] Not to competitors
- [ ] All links working

---

## Author Bio

### Bio Requirements
- [ ] Meets site's bio format
- [ ] Within word limit
- [ ] Professional tone
- [ ] Credential included
- [ ] Company link included (if allowed)
- [ ] Social link (if allowed/requested)

---

## Pre-Submission Review

### Self-Editing
- [ ] Read aloud for flow
- [ ] All facts verified
- [ ] Links tested and working
- [ ] Guidelines compliance checked
- [ ] Word count confirmed
- [ ] Spelling/grammar final check

### Peer/Manager Review
- [ ] Content reviewed by second person
- [ ] Feedback incorporated
- [ ] Final approval received

---

## Submission

### File Format
- [ ] Correct format (Google Doc, Word, etc.)
- [ ] Clean formatting
- [ ] Links embedded (not raw URLs)
- [ ] Images included with credits
- [ ] Author bio attached

### Submission Method
- [ ] Following site's submission process
- [ ] All required materials included
- [ ] Sent to correct contact
- [ ] Confirmation requested

---

## Post-Submission

### Revision Handling
- [ ] Responded promptly to requests
- [ ] Changes made as requested
- [ ] Quality maintained through revisions
- [ ] Timeline expectations managed

### Publication
- [ ] Publication date confirmed
- [ ] Live post verified
- [ ] Link checked (see QA checklist)
- [ ] Documented in tracking

## Common Issues to Avoid

### Content Issues
- [ ] NOT thin content (<1000 words)
- [ ] NOT rehashed/common knowledge only
- [ ] NOT promotional/salesy
- [ ] NOT keyword-stuffed
- [ ] NOT off-topic

### Link Issues
- [ ] NOT forced anchor text
- [ ] NOT over-linked (1-2 links max)
- [ ] NOT link in first paragraph
- [ ] NOT unrelated to surrounding content

### Format Issues
- [ ] NOT walls of text
- [ ] NOT inconsistent formatting
- [ ] NOT broken links
- [ ] NOT missing required elements
