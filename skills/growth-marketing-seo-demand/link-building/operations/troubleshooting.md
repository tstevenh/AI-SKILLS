# Link Building Troubleshooting Guide

## Campaign Performance Issues

### Problem: Low Response Rates

**Symptoms:**
- Response rate below 8%
- Few replies despite high outreach volume
- Mostly negative or no responses

**Diagnostic questions:**
1. Are you targeting the right sites?
2. Is your email getting delivered?
3. Is your pitch compelling?
4. Are you personalizing enough?
5. Is timing appropriate?

**Solutions by root cause:**

**If targeting is wrong:**
- Review site qualification criteria
- Check relevance scores
- Analyze successful placements for patterns
- Adjust prospecting focus

**If deliverability is the issue:**
- Check spam score (use mail tester tools)
- Warm up email account properly
- Reduce sending volume
- Remove spam trigger words
- Verify domain authentication (SPF, DKIM)

**If pitch quality is low:**
- A/B test subject lines
- Improve personalization
- Strengthen value proposition
- Shorten email length
- Add social proof

**If timing is off:**
- Test different days (Tue-Thu best)
- Test different times (morning often best)
- Avoid holidays and major news events

### Problem: Low Conversion After Response

**Symptoms:**
- Good response rates but few placements
- Negotiations often fail
- Interest expressed but never closes

**Diagnostic questions:**
1. Is pricing aligned with expectations?
2. Is your content quality good enough?
3. Are you handling objections effectively?
4. Is your follow-up process working?

**Solutions by root cause:**

**If pricing is the issue:**
- Research market rates for target sites
- Adjust budget or targets
- Negotiate better terms
- Offer volume commitments

**If content quality is lacking:**
- Review rejection feedback
- Improve content standards
- Invest in better writers
- Study site's content closely

**If objection handling fails:**
- Create objection response scripts
- Train team on common objections
- Practice negotiation scenarios
- Provide more value in responses

**If follow-up is weak:**
- Implement systematic follow-up
- Automate reminders
- Vary follow-up approaches
- Know when to stop

### Problem: Link Quality Below Expectations

**Symptoms:**
- DR/DA below targets
- Low traffic on linking sites
- Links from irrelevant sites
- High rejection at QA

**Diagnostic questions:**
1. Are quality standards clearly defined?
2. Is prospecting following standards?
3. Are shortcuts being taken?
4. Is QA catching issues?

**Solutions by root cause:**

**If standards are unclear:**
- Document specific minimums
- Create qualification scorecards
- Train team on quality criteria
- Review examples together

**If prospecting isn't following standards:**
- Audit prospecting output regularly
- Add quality checkpoints
- Adjust incentives for quality
- Provide coaching

**If shortcuts are being taken:**
- Identify why (pressure? incentives?)
- Address root causes
- Reinforce quality culture
- Implement controls

**If QA isn't catching issues:**
- Review QA process
- Add pre-outreach quality check
- Train QA on standards
- Increase QA sample size

### Problem: High Link Loss Rate

**Symptoms:**
- Links disappearing within 90 days
- Sites removing content
- Pages going offline

**Diagnostic questions:**
1. Are you working with low-quality sites?
2. Is your content adding value?
3. Are there technical issues?
4. Is monitoring catching losses early?

**Solutions by root cause:**

**If sites are low quality:**
- Raise quality thresholds
- Avoid sites that churn content
- Build longer-term relationships
- Vet site stability better

**If content isn't valuable:**
- Improve content quality
- Add more value to host
- Include internal links
- Make removal costly

**If technical issues:**
- Verify hosting stability
- Check for site migrations
- Monitor site health

**If monitoring is inadequate:**
- Set up loss alerts
- Implement regular checks
- Act quickly on losses
- Track recovery rates

## Outreach-Specific Issues

### Problem: Emails Going to Spam

**Symptoms:**
- Very low open rates (<10%)
- Bounce rates increasing
- Complaints from recipients

**Diagnostic checklist:**

**Email authentication:**
- [ ] SPF record configured
- [ ] DKIM signing enabled
- [ ] DMARC policy set
- [ ] Domain aged (not brand new)

**Email content:**
- [ ] No spam trigger words
- [ ] Text-to-HTML ratio balanced
- [ ] No suspicious links
- [ ] Personalization included

**Sending practices:**
- [ ] Warm-up completed
- [ ] Volume within limits
- [ ] List hygiene maintained
- [ ] Engagement is genuine

**Solutions:**
1. Use mail testing tools (mail-tester.com)
2. Warm up new domains slowly
3. Clean email lists regularly
4. Vary sending patterns
5. Monitor blacklist status

### Problem: Wrong Person Contacted

**Symptoms:**
- "I don't handle this" responses
- Forwarding requests
- Wrong department complaints

**Solutions:**
1. Research contacts more thoroughly
2. Target editor/content roles
3. Use LinkedIn for verification
4. Start with general inquiry when unsure
5. Build relationship before pitching

### Problem: Template Feeling "Obvious"

**Symptoms:**
- "This looks like a template" feedback
- Low response from experienced sites
- Unprofessional appearance

**Solutions:**
1. Increase personalization depth
2. Vary template structure
3. Add genuine insights
4. Reference specific content
5. Use natural language
6. Avoid obvious merge failures

### Problem: Negotiations Failing

**Symptoms:**
- Quoted prices too high
- Terms unacceptable
- Deals falling through late

**Solutions:**
1. Research market rates beforehand
2. Have clear budget limits
3. Practice negotiation tactics
4. Offer alternative value
5. Know when to walk away
6. Build rapport before negotiating

## Content Issues

### Problem: High Revision Rate

**Symptoms:**
- Most content requires edits
- Publishers requesting changes
- Rejection after submission

**Diagnostic questions:**
1. Are guidelines being followed?
2. Is content quality sufficient?
3. Is there style mismatch?
4. Are links integrated naturally?

**Solutions by root cause:**

**If guidelines aren't followed:**
- Create detailed briefs
- Review guidelines with writers
- Add guideline checklist to process
- Implement pre-submission review

**If quality is insufficient:**
- Improve writer vetting
- Provide better training
- Invest in editorial review
- Create quality standards document

**If style doesn't match:**
- Study target sites more closely
- Create style guides per site type
- Match voice in samples
- Get feedback from publishers

**If link integration is awkward:**
- Train on natural integration
- Review successful examples
- Provide placement guidance
- Edit for naturalness

### Problem: Content Taking Too Long

**Symptoms:**
- Missing deadlines
- Bottleneck in content production
- Campaign delays

**Solutions:**
1. Hire additional writers
2. Use content agencies
3. Create content templates
4. Build topic banks in advance
5. Improve briefing process
6. Set realistic timelines

### Problem: Low Content Acceptance Rate

**Symptoms:**
- Publishers rejecting content
- Multiple revision cycles
- Content not meeting standards

**Solutions:**
1. Raise content quality standards
2. Better qualify publishers first
3. Confirm topics before writing
4. Create detailed outlines
5. Implement editorial review
6. Learn from rejections

## Quality Control Issues

### Problem: Links Not As Described

**Symptoms:**
- Different anchor than agreed
- Nofollow when dofollow expected
- Poor placement location

**Solutions:**
1. Document agreements clearly
2. Verify before payment (when possible)
3. Request corrections promptly
4. Build relationships for compliance
5. Consider payment terms

### Problem: QA Missing Issues

**Symptoms:**
- Client complaints about quality
- Issues found after reporting
- Pattern of oversights

**Solutions:**
1. Expand QA checklist
2. Add multiple reviewers for samples
3. Implement QA training
4. Use verification tools
5. Create escalation path
6. Track QA accuracy metrics

### Problem: Links Disappearing Quickly

**Symptoms:**
- High 30/60/90 day loss rates
- Sudden content removal
- Site changes

**Solutions:**
1. Improve site vetting
2. Create higher-value content
3. Build actual relationships
4. Avoid known churning sites
5. Monitor more frequently
6. Have recovery process ready

## Emergency Response

### What to Do If Client Gets Penalized

**Immediate actions:**
1. Don't panic
2. Determine if actually penalty vs. algorithm
3. Audit link profile
4. Identify problematic links
5. Begin disavow process
6. Document timeline

**Communication:**
1. Inform client immediately
2. Explain situation honestly
3. Present remediation plan
4. Set realistic recovery timeline
5. Regular progress updates

**Prevention:**
1. Better quality standards
2. Diversified tactics
3. Regular link audits
4. Risk monitoring

### What to Do If Major Link Source Goes Down

**Immediate actions:**
1. Assess impact scope
2. Document all affected links
3. Communicate with clients
4. Find replacement sources
5. Adjust delivery timelines

**Long-term:**
1. Diversify sources
2. Don't over-rely on any single vendor
3. Build direct relationships
4. Have backup plans

## Summary

Troubleshooting link building issues requires:

1. **Diagnosis** - Understand root cause before fixing
2. **Systematic approach** - Check common issues methodically
3. **Documentation** - Track problems and solutions
4. **Prevention** - Learn from issues to prevent recurrence
5. **Communication** - Keep stakeholders informed
6. **Adaptation** - Be willing to change approach

Most problems have solutions—the key is identifying the right problem to solve.
