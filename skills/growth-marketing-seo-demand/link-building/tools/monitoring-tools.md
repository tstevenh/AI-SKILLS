# Monitoring Tools

## Why Link Monitoring Matters

Links can disappear. Pages get deleted, content gets rewritten, sites change ownership, and webmasters remove links. Without monitoring, you may lose links you paid for or worked hard to earn.

**The case for monitoring:**
- Detect lost links quickly
- Recover links before too late
- Track campaign health
- Measure ongoing link value
- Verify placements remain intact

## Link Monitoring Setup

### What to Monitor

**Your own links:**
- New backlinks acquired
- Lost backlinks
- Changes to link attributes
- Linking page status

**Competitor links:**
- Their new backlinks
- Link sources you're missing
- Tactics they're using
- Market movements

### Monitoring Frequency

| Link Type | Recommended Frequency |
|-----------|----------------------|
| Paid placements | Weekly |
| High-value earned links | Monthly |
| General profile | Monthly |
| Competitors | Monthly |

## Tool-Specific Monitoring

### Ahrefs Alerts

**Setting up backlink alerts:**

1. Go to Alerts → Backlinks
2. Add your domain
3. Set alert frequency (daily/weekly)
4. Choose new vs. lost links
5. Set email delivery

**Alert types:**
- New backlinks
- Lost backlinks
- New referring domains
- Lost referring domains

**For competitors:**
Set up same alerts for 3-5 key competitors.

**What to do with alerts:**
- New links: Verify quality, identify source
- Lost links: Investigate, attempt recovery
- Competitor links: Evaluate for replication

### Semrush Backlink Audit

**Features:**
- Automatic link monitoring
- Toxic link identification
- Change tracking
- Integration with Search Console

**Setup:**
1. Go to Backlink Audit
2. Connect domain
3. Configure monitoring settings
4. Set alert preferences

### Google Search Console

**What it shows:**
- Links Google knows about
- Top linking sites
- Top linked pages
- Internal links

**Limitations:**
- Not comprehensive
- Delayed data
- No historical tracking
- No alerts

**Best use:**
Cross-reference with other tools; use for what Google actually sees.

## Lost Link Recovery

### Why Links Get Lost

**Technical reasons:**
- Page deleted
- Site restructured
- Domain changed
- Content rewritten

**Editorial reasons:**
- Content cleaned up
- Links deemed outdated
- Competition from alternatives
- Policy changes

**Intentional removal:**
- Site owner changed mind
- New ownership
- Spam cleanup
- Link selling crackdown

### Recovery Process

**Step 1: Detect**
Set up alerts to catch lost links quickly.

**Step 2: Verify**
Confirm link is actually gone (tools can be wrong).

**Step 3: Investigate**
- Is the page still live?
- Was content changed?
- Is the link just moved?

**Step 4: Reach out**
If link was removed:

```
Hi [Name],

I noticed the link to [your URL] was removed from your article [title].

If there was an issue with our content or the link, I'd love to understand so we can address it.

If it was accidental or during a site update, would you consider restoring it?

Thanks,
[Your Name]
```

**Step 5: Document**
Track lost links, reasons, and recovery success.

### Recovery Rate Expectations

**Typical recovery rates:**
- Accidental removal: 50-70% recovery
- Content rewrite: 20-40% recovery
- Intentional removal: 5-15% recovery
- Site change/deletion: Very low recovery

## Competitor Monitoring

### What to Watch

**Link acquisition:**
- What new links are they earning?
- From which sites?
- What tactics are apparent?

**Link losses:**
- Are they losing links too?
- From where?
- Opportunities for you?

**Strategy indicators:**
- Types of links they prioritize
- Anchor text patterns
- Link velocity

### Competitive Intelligence Workflow

**Monthly competitor review:**

1. Check each competitor's new backlinks
2. Filter for relevant, high-quality links
3. Identify sites that might link to you
4. Note tactics being used
5. Prioritize opportunities

**Quarterly strategic review:**

1. Analyze link gap vs. competitors
2. Assess what's working for them
3. Identify gaps in your strategy
4. Adjust approach accordingly

## Building Your Monitoring System

### Monitoring Checklist Setup

**Weekly:**

## Summary

Effective link monitoring requires:

1. **Tool setup** - Ahrefs/Semrush alerts at minimum
2. **Regular review** - Weekly for paid, monthly for general
3. **Lost link recovery** - Detect and act quickly
4. **Competitor intelligence** - Know what they're doing
5. **Systematic tracking** - Spreadsheet or dedicated tool
6. **Regular reporting** - Monthly at minimum

Monitoring protects your investment and provides intelligence for strategy. Don't skip it.
