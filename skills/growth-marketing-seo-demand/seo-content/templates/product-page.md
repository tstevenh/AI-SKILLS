## Product Description Framework

### The AIDA Structure

**A - Attention**: Hook with the main benefit
```
[Product] is the [superlative] way to [achieve outcome].
```

**I - Interest**: Build interest with features
```
With [feature], you can [benefit]. Plus, [another feature] 
means [another benefit].
```

**D - Desire**: Create desire with outcomes
```
Imagine [positive outcome]. That's what [product] delivers.
```

**A - Action**: Clear call to action
```
[CTA Button]
```

## Schema Markup Template

```json
{
  "@context": "https://schema.org",
  "@type": "Product",
  "name": "[Product Name]",
  "description": "[Product description]",
  "image": "[Product image URL]",
  "brand": {
    "@type": "Brand",
    "name": "[Brand Name]"
  },
  "offers": {
    "@type": "Offer",
    "price": "[Price]",
    "priceCurrency": "USD",
    "availability": "https://schema.org/InStock",
    "url": "[Product page URL]"
  },
  "aggregateRating": {
    "@type": "AggregateRating",
    "ratingValue": "[Average rating]",
    "reviewCount": "[Number of reviews]"
  }
}
```
