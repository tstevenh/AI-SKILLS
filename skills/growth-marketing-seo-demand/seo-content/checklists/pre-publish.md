# Pre-Publish Checklist

> Run through this checklist before publishing ANY content. Catch issues before they cost you rankings.

---

## ⚡ Quick Check (Under 5 Minutes)

- [ ] **Title tag** is 50-60 characters with primary keyword
- [ ] **Meta description** is 105-155 characters with CTA
- [ ] **URL** is short, descriptive, includes keyword
- [ ] **H1** includes primary keyword and matches title
- [ ] **Primary keyword** appears in first 100 words
- [ ] **Images** have descriptive alt text
- [ ] **"Last updated" date** is visible
- [ ] **At least one CTA** is present

---

## 📝 Content Quality

### Structure
- [ ] One H1 per page
- [ ] Logical H2/H3 hierarchy
- [ ] Paragraphs are 2-3 sentences max
- [ ] Visual break every 300 words (image, list, heading)
- [ ] Table of contents (if >1,500 words)

### Readability
- [ ] 6th-8th grade reading level (check with Hemingway)
- [ ] Active voice predominant
- [ ] No walls of text
- [ ] Jargon explained or avoided
- [ ] Content reads naturally aloud

### Depth
- [ ] Matches or exceeds competitor depth
- [ ] Covers subtopics competitors cover
- [ ] Answers "People Also Ask" questions
- [ ] Includes original insights/examples
- [ ] FAQ section with 3-5 questions

---

## 🔑 Keyword Optimization

### Primary Keyword
- [ ] In title tag (near beginning)
- [ ] In H1
- [ ] In URL slug
- [ ] In first 100 words
- [ ] In at least one H2

### Secondary Keywords
- [ ] Included in H2s where natural
- [ ] Sprinkled throughout body copy
- [ ] Used in FAQ questions

### Semantic SEO
- [ ] Related terms included naturally
- [ ] LSI keywords present
- [ ] Topic covered comprehensively (not just keyword)

---

## 🔗 Links

### Internal Links
- [ ] Links to 3-5 relevant internal pages
- [ ] Uses descriptive anchor text
- [ ] Links to and from pillar/hub content
- [ ] No broken internal links

### External Links
- [ ] Links to authoritative sources
- [ ] Statistics and claims are cited
- [ ] External links open in new tab (optional)
- [ ] No links to low-quality sites

---

## 🖼️ Media

### Images
- [ ] At least one relevant image
- [ ] All images have alt text with description
- [ ] Images are compressed (fast loading)
- [ ] Original images preferred over stock (E-E-A-T)
- [ ] Screenshots are current

### Other Media
- [ ] Videos embedded if relevant
- [ ] Tables used for comparisons
- [ ] Infographics if helpful

---

## 🛡️ E-E-A-T Signals

### Experience
- [ ] Personal experience shared where relevant
- [ ] Original examples/case studies included
- [ ] "I tested/used/did" language where true

### Expertise
- [ ] Author byline with name
- [ ] Author bio with credentials
- [ ] Link to author page

### Authoritativeness
- [ ] Citations to authoritative sources
- [ ] Data backed by research
- [ ] Expert quotes if relevant

### Trust
- [ ] Information is accurate and current
- [ ] Disclosure for affiliate/sponsored (if applicable)
- [ ] Contact information accessible on site

---

## 🔧 Technical

### Schema Markup
- [ ] Article schema added
- [ ] FAQPage schema (if FAQ section exists)
- [ ] HowTo schema (if how-to content)
- [ ] Product/Review schema (if applicable)

### Page Experience
- [ ] Mobile-friendly (preview on phone)
- [ ] Fast loading (under 3 seconds)
- [ ] No intrusive interstitials
- [ ] HTTPS

### Indexing
- [ ] Page is indexable (no noindex tag)
- [ ] Canonical URL is correct
- [ ] In XML sitemap

---

## 📊 For AI Search (GEO)

- [ ] Clear definitions for key terms
- [ ] Listicle format (if "Best X" topic)
- [ ] Year in title (if time-sensitive)
- [ ] Self-contained sections with complete answers
- [ ] Structured data that AI can extract

---

## 🚀 Final Steps

### Before Publishing
- [ ] Proofread for typos and grammar
- [ ] All links tested and working
- [ ] Preview on mobile device
- [ ] Social sharing image set (Open Graph)

### After Publishing
- [ ] Submit URL to Google Search Console
- [ ] Add to internal linking from relevant pages
- [ ] Share on social channels
- [ ] Schedule promotion activities

---

## Content-Type Specific Checks

### For Listicles ("Best X")
- [ ] Number in title
- [ ] Year in title
- [ ] Quick comparison table at top
- [ ] Each item has full review
- [ ] Each item has unique "best for" category
- [ ] Methodology section included

### For How-To Guides
- [ ] Steps are clearly numbered
- [ ] Each step is one action
- [ ] Screenshots/visuals for complex steps
- [ ] Common mistakes section
- [ ] HowTo schema added

### For Comparison Posts
- [ ] TL;DR verdict in intro
- [ ] Comparison table at top
- [ ] Both products covered fairly
- [ ] "When to choose" sections
- [ ] Clear final verdict

### For Landing Pages
- [ ] Above-fold CTA visible
- [ ] Problem → Solution flow
- [ ] Social proof present
- [ ] Pricing clear
- [ ] FAQ addresses objections

---

## Quick Fixes for Common Issues

| Issue | Quick Fix |
|-------|-----------|
| Title too long | Cut to 55-60 chars, keep keyword |
| Keyword not in title | Add primary keyword near start |
| Walls of text | Break into 2-3 sentence paragraphs |
| No images | Add 1 relevant image with alt text |
| No internal links | Add 3-5 links to related content |
| Missing FAQ | Add 3-5 questions from PAA |
| No date | Add "Last updated: [Date]" |
| Missing author | Add byline and link to author page |
