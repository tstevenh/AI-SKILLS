# A/B Test Design

Master statistical experimentation for optimizing websites, apps, and marketing campaigns.

## What It Does

This skill provides comprehensive knowledge for designing, running, and analyzing A/B tests and multivariate experiments. It covers statistical foundations, sample size calculations, Bayesian vs Frequentist approaches, and industry-specific testing methodologies.

## Features

- Statistical foundations (hypothesis testing, confidence intervals, power analysis)
- Sample size and test duration calculators
- Bayesian and Frequentist testing approaches
- Multi-armed bandits and sequential testing
- Segmentation analysis and personalization
- Testing prioritization frameworks (PIE, ICE, RICE)
- Industry-specific testing strategies (SaaS, e-commerce, B2B)
- Common testing mistakes and how to avoid them

## Installation

1. Copy the `ab-test-design/` folder into your OpenClaw skills directory
2. Restart OpenClaw

## Usage

Use this skill when:
- Planning a new A/B test or experiment
- Calculating required sample sizes
- Analyzing test results for statistical significance
- Choosing between Bayesian and Frequentist approaches
- Designing multi-variate tests
- Building a testing roadmap and prioritizing experiments

## Requirements

- Basic understanding of statistics (helpful but not required)
- Access to A/B testing platform (Optimizely, VWO, Google Optimize, or custom)
- Sufficient traffic volume to reach statistical significance
