## Customization Guide

### Replace These Placeholders

| Placeholder | Replace With |
|-------------|--------------|
| [Newsletter Name] | Your actual newsletter name |
| [X]+ | Your subscriber count |
| [frequency] | Daily/Weekly/Bi-weekly |
| [audience] | Your target reader |
| [benefit] | Your core value |
| [Name] | Testimonial names |
| [Logo] | Company logos |

### Testing Your Copy

1. Read aloud - does it flow?
2. Is the value clear in 5 seconds?
3. Is the CTA obvious?
4. Would you sign up?
5. Show to 3 people - do they understand?

### A/B Testing Priority

| Test | Impact |
|------|--------|
| Headline | Highest |
| Social proof | High |
| CTA button text | Medium |
| Subheadline | Medium |
| Value descriptions | Lower |

## Quick Optimization Checklist

Before launching any landing page:

- [ ] Headline states clear value
- [ ] Subheadline expands on promise
- [ ] Form is above the fold
- [ ] CTA text is action-oriented
- [ ] Social proof is visible
- [ ] Mobile version works
- [ ] Page loads fast (<3 sec)
- [ ] Tracking is installed
- [ ] Thank you page is set up
