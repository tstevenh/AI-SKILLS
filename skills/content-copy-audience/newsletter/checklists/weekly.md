# Weekly Newsletter Operations Checklist

A comprehensive weekly checklist for consistent newsletter operations. Use this to maintain quality and never miss critical tasks.

---

## Weekly Overview

```
WEEKLY SCHEDULE AT A GLANCE

Monday: Planning + Start Content
Tuesday: Drafting
Wednesday: Editing + Sponsor Coordination  
Thursday: Final Prep
Friday: Send + Analysis
Weekend: Review + Ideas
```

---

## Monday: Planning and Start

### Morning (1-2 hours)

**Review Last Week:**
- [ ] Check last send's performance
- [ ] Note open rate vs. benchmark
- [ ] Note click rate vs. benchmark
- [ ] Review any notable replies
- [ ] Document learnings

**This Week's Content:**
- [ ] Confirm topic from calendar
- [ ] Review outline/research
- [ ] Identify key points to cover
- [ ] Note any timely angles
- [ ] Gather resources/links

**Growth Check:**
- [ ] Review subscriber count
- [ ] Check growth sources
- [ ] Note referral activity
- [ ] Review paid acquisition (if running)

### Afternoon (2-3 hours)

**Start Drafting:**
- [ ] Write opening hook
- [ ] Draft main sections
- [ ] Outline supporting points
- [ ] Don't edit while writing
- [ ] Save progress

---

## Tuesday: Drafting Day

### Full Day Focus (3-4 hours)

**Complete Draft:**
- [ ] Finish all sections
- [ ] Add supporting links
- [ ] Write CTA
- [ ] Include all elements:
  - [ ] Opening hook
  - [ ] Main content
  - [ ] Sponsor section placeholder
  - [ ] CTA
  - [ ] Footer

**First Self-Edit:**
- [ ] Read through once
- [ ] Note major issues
- [ ] Don't polish yet

**Sponsor Check:**
- [ ] Confirm this week's sponsor
- [ ] Check creative deadline
- [ ] Send reminder if needed

---

## Wednesday: Editing + Sponsors

### Morning (1-2 hours)

**Content Editing:**
- [ ] Read draft aloud
- [ ] Fix awkward sentences
- [ ] Improve transitions
- [ ] Strengthen opening
- [ ] Clarify confusing points
- [ ] Cut unnecessary words

### Afternoon (1-2 hours)

**Sponsor Management:**
- [ ] Receive sponsor creative (deadline today)
- [ ] Review creative for:
  - [ ] Copy within limits
  - [ ] Appropriate content
  - [ ] Working links
  - [ ] Tracking parameters
- [ ] Request changes if needed
- [ ] Confirm final creative

**Subject Line:**
- [ ] Write 3-5 subject line options
- [ ] Write preview text options
- [ ] Select A/B test candidates

---

## Thursday: Final Preparation

### Morning (1-2 hours)

**Final Edit:**
- [ ] Complete final read-through
- [ ] Verify all facts
- [ ] Check all links work
- [ ] Confirm sponsor content final
- [ ] Finalize subject line choices
- [ ] Lock preview text

### Afternoon (1 hour)

**Platform Setup:**
- [ ] Load content into platform
- [ ] Format properly
- [ ] Add images
- [ ] Insert sponsor content
- [ ] Set up A/B test (if using)
- [ ] Schedule send time

**Pre-Send Checks:**
- [ ] Send test email to yourself
- [ ] Check on desktop
- [ ] Check on mobile
- [ ] Verify all links
- [ ] Confirm sponsor links
- [ ] Review formatting
- [ ] Check unsubscribe link

---

## Friday: Send Day

### Pre-Send (30 min before)

**Final Verification:**
- [ ] One more link check
- [ ] Confirm schedule is set
- [ ] Review subject line
- [ ] Deep breath

### At Send Time

**Monitor:**
- [ ] Confirm send started
- [ ] Watch for immediate issues
- [ ] Monitor deliverability

### Post-Send (Throughout Day)

**Engagement:**
- [ ] Respond to replies
- [ ] Monitor social mentions
- [ ] Note any issues reported
- [ ] Track initial metrics

**Sponsor Report:**
- [ ] Note impressions (opens)
- [ ] Track sponsor clicks
- [ ] Calculate CTR
- [ ] Prepare preliminary report

---

## Weekend: Review + Ideas

### Saturday or Sunday (1-2 hours)

**Performance Review:**
- [ ] Pull full metrics
- [ ] Compare to benchmarks
- [ ] Note what worked
- [ ] Note what didn't
- [ ] Document learnings

**Idea Collection:**
- [ ] Review saved articles/ideas
- [ ] Check industry news
- [ ] Review reader feedback
- [ ] Add to idea backlog

**Next Week Prep:**
- [ ] Review next week's topic
- [ ] Start research if needed
- [ ] Confirm sponsor status
- [ ] Mental preparation

## Emergency Protocols

### If You're Behind Schedule

**Wednesday and not started:**
- Use simpler format (curation, list)
- Pull from idea backlog
- Delegate editing if possible

**Thursday and not ready:**
- Send Friday instead of usual day
- Communicate if regular readers expect specific day
- Prioritize quality over timing

**Can't send this week:**
- Skip is better than garbage
- Brief note to subscribers (optional)
- Double effort next week
- Don't make it a habit

### If Something Goes Wrong

**Wrong link sent:**
- Send correction immediately
- Apologize briefly
- Fix and move on

**Sponsor issue:**
- Contact sponsor immediately
- Offer make-good
- Document for future

**Major error in content:**
- Assess severity
- Correct if important
- Brief apology if needed
- Learn and improve process

## Customizing Your Checklist

### Adapt to Your Schedule

**If you send daily:**
- Batch content creation (Monday-Tuesday)
- Daily: Edit + send (30 min/day)
- Weekly: Review + planning (2 hours)

**If you send bi-weekly:**
- Week A: Create and send
- Week B: Growth focus, next content prep

**If you have a team:**
- Assign checklist items to team members
- Add review/approval steps
- Build in handoff points

### Add Your Specifics

**Customize for:**
- Your platform's requirements
- Your sponsor process
- Your growth activities
- Your unique workflows

**Review monthly:**
- What's working?
- What's missing?
- What can be simplified?
