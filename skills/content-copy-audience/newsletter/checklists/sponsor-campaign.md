# Sponsor Campaign Checklist: Complete Workflow

A comprehensive checklist for managing sponsor campaigns from inquiry to completion.

---

## Phase 1: Sales and Qualification

### Initial Inquiry

**When inquiry comes in:**
- [ ] Log in sponsor tracker
- [ ] Review company/product
- [ ] Check brand fit
- [ ] Assess audience alignment
- [ ] Review their budget signals

**Qualify the lead:**
- [ ] Relevant product for audience? (Y/N)
- [ ] Reputable company? (Y/N)
- [ ] Realistic budget? (Y/N)
- [ ] Timing works? (Y/N)
- [ ] Brand safety check passed? (Y/N)

**If not qualified:**
- [ ] Polite decline sent
- [ ] Note reason in tracker
- [ ] Consider for future if applicable

### Discovery

**On discovery call/email:**
- [ ] Understand campaign goals
- [ ] Confirm budget range
- [ ] Discuss timing/dates
- [ ] Explain your offering
- [ ] Share media kit
- [ ] Answer questions

**Document:**
- [ ] Contact name and email
- [ ] Company name
- [ ] Campaign goals
- [ ] Budget
- [ ] Preferred timing
- [ ] Notes

---

## Phase 2: Proposal and Negotiation

### Create Proposal

**Proposal includes:**
- [ ] Newsletter overview
- [ ] Audience data
- [ ] Sponsorship option(s)
- [ ] Pricing
- [ ] Available dates
- [ ] Specifications
- [ ] Next steps

**Pricing calculation:**
```
CPM × (Subscribers × Open Rate / 1000) = Price

Example:
$50 CPM × (25,000 × 40% / 1000) = $500
```

### Send Proposal

- [ ] Personalize for sponsor
- [ ] Attach media kit
- [ ] Include clear CTA
- [ ] Propose specific dates
- [ ] Set follow-up reminder

### Handle Objections

**Common objections:**

| Objection | Response |
|-----------|----------|
| "Too expensive" | Discuss value, offer smaller placement |
| "Need to think about it" | Provide case studies, set follow-up |
| "Wrong timing" | Book for future, stay in touch |
| "Need approval" | Provide materials for their pitch |

### Negotiation

- [ ] Understand their constraints
- [ ] Know your floor price
- [ ] Consider package deals
- [ ] Discuss payment terms
- [ ] Document agreed terms

## Phase 3: Booking and Confirmation

### Booking Confirmed

**When sponsor says yes:**
- [ ] Send insertion order/contract
- [ ] Include all agreed terms
- [ ] Set payment deadline
- [ ] Set creative deadline
- [ ] Confirm send date(s)

**Insertion order includes:**
- [ ] Sponsor company name
- [ ] Contact information
- [ ] Newsletter name
- [ ] Placement type
- [ ] Send date(s)
- [ ] Price
- [ ] Payment terms
- [ ] Creative specifications
- [ ] Creative deadline
- [ ] Cancellation policy

### Payment Processing

- [ ] Invoice sent
- [ ] Payment received
- [ ] Marked paid in tracker
- [ ] Thank you confirmation sent

**Payment terms:**
- [ ] Due before send (standard)
- [ ] Net 30 (established sponsors)
- [ ] Other: _____________

### Calendar Update

- [ ] Add to editorial calendar
- [ ] Block sponsor slot
- [ ] Note creative deadline
- [ ] Set reminder for follow-up

## Phase 4: Creative Collection

### Creative Request

**Send creative request (deadline - 5 days):**

```
Subject: [Newsletter Name] x [Company] — Creative needed by [Date]

Hi [Name],

Excited for your sponsorship on [Send Date]!

Here's what we need by [Deadline]:

REQUIRED:
• Copy: [Word count]
• Link: Your destination URL
• CTA: Button text

OPTIONAL:
• Image: [Dimensions] (PNG/JPG)

TRACKING:
Please include UTM parameters for tracking.

Questions? Reply here.

Best,
[Your name]
```

### Creative Receipt

**When creative received:**
- [ ] Download/save all assets
- [ ] Check copy word count
- [ ] Verify link works
- [ ] Confirm image specs
- [ ] Review for compliance
- [ ] Check tracking parameters

### Creative Review

**Review checklist:**
- [ ] Copy within word limit
- [ ] No prohibited content
- [ ] Claims are accurate
- [ ] Links work
- [ ] Image appropriate size
- [ ] On brand/appropriate
- [ ] CTA is clear

**If changes needed:**
- [ ] Send specific feedback
- [ ] Request revised deadline
- [ ] Follow up if not received

### Creative Approved

- [ ] Send approval confirmation
- [ ] Save final assets
- [ ] Mark creative complete in tracker

## Phase 5: Campaign Execution

### Pre-Send (Day Before)

**Newsletter preparation:**
- [ ] Insert sponsor content
- [ ] Format correctly
- [ ] Add sponsor image
- [ ] Insert tracking links
- [ ] Preview sponsor section

**Quality check:**
- [ ] Sponsor link works
- [ ] Tracking parameters present
- [ ] Image displays correctly
- [ ] Copy reads well
- [ ] Mobile preview checked

**Optional: Sponsor preview:**
- [ ] Send preview to sponsor
- [ ] Get final approval
- [ ] Make any last changes

### Send Day

**Before send:**
- [ ] Final link verification
- [ ] Confirm tracking
- [ ] All looks good

**At send:**
- [ ] Monitor delivery
- [ ] Watch for issues
- [ ] Document send time

**Post-send:**
- [ ] Note any issues
- [ ] Initial metrics check
- [ ] Respond to questions

## Problem Resolution

### Late Creative

**If creative not received by deadline:**
1. Send reminder (deadline - 2 days)
2. Call/urgent email (deadline - 1 day)
3. Offer to write copy (deadline day)
4. Reschedule if necessary (last resort)

### Creative Not Approved

**If creative doesn't meet guidelines:**
1. Specify exact issues
2. Provide examples of acceptable
3. Offer to rewrite if simple
4. Set revised deadline (24-48 hours)

### Sponsor Cancellation

**If sponsor cancels:**
1. Check cancellation policy
2. Process any refund due
3. Update calendar
4. Document reason
5. Consider for future

### Poor Performance

**If results disappoint sponsor:**
1. Review expectations set
2. Analyze what happened
3. Offer make-good if appropriate
4. Learn for future
5. Maintain relationship
