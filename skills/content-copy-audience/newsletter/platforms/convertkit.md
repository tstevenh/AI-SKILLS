# ConvertKit (Kit): Complete Platform Guide

ConvertKit, recently rebranded to Kit, is the creator-focused email platform known for powerful automation and commerce features. Built specifically for creators selling digital products, it excels at turning subscribers into customers.

---

## Platform Overview

### What ConvertKit/Kit Is

Kit is an email marketing platform designed for:
- Creators selling digital products and courses
- Publishers needing sophisticated automation
- Businesses with complex subscriber journeys
- Anyone monetizing through info products

### History and Positioning

Founded in 2013 by Nathan Barry, ConvertKit was built as "email marketing for professional bloggers" before evolving into a full creator economy platform. The 2024 rebrand to "Kit" reflects expansion beyond email into commerce, landing pages, and creator tools.

**Key Philosophy:**
- Creator-first design
- Powerful but accessible automation
- Commerce built-in, not bolted-on
- Subscriber relationships over list size

## Migration to Kit

### From Mailchimp

1. Export subscriber list from Mailchimp
2. Map fields during Kit import
3. Recreate automations in visual builder
4. Update website forms
5. Test sequences before going live

### From Substack

1. Export subscribers (CSV)
2. Import to Kit with tagging
3. Set up paid subscription (if migrating)
4. Recreate newsletter feed
5. Handle Stripe subscription migration

### From Beehiiv

1. Export subscriber list
2. Import with engagement data
3. Recreate automations (more powerful in Kit)
4. Set up commerce if needed
5. Update integrations

### Migration Support

- Free for Creator and Pro plans
- Assisted import process
- Form and landing page recreation
- Automation consultation

## Kit Tips and Best Practices

### Quick Wins

1. **Set up tagging strategy early** - Tags are Kit's power
2. **Create welcome sequence immediately** - First impressions matter
3. **Use visual automations** - Worth learning the builder
4. **Enable SparkLoop (Pro)** - Free referral program
5. **Segment by engagement** - Target your best subscribers

### Advanced Tactics

1. **Behavior-triggered automations** - Link clicks → sequences
2. **Product launch sequences** - Build anticipation
3. **Re-engagement campaigns** - Win back cold subscribers
4. **Survey → segmentation** - Ask, then personalize
5. **Conditional email content** - One email, personalized versions

### Common Mistakes

1. **Over-tagging** - Keep tag structure clean
2. **Complex automations too early** - Start simple
3. **Ignoring engagement scores** - Valuable data
4. **Not using sequences** - They're powerful
5. **Skipping A/B tests** - Easy optimization
