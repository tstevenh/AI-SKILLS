# YouTube Mastery

Expert-level YouTube strategy for creators building channels that grow and monetize.

## What It Does

This skill provides comprehensive YouTube knowledge—from algorithm optimization and content strategy to thumbnails, retention, and monetization. Build a channel that attracts views and generates revenue.

## Features

- YouTube algorithm deep dive (CTR + AVD focus)
- Title formulas and thumbnail frameworks
- Hook writing for first 30 seconds
- Video structure templates for retention
- Shorts strategy and algorithm
- Analytics interpretation and optimization
- Monetization beyond AdSense
- Content repurposing (1 video → 20 pieces)
- Team building and delegation

## Installation

1. Copy the `youtube/` folder into your OpenClaw skills directory
2. Restart OpenClaw

## Usage

Use this skill when:
- Planning YouTube content strategy
- Writing titles and designing thumbnails
- Scripting video hooks and content
- Optimizing for retention
- Analyzing channel analytics
- Creating Shorts content
- Planning monetization strategy
- Building a content team

## Requirements

- YouTube channel with access to analytics
- Video production equipment
- Thumbnail design tools (Canva, Photoshop)
- Editing software
- TubeBuddy or VidIQ (optional)
- Minimum 1000 subs/4000 hours for monetization
