# Pre-Publish Checklist

Complete this checklist before publishing any short-form video.

---

## Content Quality

### Hook (First 3 Seconds)
- [ ] First frame is visually compelling
- [ ] Text hook appears immediately (if using)
- [ ] Verbal hook is clear and engaging
- [ ] Would YOU stop scrolling for this?

### Content Body
- [ ] Value is delivered (entertainment, education, emotion)
- [ ] Pacing is appropriate (no dead air)
- [ ] Content makes sense without additional context
- [ ] Message is clear

### Ending
- [ ] Clean ending (not abrupt or dragging)
- [ ] Call to action included (follow, save, comment)
- [ ] Loop-friendly if possible

### Length
- [ ] Video is as short as possible while delivering value
- [ ] No unnecessary content
- [ ] Appropriate for platform (under 60s for Shorts, under 90s for Reels)

---

## Technical Quality

### Video
- [ ] Resolution is 1080x1920 (or higher)
- [ ] Aspect ratio is 9:16 (vertical)
- [ ] No blurriness or pixelation
- [ ] Good lighting (face visible, no harsh shadows)
- [ ] Framing is appropriate (subject in frame throughout)

### Audio
- [ ] Audio is clear and audible
- [ ] No background noise/echo
- [ ] Music/sound levels are balanced
- [ ] Voice is understandable

### Captions/Text
- [ ] Captions are accurate (if used)
- [ ] Text is readable (size, contrast, font)
- [ ] Text isn't covered by UI elements
- [ ] Timing is synced with audio

---

## Platform Optimization

### TikTok
- [ ] No watermarks from other platforms
- [ ] Trending sound used (if relevant)
- [ ] FYP-appropriate content
- [ ] Caption written (with CTA)

### YouTube Shorts
- [ ] Under 60 seconds
- [ ] #Shorts in description
- [ ] Title is SEO-friendly
- [ ] No competitor watermarks

### Instagram Reels
- [ ] Under 90 seconds
- [ ] Visual quality is high
- [ ] Instagram audio used (if using sound)
- [ ] Ready to share to Stories

---

## Caption and Hashtags

### Caption
- [ ] Hook/compelling first line
- [ ] Value or context added
- [ ] CTA included (save, share, comment, follow)
- [ ] Character limit appropriate for platform

### Hashtags
- [ ] 3-5 relevant hashtags (TikTok)
- [ ] 5-10 relevant hashtags (Instagram)
- [ ] #Shorts included (YouTube)
- [ ] Mix of broad and niche tags
- [ ] No irrelevant/spam hashtags

---

## Compliance

### Disclosures
- [ ] #ad or sponsorship label (if sponsored)
- [ ] Paid partnership tag enabled (if applicable)
- [ ] Any required brand mentions included
- [ ] Music properly credited (if required)

### Platform Guidelines
- [ ] No community guideline violations
- [ ] No copyrighted content (unless licensed)
- [ ] Age-appropriate content
- [ ] No misleading claims

---

## Final Review

### Self-Check
- [ ] Watched video from start to finish
- [ ] Checked on mobile (how viewers see it)
- [ ] Verified all text is readable
- [ ] Confirmed audio sounds good

### Comparison
- [ ] Quality matches or exceeds your typical content
- [ ] Would this perform in your niche?
- [ ] Does it feel authentically YOU?

### Gut Check
- [ ] Would you watch this to the end?
- [ ] Would you save or share this?
- [ ] Are you proud of this content?

---

## Quick Pre-Publish Summary

Before hitting publish, verify:

1. ✅ Hook in first 3 seconds
2. ✅ Value delivered throughout
3. ✅ Clean ending with CTA
4. ✅ Good video and audio quality
5. ✅ Captions accurate and readable
6. ✅ No watermarks from other platforms
7. ✅ Caption and hashtags complete
8. ✅ Disclosures included (if needed)
9. ✅ Passes the "would I watch this?" test

---

## Common Last-Minute Fixes

### Weak Hook
→ Add text overlay in first second
→ Reorder to start with strongest moment
→ Record new hook intro

### Too Long
→ Remove slowest section
→ Increase pace through jump cuts
→ Cut unnecessary context

### Poor Audio
→ Add background music to mask issues
→ Re-record voiceover
→ Use text instead of audio

### Unclear Message
→ Add explanatory text
→ Simplify the content
→ Cut confusing sections
