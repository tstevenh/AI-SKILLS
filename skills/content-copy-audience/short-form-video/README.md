# Short-Form Video Mastery

Create, optimize, and monetize short-form content across TikTok, YouTube Shorts, and Instagram Reels.

## What It Does

This skill provides comprehensive knowledge for short-form video creation—repurposing long-form content, creating native shorts, optimizing for algorithms, and building a consistent content engine.

## Features

- Platform algorithm deep dives (TikTok, Shorts, Reels)
- Hook frameworks for first 1-3 seconds
- Content repurposing from long-form (1 video → 20 shorts)
- Thumbnail and title optimization for shorts
- Posting strategies and timing
- AI clipping tools (OpusClip, Vidyo.ai)
- Cross-platform distribution
- Monetization beyond platform payouts

## Installation

1. Copy the `short-form-video/` folder into your OpenClaw skills directory
2. Restart OpenClaw

## Usage

Use this skill when:
- Repurposing YouTube videos into Shorts
- Creating native short-form content
- Optimizing for each platform's algorithm
- Writing scroll-stopping hooks
- Planning content calendars
- Analyzing short-form performance

## Requirements

- Video editing software (CapCut, Premiere Pro)
- AI clipping tool (OpusClip, Quso.ai)
- Access to TikTok, YouTube, Instagram accounts
- Scheduling tool (optional)
- Analytics access for each platform
