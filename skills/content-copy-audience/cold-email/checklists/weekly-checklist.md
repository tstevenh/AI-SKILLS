# Weekly Checklist

Complete these tasks every week to optimize performance and maintain health.

## Metrics Review (Monday)

### Campaign Performance
- [ ] Review open rates by campaign
- [ ] Review reply rates by campaign
- [ ] Analyze positive reply rate
- [ ] Compare to previous week

### Sequence Analysis
- [ ] Check performance by sequence step
- [ ] Identify best/worst performing emails
- [ ] Note patterns (day of week, time)

### Pipeline Impact
- [ ] Count meetings booked
- [ ] Calculate cost per meeting
- [ ] Track pipeline generated
- [ ] Update forecasts

## Infrastructure Health (Tuesday)

### Deliverability
- [ ] Check Google Postmaster Tools
- [ ] Review domain reputation
- [ ] Scan for blacklist issues
- [ ] Verify authentication passing

### Mailboxes
- [ ] Check mailbox health in tool
- [ ] Review warmup metrics
- [ ] Assess sending volume per mailbox
- [ ] Identify any struggling mailboxes

## List Hygiene (Wednesday)

### Data Quality
- [ ] Process week's bounces
- [ ] Remove unsubscribes
- [ ] Update suppression list
- [ ] Review data freshness

### Prospecting
- [ ] Add new prospects as needed
- [ ] Verify new list additions
- [ ] Check data sources for quality

## Optimization (Thursday)

### Testing
- [ ] Review A/B test results
- [ ] Plan new tests for next week
- [ ] Document test learnings

### Copy Iteration
- [ ] Identify underperforming emails
- [ ] Draft improvements
- [ ] Update templates as needed

## Planning (Friday)

### Next Week
- [ ] Set priorities
- [ ] Plan campaigns
- [ ] Prepare new sequences if needed

### Reporting
- [ ] Generate weekly report
- [ ] Share with stakeholders
- [ ] Log learnings
