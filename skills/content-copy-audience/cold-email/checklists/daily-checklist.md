# Daily Checklist

Complete these tasks every day to maintain healthy cold email operations.

## Morning (Start of Day)

### Replies
- [ ] Check inbox for overnight replies
- [ ] Categorize replies (positive, objection, etc.)
- [ ] Respond to positive replies immediately
- [ ] Queue objections and questions for response

### Metrics
- [ ] Check bounce rate (any spikes?)
- [ ] Review complaint rate (any issues?)
- [ ] Glance at open/reply rates
- [ ] Note any anomalies

### Sending
- [ ] Verify scheduled sends are running
- [ ] Check for any sending errors
- [ ] Confirm mailbox health

## Throughout Day

### Replies (Every 2-3 Hours)
- [ ] Check for new replies
- [ ] Respond to positive replies within 2 hours
- [ ] Handle objections within 4 hours
- [ ] Process unsubscribes immediately

### Meetings
- [ ] Confirm today's meetings
- [ ] Prep for upcoming calls
- [ ] Log meeting outcomes

## End of Day

### Cleanup
- [ ] Clear reply queue
- [ ] Process any bounces
- [ ] Update CRM notes
- [ ] Log day's activity

### Review
- [ ] Quick metrics check
- [ ] Note any issues for tomorrow
- [ ] Plan next day priorities

## Quick Reference

### Reply Response Times
| Type | Target Time |
|------|-------------|
| Positive | <2 hours |
| Objection | <4 hours |
| Question | <4 hours |
| Referral | <24 hours |
| Negative | <24 hours |

### Alert Triggers
Take immediate action if:
- Bounce rate >5% in one day
- Spam complaints spike
- Delivery rate drops suddenly
- Mailbox suspended
