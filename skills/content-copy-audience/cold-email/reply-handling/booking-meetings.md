# Booking Meetings

Converting positive replies to calendar appointments is where cold email becomes pipeline. This chapter covers the art and science of getting from "interested" to "booked."

## The Booking Funnel

**Positive Reply** → **Meeting Request** → **Time Confirmation** → **Calendar Event** → **Meeting Happens**

Each step has drop-off. Optimization matters.

---

## Speed to Response

### Why Speed Matters
- **Reply within 5 minutes**: 21x more likely to qualify
- **Reply within 1 hour**: 7x more likely than 24 hours
- **After 24 hours**: Interest often cools

### Speed Optimization
1. Enable notifications for positive replies
2. Use mobile app for immediate response
3. Have booking templates ready
4. Use calendar links to reduce friction
5. Check inbox frequently during business hours

## Booking Response Templates

### Standard Positive Reply
```
[First name],

Great to hear from you!

I've got some availability this week:
- [Day] at [Time]
- [Day] at [Time]
- [Day] at [Time]

Or grab a time that works: [Calendar Link]

Looking forward to it.

[Your name]
```

### Enthusiastic Reply
```
[First name],

Excellent—excited to connect!

Does [specific day/time] work for a quick 15-minute call?

If not, here's my calendar: [Link]

Talk soon.

[Your name]
```

### Busy Prospect
```
[First name],

Thanks for getting back to me.

I know you're busy—just need 15 minutes.

Easiest way: [Calendar Link] (pick what works)

Or reply with times and I'll send an invite.

[Your name]
```

## Calendar Tools

### Calendly
- Most recognized
- Easy to use
- Good free tier
- $12-20/month for premium

### Cal.com
- Open source
- Free tier available
- Self-hostable
- $12/month for premium

### SavvyCal
- Recipient-focused
- Better experience for them
- $12/user/month

### Best Practices
- Use branded links (Calendly.com/yourcompany)
- Set appropriate meeting lengths (15-30 min)
- Buffer between meetings
- Include timezone detection
- Send confirmation + reminder

## Confirmation Sequences

### Immediately After Booking
```
Subject: Confirmed: [Meeting] on [Date]

[First name],

We're set for [Day] at [Time] [Timezone].

Here's what we'll cover:
- [Agenda item 1]
- [Agenda item 2]

Ahead of our call, it might help to [prep request if any].

See you then.

[Your name]

[Calendar invite attached/link]
```

### 24 Hours Before
```
Subject: Tomorrow: Quick chat

[First name],

Looking forward to our call tomorrow at [Time].

We'll focus on [topic]—should take about [duration].

Here's the link: [Meeting Link]

Any questions before we connect?

[Your name]
```

### 1 Hour Before
```
Subject: We're on in 1 hour

[First name],

Just a heads up—we're connecting at [Time].

Join link: [Link]

See you shortly.

[Your name]
```

## No-Show Handling

### If They Don't Show (within 5 minutes)
```
Subject: Did I miss you?

[First name],

I was on the call but didn't see you join. All good?

Happy to reschedule—here's my calendar: [Link]

Or let me know if something came up.

[Your name]
```

### Follow-Up After No-Show (next day)
```
Subject: Let's reschedule

[First name],

We missed each other yesterday—no worries, it happens.

Still happy to connect if you're interested.

Here's my calendar: [Link]

Or let me know if priorities have changed.

[Your name]
```

### Second No-Show
```
Subject: One more try

[First name],

Tried to connect twice now—guessing timing isn't right.

I'll hold off unless you want to reschedule.

Reach out whenever it makes sense.

[Your name]
```

## Reschedule Handling

### They Request Reschedule
```
[First name],

No problem at all—things happen.

Here's some new times:
- [Day/Time 1]
- [Day/Time 2]

Or grab what works: [Calendar Link]

[Your name]
```

### You Need to Reschedule
```
[First name],

I apologize—something came up and I need to reschedule our call on [original date].

Would any of these work instead?
- [Day/Time 1]
- [Day/Time 2]

Sorry for the inconvenience.

[Your name]
```

## Summary: Booking Principles

**Speed is Critical**: Respond to positive replies immediately.

**Reduce Friction**: Calendar links beat email ping-pong.

**Confirm Everything**: Reminder sequences reduce no-shows.

**Handle No-Shows Gracefully**: They happen; stay professional.

**Prepare Thoroughly**: Good meetings start before the meeting.
