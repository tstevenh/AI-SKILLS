## Sequence Best Practices

### Customization Requirements
Every sequence should be customized with:
- Specific company/prospect details
- Relevant case studies
- Industry-appropriate language
- Your product/service specifics

### Testing Approach
1. Start with one sequence
2. Run with 100+ prospects
3. Analyze which emails perform
4. Iterate on underperformers
5. Test variations

### Common Sequence Mistakes

**Too Generic**
Templates need customization. Don't just fill in {{company}}.

**Same Angle Repeated**
Each email should try a different approach.

**Too Frequent**
Respect timing guidelines (3-7 days between emails).

**No Breakup**
Always include a final "closing the loop" email.

## Building Custom Sequences

### Structure Template

**Email 1: Introduction**
- Personalized opening
- Value proposition
- Social proof
- Soft CTA

**Email 2: The Bump (Day 3-4)**
- Brief follow-up
- Restate value
- Clear CTA

**Email 3: New Angle (Day 7-10)**
- Different approach
- New proof point
- Easy CTA

**Email 4: Value Add (Day 14-16)**
- Provide something useful
- Build goodwill
- Soft CTA

**Email 5: Social Proof (Day 18-21)**
- Case study focus
- Specific results
- Direct CTA

**Email 6: Breakup (Day 25-28)**
- Acknowledge silence
- Leave door open
- Professional close
