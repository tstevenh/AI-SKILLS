# Cold Email Mastery

Expert cold email strategy for B2B outbound sales and lead generation.

## What It Does

This skill teaches systematic B2B cold email outreach—from infrastructure setup and list building to copywriting and reply handling. Build predictable lead generation systems that convert.

## Features

- Infrastructure setup (domains, mailboxes, warming)
- Copywriting frameworks and templates
- List building and data enrichment strategies
- Subject line and opening line formulas
- Multi-touch sequence design
- Deliverability optimization
- Reply handling and objection responses
- Performance benchmarking and optimization

## Installation

1. Copy the `cold-email/` folder into your OpenClaw skills directory
2. Restart OpenClaw

## Usage

Use this skill when:
- Setting up cold email infrastructure
- Writing cold email sequences
- Troubleshooting deliverability issues
- Building prospect lists
- Handling replies and objections
- Scaling outbound campaigns

## Requirements

- Multiple domains and mailboxes (Google Workspace/Office 365)
- Cold email tool (Instantly, Smartlead, or similar)
- Email verification service
- List building tools (Apollo, Clay, or similar)
- Compliance with CAN-SPAM/GDPR regulations
