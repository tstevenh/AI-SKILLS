# Community Scaling Checklist

## Overview

Use this checklist to assess your readiness to scale and track progress as you grow your community. Each section corresponds to a scaling milestone.

---

## Stage 1: Foundation (0-500 Members)

### Community Fundamentals
- [ ] Clear community purpose defined
- [ ] Target audience identified
- [ ] Value proposition articulated
- [ ] Pricing validated
- [ ] Platform selected and configured

### Basic Operations
- [ ] Welcome process documented
- [ ] Community guidelines published
- [ ] Content calendar created
- [ ] Weekly rhythm established
- [ ] Basic metrics tracking in place

### Engagement Foundation
- [ ] Daily engagement happening
- [ ] Weekly events scheduled
- [ ] Member introductions encouraged
- [ ] Wins celebrated regularly
- [ ] Questions answered promptly

### Revenue Basics
- [ ] Payment processing working
- [ ] Revenue tracking set up
- [ ] Basic financial projections created
- [ ] Unit economics understood
- [ ] Churn being monitored

### Stage 1 Metrics Targets
```
Members: 500
MRR: $2,500-25,000
Churn: <15% monthly
Engagement: 30%+ monthly active
```

## Pre-Scaling Readiness Assessment

### Answer These Questions

**Product-Market Fit**
- [ ] Do members consistently see value? (NPS >40)
- [ ] Is retention healthy? (<10% monthly churn)
- [ ] Do members refer others? (referral rate >10%)
- [ ] Is demand consistent? (no feast/famine)

**Unit Economics**
- [ ] Is LTV > 3x CAC?
- [ ] Is gross margin healthy? (>60%)
- [ ] Is churn manageable?
- [ ] Can you acquire members profitably?

**Operations**
- [ ] Are processes documented?
- [ ] Could someone else run daily ops?
- [ ] Are systems not founder-dependent?
- [ ] Is quality consistent?

**Team**
- [ ] Do you have (or can hire) help?
- [ ] Is training material ready?
- [ ] Are expectations clear?
- [ ] Is there budget for team?

### Ready to Scale?
Score yourself 1-5 on each area above.
- 4-5 average: Ready to scale
- 3-4 average: Address gaps first
- <3 average: Not ready yet

## Scaling Warning Signs

### Watch for These Issues

**Growth Too Fast**
- [ ] Quality declining
- [ ] Engagement dropping
- [ ] Support overwhelmed
- [ ] Culture diluting
- [ ] Team burning out

**Growth Too Slow**
- [ ] Market passing you by
- [ ] Competitors gaining
- [ ] Revenue plateauing
- [ ] Team disengaged
- [ ] Innovation stalled

### Course Corrections

**If Growing Too Fast:**
1. Slow down acquisition
2. Fix operational issues
3. Hire/train team
4. Improve systems
5. Resume growth when ready

**If Growing Too Slow:**
1. Review product-market fit
2. Increase marketing investment
3. Test new channels
4. Improve conversion
5. Address retention issues
