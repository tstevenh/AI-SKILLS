# Functionality QA Skill - Chapter Index

This directory contains the **Functionality QA** skill documentation, structured as a two-part learning system:

## Structure Overview

- **SKILL.md** - The main skill documentation covering core functionality QA concepts, test case design, and practical testing workflows
- **CHAPTER-XX.md** files - Deep-dive chapters covering specialized testing topics in detail

## Files in this Directory

| File | Topic | Description |
|------|-------|-------------|
| **SKILL.md** | Core Functionality QA | Main skill documentation covering test case design, E2E workflows, API testing, database testing, form validation, session management, OAuth flows, payment testing, email testing, webhooks, cron jobs, WordPress testing, and CI/CD integration |
| **CHAPTER-01.md** | API Testing Deep Dive | Comprehensive guide to API testing methodologies, tools, authentication patterns, and advanced techniques |
| **CHAPTER-02.md** | Performance and Load Testing | Load testing strategies, performance benchmarking, JMeter and k6 usage, bottleneck analysis |
| **CHAPTER-03.md** | Security Testing Foundations | OWASP Top 10, SQL injection, XSS testing, authentication/authorization testing |
| **CHAPTER-07.md** | Security Testing for QA Engineers | Advanced security testing techniques tailored for QA professionals |
| **CHAPTER-08.md** | Chaos Engineering and Reliability | Chaos engineering principles, reliability testing, fault injection, system resilience |
| **CHAPTER-09.md** | E2E Testing Strategy | End-to-end testing approaches, test architecture, maintenance strategies |
| **CHAPTER-10.md** | QA Leadership and Team Management | Team leadership, process improvement, metrics, stakeholder communication |
