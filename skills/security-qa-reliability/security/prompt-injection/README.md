# Prompt Injection Defense

Comprehensive defenses against indirect prompt injection attacks on AI agents.

## What It Does

This skill provides protection strategies against indirect prompt injection (IPI)—malicious instructions hidden in external content that attempt to hijack AI agent behavior. Essential for web research agents and autonomous AI systems.

## Features

- Indirect prompt injection threat modeling
- Content sanitization and detection patterns
- Spotlighting and data marking techniques
- Context isolation (XML tagging)
- Instruction hierarchy enforcement
- Two-agent architecture (Reader/Executor)
- Output filtering and canary tokens
- Behavioral monitoring for attacks

## Installation

1. Copy the `prompt-injection-defense/` folder into your OpenClaw skills directory
2. Restart OpenClaw

## Usage

Use this skill when:
- Building web research agents
- Processing untrusted external content
- Implementing AI agent security
- Designing safe browsing systems
- Testing agent vulnerabilities
- Creating content sanitization pipelines

## Requirements

- AI agent framework or implementation
- Content sanitization scripts
- Web content processing pipeline
- Monitoring and logging infrastructure
- Regular security testing schedule
