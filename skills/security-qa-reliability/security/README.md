# Penetration Testing

Comprehensive web application security testing and ethical hacking methodologies.

## What It Does

This skill teaches systematic penetration testing for web applications—identifying vulnerabilities, exploiting security weaknesses, and providing remediation guidance. Learn OWASP Top 10 testing and beyond.

## Features

- OWASP Top 10 2021/2025 deep dives
- Web application pentesting methodology
- Information gathering and reconnaissance
- Vulnerability scanning techniques
- Authentication and session management testing
- API security testing
- WordPress security testing
- SSL/TLS and security headers validation
- Vulnerability reporting and disclosure

## Installation

1. Copy the `penetration-testing/` folder into your OpenClaw skills directory
2. Restart OpenClaw

## Usage

Use this skill when:
- Performing security assessments
- Testing web applications for vulnerabilities
- Conducting penetration tests
- Validating security controls
- Writing vulnerability reports
- Learning security testing methodologies

## Requirements

- **WARNING**: Only use on systems with explicit written authorization
- Burp Suite or OWASP ZAP
- Kali Linux or security testing distribution
- Vulnerable practice environments (DVWA, WebGoat)
- Proper legal agreements and scope documentation
